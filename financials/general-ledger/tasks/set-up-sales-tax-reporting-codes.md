# <a name="set-up-sales-tax-reporting-codes"></a>Imposta i codici reporting IVA

[!include[task guide banner](../../includes/task-guide-banner.md)]

Questi codici reporting IVA fanno riferimento a un numero di campo in un report IVA. Vengono utilizzati nei layout di report specifici del paese e nel report Pagamento IVA per codice per stampare gli importi IVA per un periodo di liquidazione riepilogato per codice reporting. Dopo aver creato i codici reporting IVA, è possibile utilizzarli come riferimenti nelle Schede dettaglio Impostazione report della pagina Codice IVA. 

Questa registrazione utilizza la società dimostrativa DEMF.



1. Passare a Imposta > Impostazione > IVA > Codici reporting IVA.
2. Fare clic su Nuovo.
3. Selezionare il layout del report a cui appartiene il codice reporting.
    * Questo layout viene utilizzato per filtrare i codici reporting disponibili per un codice IVA. Ogni codice IVA appartiene a un periodo di liquidazione appartenente a un ufficio IVA che utilizza un layout di report.  
4. Immettere un numero che faccia riferimento a un campo in un report IVA.
5. Nel campo Testo report immettere una descrizione da visualizzare nei report.
6. Nel campo Breve descrizione immettere una descrizione per scopi interni.
7. Fare clic su Salva.
