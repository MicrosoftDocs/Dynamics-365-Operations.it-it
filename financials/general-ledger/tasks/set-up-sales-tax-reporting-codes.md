[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="aa44e-104">Questi codici reporting IVA fanno riferimento a un numero di campo in un report IVA.</span><span class="sxs-lookup"><span data-stu-id="aa44e-104">The Sales tax reporting codes refer to a field number on a sales tax report.</span></span> <span data-ttu-id="aa44e-105">Vengono utilizzati nei layout di report specifici del paese e nel report Pagamento IVA per codice per stampare gli importi IVA per un periodo di liquidazione riepilogato per codice reporting.</span><span class="sxs-lookup"><span data-stu-id="aa44e-105">They are used on country specific report layouts and the Sales tax payment by code report to print sales tax amounts for a settlement period summarized per reporting code.</span></span> <span data-ttu-id="aa44e-106">Dopo aver creato i codici reporting IVA, è possibile utilizzarli come riferimenti nelle Schede dettaglio Impostazione report della pagina Codice IVA.</span><span class="sxs-lookup"><span data-stu-id="aa44e-106">After you create Sales tax reporting codes, you can refer to them on the Report setup FastTabs in the Sales tax code page.</span></span> 

<span data-ttu-id="aa44e-107">Questa registrazione utilizza la società dimostrativa DEMF.</span><span class="sxs-lookup"><span data-stu-id="aa44e-107">This recording uses the DEMF demo company.</span></span>

3. <span data-ttu-id="aa44e-110">Selezionare il layout del report a cui appartiene il codice reporting.</span><span class="sxs-lookup"><span data-stu-id="aa44e-110">Select the report layout that the reporting code belongs to.</span></span>
    * <span data-ttu-id="aa44e-111">Questo layout viene utilizzato per filtrare i codici reporting disponibili per un codice IVA.</span><span class="sxs-lookup"><span data-stu-id="aa44e-111">This layout is used to filter the available reporting codes for a Sales tax code.</span></span> <span data-ttu-id="aa44e-112">Ogni codice IVA appartiene a un periodo di liquidazione appartenente a un ufficio IVA che utilizza un layout di report.</span><span class="sxs-lookup"><span data-stu-id="aa44e-112">Each Sales tax code belongs to a settlement period which belongs to a Sales tax authority which uses a Report layout.</span></span>

7. <span data-ttu-id="aa44e-116">Fare clic su Salva.</span><span class="sxs-lookup"><span data-stu-id="aa44e-116">Click Save.</span></span>
