# <a name="create-accrual-schemes"></a>Creare schemi di attribuzione per competenza

[!include[task guide banner](../../includes/task-guide-banner.md)]

Questa guida attività descrive la creazione di uno schema di attribuzione. In questa attività viene utilizzata la società dimostrativa USMF.

1. Andare a Contabilità generale > Impostazione giornale di registrazione > Schemi di attribuzione per competenza.
2. Fare clic su Nuovo.
3. Digitare un valore nel campo Identificazione attribuzione per competenza.
4. Nel campo Descrizione dello schema di attribuzione per competenza digitare un valore.
5. Nel campo Dare, specificare i valori desiderati.
    * Il conto principale definito sostituirà il conto principale in Dare nella riga di giustificativo giornale di registrazione e verrà inoltre utilizzato per lo storno del differimento basato sulle transazioni di attribuzione per competenza in contabilità generale.  
6. Nel campo Avere, specificare i valori desiderati.
    * Il conto principale definito sostituirà il conto principale in Avere nella riga di giustificativo giornale di registrazione e verrà inoltre utilizzato per lo storno del differimento basato sulle transazioni di attribuzione per competenza in contabilità generale.  
7. Nel campo Giustificativo, selezionare la modalità di determinazione del giustificativo quando vengono registrate le transazioni.
8. Nel campo Descrizione, immettere un valore per descrivere le transazioni che verranno registrate.
9. Nel campo Frequenza periodo, selezionare la frequenza delle transazioni.
10. Nel campo Numero di occorrenze per periodo, immettere un numero.
11. Nel campo Registra transazioni specificare quando le transazioni devono essere registrate, ad esempio mensilmente.
