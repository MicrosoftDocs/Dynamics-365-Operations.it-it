# <a name="change-a-fixed-asset-group"></a>Cambiare un gruppo di cespiti

[!include[task guide banner](../../includes/task-guide-banner.md)]

I cespiti devono essere assegnati al gruppo di cespiti corretto. Il gruppo di cespiti viene utilizzato quando si completano le attività seguenti:

 - Creazione di richieste di informazioni e report

 - Impostazione di nuovi cespiti

 - Integrazione dei saldi contabili e registrazione delle transazioni cespiti nei conti CoGe appropriati

Questa guida utilizza la società dimostrativa USMF.

1. Passare a Cespiti > Cespiti > Cespiti.
2. Selezionare il cespite per cui cambiare il gruppo cespite.
3. Fare clic su Cambia gruppo cespite.
4. Nel campo Nuovo gruppo immettere o selezionare un valore.
5. Selezionare questa opzione per assegnare un numero cespite al cespite selezionato.
    * Il campo Numero cespite è disponibile se si seleziona l'opzione Nuovo numero cespite.   Se per i cespiti è impostata la numerazione automatica, nel campo viene visualizzato il successivo numero cespite disponibile. Il numero può essere modificato.   Se è impostata la numerazione manuale, il campo risulta vuoto ed è necessario immettere il nuovo numero cespite.     
6. Fare clic su OK.
7. Fare clic su Sì.
