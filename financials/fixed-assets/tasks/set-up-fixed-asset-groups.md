# <a name="set-up-fixed-asset-groups"></a><span data-ttu-id="8057b-103">Imposta i gruppi cespite</span><span class="sxs-lookup"><span data-stu-id="8057b-103">Set up fixed asset groups</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="8057b-104">In questa procedura viene illustrato come creare un nuovo gruppo cespite.</span><span class="sxs-lookup"><span data-stu-id="8057b-104">This procedure shows how to create a new fixed asset group.</span></span> <span data-ttu-id="8057b-105">Utilizza il ruolo Ragioniere e i dati dimostrativi per la persona giuridica USMF.</span><span class="sxs-lookup"><span data-stu-id="8057b-105">It uses the Accountant role and demo data for the USMF legal entity.</span></span>

1. <span data-ttu-id="8057b-106">Andare a Cespiti > Impostazioni > Gruppi cespite.</span><span class="sxs-lookup"><span data-stu-id="8057b-106">Go to Fixed assets > Setup > Fixed asset groups.</span></span>
2. <span data-ttu-id="8057b-107">Fare clic su Nuovo.</span><span class="sxs-lookup"><span data-stu-id="8057b-107">Click New.</span></span>
3. <span data-ttu-id="8057b-108">Nel campo Gruppo cespite digitare un valore.</span><span class="sxs-lookup"><span data-stu-id="8057b-108">In the Fixed asset group field, type a value.</span></span>
4. <span data-ttu-id="8057b-109">Digitare un valore nel campo Nome.</span><span class="sxs-lookup"><span data-stu-id="8057b-109">In the Name field, type a value.</span></span>
    * <span data-ttu-id="8057b-110">Numerazione automatica cespiti e Codice sequenza numerica in Gruppo cespite sostituiranno le impostazioni di Parametri cespiti.</span><span class="sxs-lookup"><span data-stu-id="8057b-110">Autonumber fixed assets and Number sequence code on the Fixed asset group will override the settings on the Fixed assets parameters.</span></span> <span data-ttu-id="8057b-111">È possibile modificarlo in questo campo se i cespiti del gruppo cespite avranno una numerazione diversa da altri gruppi.</span><span class="sxs-lookup"><span data-stu-id="8057b-111">You can change it here if the assets in this fixed asset group will have different numbering from other groups.</span></span>  
5. <span data-ttu-id="8057b-112">Fare clic su Libri.</span><span class="sxs-lookup"><span data-stu-id="8057b-112">Click Books.</span></span>
6. <span data-ttu-id="8057b-113">Nel campo Libro immettere o selezionare un valore.</span><span class="sxs-lookup"><span data-stu-id="8057b-113">In the Book field, enter or select a value.</span></span>
    * <span data-ttu-id="8057b-114">Il campo Calcola ammortamento è impostato su Sì, quindi il libro del cespite verrà incluso nelle proposte di ammortamento.</span><span class="sxs-lookup"><span data-stu-id="8057b-114">The Calculate depreciation field is set to Yes, so the asset book will be included in depreciation proposals.</span></span> <span data-ttu-id="8057b-115">Se Calcola ammortamento è impostato su No, il cespite non verrà ammortizzato automaticamente.</span><span class="sxs-lookup"><span data-stu-id="8057b-115">If Calculate depreciation is set to No, the asset will not be automatically depreciated.</span></span>  
7. <span data-ttu-id="8057b-116">Impostare la vita utile del cespite, espressa in anni.</span><span class="sxs-lookup"><span data-stu-id="8057b-116">Set the Service life of the asset, in years.</span></span>
    * <span data-ttu-id="8057b-117">Si noti che il valore del campo Periodi di ammortamento viene calcolato dopo aver impostato la Vita utile.</span><span class="sxs-lookup"><span data-stu-id="8057b-117">Note that the Depreciation periods field value is calculated after setting the Service life.</span></span>  
8. <span data-ttu-id="8057b-118">Nel campo Convenzione di ammortamento selezionare un'opzione.</span><span class="sxs-lookup"><span data-stu-id="8057b-118">In the Depreciation convention field, select an option.</span></span>
9. <span data-ttu-id="8057b-119">Chiudere la pagina.</span><span class="sxs-lookup"><span data-stu-id="8057b-119">Close the page.</span></span>
