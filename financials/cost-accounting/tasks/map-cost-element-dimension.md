# <a name="map-a-cost-element-dimension"></a><span data-ttu-id="8d630-103">Mappare una dimensione elemento di costo</span><span class="sxs-lookup"><span data-stu-id="8d630-103">Map a cost element dimension</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="8d630-104">Un controllore costi può utilizzare questa procedura per mappare una dimensione dell'elemento di costo in una dimensione dell'elemento di costo nella persona giuridica MXMF.</span><span class="sxs-lookup"><span data-stu-id="8d630-104">A cost controller can use this procedure to map a cost element dimension to a cost element dimension in the MXMF legal entity.</span></span> <span data-ttu-id="8d630-105">Questa registrazione utilizza i dati dimostrativi della società USP2.</span><span class="sxs-lookup"><span data-stu-id="8d630-105">This recording uses the USP2 demo data company.</span></span>

1. <span data-ttu-id="8d630-106">Passare a Contabilità industriale > Dimensioni > Dimensioni elemento di costo.</span><span class="sxs-lookup"><span data-stu-id="8d630-106">Go to Cost accounting > Dimensions > Cost element dimensions.</span></span>
2. <span data-ttu-id="8d630-107">Nell'elenco trovare e selezionare il record desiderato.</span><span class="sxs-lookup"><span data-stu-id="8d630-107">In the list, find and select the desired record.</span></span>
    * <span data-ttu-id="8d630-108">Per questo esempio, selezionare Elementi di costo.</span><span class="sxs-lookup"><span data-stu-id="8d630-108">For this example, select Cost elements.</span></span>  
3. <span data-ttu-id="8d630-109">Fare clic su Mapping dimensione.</span><span class="sxs-lookup"><span data-stu-id="8d630-109">Click Dimension mappings.</span></span>
4. <span data-ttu-id="8d630-110">Fare clic su Configura mapping dalla dimensione.</span><span class="sxs-lookup"><span data-stu-id="8d630-110">Click Configure mappings from this dimension.</span></span>
5. <span data-ttu-id="8d630-111">Fare clic su Nuovo.</span><span class="sxs-lookup"><span data-stu-id="8d630-111">Click New.</span></span>
6. <span data-ttu-id="8d630-112">Nel campo A dimensione immettere o selezionare un valore.</span><span class="sxs-lookup"><span data-stu-id="8d630-112">In the To dimension field, enter or select a value.</span></span>
    * <span data-ttu-id="8d630-113">Per questo esempio, selezionare gli elementi di costo MXMF.</span><span class="sxs-lookup"><span data-stu-id="8d630-113">For this example, select MXMF Cost elements.</span></span>  
7. <span data-ttu-id="8d630-114">Fare clic su Nuovo.</span><span class="sxs-lookup"><span data-stu-id="8d630-114">Click New.</span></span>
8. <span data-ttu-id="8d630-115">Nell'elenco contrassegnare la riga selezionata.</span><span class="sxs-lookup"><span data-stu-id="8d630-115">In the list, mark the selected row.</span></span>
9. <span data-ttu-id="8d630-116">Nel campo Membro di dimensione di inizio immettere o selezionare un valore.</span><span class="sxs-lookup"><span data-stu-id="8d630-116">In the From dimension member field, enter or select a value.</span></span>
    * <span data-ttu-id="8d630-117">Per questo esempio, selezionare il membro di dimensione 606400 Spese telefono e fax.</span><span class="sxs-lookup"><span data-stu-id="8d630-117">For this example, select dimension member 606400 Telephone & Fax Expense.</span></span>  
10. <span data-ttu-id="8d630-118">Nel campo Membro di dimensione di fine immettere o selezionare un valore.</span><span class="sxs-lookup"><span data-stu-id="8d630-118">In the To dimension member field, enter or select a value.</span></span>
    * <span data-ttu-id="8d630-119">Per questo esempio selezionare il membro di dimensione 6001004 Telefono.</span><span class="sxs-lookup"><span data-stu-id="8d630-119">For this example, select dimension member 6001004 Telefono.</span></span>  
11. <span data-ttu-id="8d630-120">Fare clic su Salva.</span><span class="sxs-lookup"><span data-stu-id="8d630-120">Click Save.</span></span>
