<span data-ttu-id="11286-105">La società di dati dimostrativi utilizzata per creare questa procedura è DEMF.</span><span class="sxs-lookup"><span data-stu-id="11286-105">The demo data company used to create this procedure is DEMF.</span></span>

3. <span data-ttu-id="11286-108">Digitare un valore nel campo Certificato di entrata.</span><span class="sxs-lookup"><span data-stu-id="11286-108">In the Entry certificate field, type a value.</span></span>

6. <span data-ttu-id="11286-111">Fare clic su Salva.</span><span class="sxs-lookup"><span data-stu-id="11286-111">Click Save.</span></span>
    * <span data-ttu-id="11286-112">Allegare informazioni aggiuntive al giornale di registrazione.</span><span class="sxs-lookup"><span data-stu-id="11286-112">Attach additional information to the journal.</span></span>  
    * <span data-ttu-id="11286-113">Ad esempio, allegare un'immagine digitalizzata del certificato di entrata UE nel giornale di registrazione.</span><span class="sxs-lookup"><span data-stu-id="11286-113">For example, attach a scanned image of the EU Entry certificate to the journal.</span></span>
