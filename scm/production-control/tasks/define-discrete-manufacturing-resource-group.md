# <a name="define-discrete-manufacturing-resource-group"></a>Definire il gruppo di risorse di produzione discreta

[!include[task guide banner](../../includes/task-guide-banner.md)]

Un gruppo di risorse è un set di risorse operative che in genere corrisponde all'organizzazione fisica delle celle di lavoro, definito dalle righe gialle nello shop floor di produzione. In questa procedura viene illustrato come definire un gruppo di risorse da utilizzare nella produzione discreta. È possibile eseguire questa procedura nella società di dati dimostrativi USMF oppure utilizzare i propri dati.

1. Fare clic su Gruppo di risorse.
2. Fare clic su Nuovo.
3. Digitare un valore nel campo Gruppo di risorse.
4. Nel campo Descrizione digitare un valore.
5. Nel campo Sito immettere o selezionare un valore.
6. Nel campo Unità di produzione immettere o selezionare un valore.

## <a name="define-default-operational-parameters"></a>Definisce i parametri operativi predefiniti
1. Espandere la sezione Operazione.
2. Immettere un numero nel campo Percentuale scarti.
3. Nel campo Categoria tempo di attrezzaggio immettere o selezionare un valore.
4. Nel campo Categoria tempo di esecuzione immettere o selezionare un valore.
5. Nel campo Percentuale programmazione operazioni, immettere un numero.

## <a name="define-operating-hours"></a>Definisce le ore di funzionamento
1. Espandere la sezione Calendari.
2. Scegliere Aggiungi.
3. Nel campo Calendario immettere o selezionare un valore.

## <a name="add-operations-resources"></a>Aggiunge le risorse operative
1. Espandere la sezione Risorse.
2. Scegliere Aggiungi.
3. Nel campo Risorsa immettere o selezionare un valore.
4. Scegliere Aggiungi.
5. Nel campo Risorsa immettere o selezionare un valore.
6. Nell'elenco trovare e selezionare il record desiderato.
7. Nell'elenco fare clic sul collegamento nella riga selezionata.
