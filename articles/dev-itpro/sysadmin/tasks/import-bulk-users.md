# <a name="import-users-in-bulk"></a>Importare utenti in blocco

[!include [task guide banner](../../includes/task-guide-banner.md)]

Questa procedura può essere utilizzata dagli amministratori di sistema per importare un numero elevato di utenti da Azure Active Directory.


## <a name="run-as-a-batch-job"></a>Esegui come processo batch
1. Andare a Amministrazione sistema > Utenti > Utenti.
2. Fare clic su Importazione batch.
3. Espandere la sezione Esecuzione in background.
4. Selezionare Sì nel campo Elaborazione batch.
5. Nel campo Descrizione attività immettere un valore.
6. Nel campo Gruppo batch immettere o selezionare un valore.
    * Questo passaggio è facoltativo.  
7. Selezionare Sì nel campo Privato.
    * Questo passaggio è facoltativo.  
8. Selezionare Sì nel campo Processo critico.
    * Questo passaggio è facoltativo.  
9. Selezionare un'opzione nel campo Categoria di monitoraggio.
10. Fare clic su OK.

## <a name="run-in-a-sandbox-environment"></a>Eseguire l'operazione in un ambiente sandbox
1. Fare clic su Importazione batch.
2. Fare clic su OK.
