    - <span data-ttu-id="909aa-109">Amministratore di sistema</span><span class="sxs-lookup"><span data-stu-id="909aa-109">System administrator</span></span>

2. <span data-ttu-id="909aa-110">Andare ad **Amministrazione organizzazione** &gt; **Creazione di report elettronici**.</span><span class="sxs-lookup"><span data-stu-id="909aa-110">Go to **Organization administration** &gt; **Electronic reporting**.</span></span>

    1. <span data-ttu-id="909aa-116">Fare clic su **Aggiungi** per aggiungere un nuovo archivio.</span><span class="sxs-lookup"><span data-stu-id="909aa-116">Click **Add** to add a new repository.</span></span>
    2. <span data-ttu-id="909aa-117">Selezionare **LCS** come tipo di archivio.</span><span class="sxs-lookup"><span data-stu-id="909aa-117">Select **LCS** as the repository type.</span></span>
    3. <span data-ttu-id="909aa-118">Fare clic su **Crea archivio**.</span><span class="sxs-lookup"><span data-stu-id="909aa-118">Click **Create repository**.</span></span>
    4. <span data-ttu-id="909aa-119">Se richiesto, seguire le istruzioni di autorizzazione.</span><span class="sxs-lookup"><span data-stu-id="909aa-119">If prompted, follow the authorization instructions.</span></span>
    5. <span data-ttu-id="909aa-120">Immettere un nome e una descrizione per l'archivio.</span><span class="sxs-lookup"><span data-stu-id="909aa-120">Enter a name and description for the repository.</span></span>

    <span data-ttu-id="909aa-124">[![update-er-from-lcs-for-ms-make-lcs-repository](./media/update-er-from-lcs-for-ms-make-lcs-repository.png)](./media/update-er-from-lcs-for-ms-make-lcs-repository.png)</span><span class="sxs-lookup"><span data-stu-id="909aa-124">[![update-er-from-lcs-for-ms-make-lcs-repository](./media/update-er-from-lcs-for-ms-make-lcs-repository.png)](./media/update-er-from-lcs-for-ms-make-lcs-repository.png)</span></span>

7. <span data-ttu-id="909aa-125">Nella struttura di configurazioni nel riquadro sinistro, selezionare la configurazione per la creazione di report elettronici desiderata.</span><span class="sxs-lookup"><span data-stu-id="909aa-125">In the configurations tree in the left pane, select the ER configuration that you require.</span></span>
