# <a name="troubleshoot-store-commerce-performance-issues"></a>Risolvere i problemi di prestazioni di Store Commerce

[!include [banner](../includes/banner.md)]

Questo articolo spiega come risolvere i problemi di prestazioni nell'app Microsoft Dynamics 365 Commerce Store Commerce.

- Quando risolvi i problemi di prestazioni per Store Commerce, evita l'accesso remoto. Accedi invece direttamente al dispositivo Store Commerce.
- Se l'app Store Commerce è lenta, controlla l'utilizzo di CPU, memoria, rete e disco utilizzando Task Manager o Resource Monitor. Assicurati che non vi sia un consumo significativo delle risorse del dispositivo.
- Controlla lo stato della connettività di rete e verifica che le richieste/risposte HTTPS non richiedano più di pochi secondi.
