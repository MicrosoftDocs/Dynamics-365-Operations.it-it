<span data-ttu-id="c1a2d-106">Un modulo Posizionamento contenuti è un contenitore speciale in cui è possibile includere altri moduli in un layout specifico.</span><span class="sxs-lookup"><span data-stu-id="c1a2d-106">A content placement module is a special container that other modules can be put inside in a specific layout.</span></span> <span data-ttu-id="c1a2d-107">I moduli Posizionamento contenuti supportano i seguenti tipi di moduli come moduli figlio: Elemento posizionamento contenuti, Elemento messaggio di benvenuto account, Elemento elenco preferenze account ed Elemento profilo account.</span><span class="sxs-lookup"><span data-stu-id="c1a2d-107">Content placement modules support the following types of modules as child modules: content placement item, account welcome item, account order item, account wish list item, and account profile item.</span></span>
