# <a name="browser-requirements-for-dynamics-365-commerce-sites"></a>Requisiti del browser per i siti Dynamics 365 Commerce

[!include [banner](includes/banner.md)]

Questo articolo tratta i requisiti del browser per i siti Microsoft Dynamics 365 Commerce.

I siti Dynamics 365 Commerce sono supportati al meglio dai seguenti browser web. Per trovare l'ultima versione rilasciata di ogni Web browser, andare al sito Web del produttore software.

- [Microsoft Edge](https://www.microsoft.com/edge) (fino alla versione corrente meno 1)
- [Mozilla Firefox](https://www.mozilla.org/firefox/new) (fino alla versione corrente meno 1)
- [Google Chrome](https://www.google.com/chrome) (fino alla versione corrente meno 1)
- [Apple Safari](https://support.apple.com/downloads/safari) (fino alla versione corrente meno 1)

> [!NOTE]
> Internet Explorer non è supportato. Se usi Internet Explorer, alcune funzionalità potrebbero non funzionare e gli utenti potrebbero non avere un'esperienza standard.
