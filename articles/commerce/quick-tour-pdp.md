## <a name="buy-box-module"></a><span data-ttu-id="31541-113">Modulo Casella acquisti</span><span class="sxs-lookup"><span data-stu-id="31541-113">Buy box module</span></span>

## <a name="recommendations-module"></a><span data-ttu-id="31541-129">Modulo Suggerimenti</span><span class="sxs-lookup"><span data-stu-id="31541-129">Recommendations module</span></span>

- <span data-ttu-id="31541-134">L'elenco **Alle persone piace anche** è basato sul machine learning.</span><span class="sxs-lookup"><span data-stu-id="31541-134">The **People also like** list is based on machine learning.</span></span> <span data-ttu-id="31541-135">Utilizza lo storico transazioni di altri clienti per fornire suggerimenti.</span><span class="sxs-lookup"><span data-stu-id="31541-135">It uses the transaction history of other customers to provide recommendations.</span></span> <span data-ttu-id="31541-136">Questo elenco viene generato dal servizio Suggerimenti ed è simile agli elenchi "Chi ha acquistato questo articolo ha acquistato anche...".</span><span class="sxs-lookup"><span data-stu-id="31541-136">This list is generated by the recommendations service and resembles "Customers who bought this also bought..." lists.</span></span> <span data-ttu-id="31541-137">Un ID prodotto è necessario per generare questo elenco.</span><span class="sxs-lookup"><span data-stu-id="31541-137">A product ID is required to generate this list.</span></span>
