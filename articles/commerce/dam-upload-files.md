# <a name="upload-files-other-than-images-and-videos"></a>Caricare file diversi da immagini e video

[!include [banner](includes/banner.md)]

Questo argomento descrive come caricare file diversi da immagini e video in Creazione di siti Web Microsoft Dynamics 365 Commerce.

## <a name="overview"></a>Panoramica

La libreria multimediale Creazione di siti Web Commerce supporta il caricamento di risorse binarie diverse da immagini o video. Ad esempio, si potrebbe voler caricare file Microsoft Excel, Microsoft Word, Microsoft PowerPoint o PDF.

Sono supportati i seguenti tipi di documento:
- 7Z
- AVI
- CS
- CSS
- DOC
- DOCX
- EPUB
- GIF
- INDD
- JAR
- JPG
- JPEG
- JS
- MP3
- MP4
- MPEG
- MPG
- ODP
- ODS
- ODT
- PDF
- PNG
- PPT
- PPTX
- PS
- QXP
- RAR
- RTF
- SVG
- TAR
- TGZ
- TXT
- WMV
- XLS
- XLSX
- XML
- ZIP

## <a name="upload-a-file"></a>Carica un file

Per caricare un file in Creazione di siti Web Commerce, effettuare le seguenti operazioni.

1. Nel riquadro di spostamento sinistro, selezionare **Libreria multimediale**.
1. Sulla barra dei comandi, selezionare **Carica \> Carica elementi multimediali**.
1. In Esplora file, selezionare uno o pi√π file, quindi selezionare **Apri**.
1. Nella finestra di dialogo **Carica elemento multimediale**, immettere titolo, descrizione e metadati delle parole chiave, se necessario.
1. Per pubblicare i file immediatamente dopo averli caricati, selezionare la casella di controllo **Pubblica elementi multimediali dopo il caricamento**.
1. Selezionare **OK**.
