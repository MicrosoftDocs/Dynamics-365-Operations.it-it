# <a name="digital-asset-management-overview"></a>Panoramica della gestione cespiti digitali

[!include [banner](includes/banner.md)]

Questo articolo fornisce una panoramica della gestione risorse digitali in Creazione di siti Web Microsoft Dynamics 365 Commerce.

La libreria multimediale di Creazione di siti Web Dynamics 365 Commerce offre una funzionalità di gestione avanzata delle risorse digitali, incluso il supporto per quanto segue:
- Risorse di immagine
- Risorse video
- Altri tipi di risorse binarie, come documenti di Office e file PDF
- Immagini localizzate
- Caricamenti di cartelle
- Ritaglio di immagini
- Personalizzazione dei punti focali dell'immagine
- Risorse immagine omnicanale (immagini prodotto, varianti di prodotto, catalogo, categoria, lavoratore e cliente)

Le risorse digitali sono specifiche del canale, ad eccezione delle risorse omnicanale che sono predefinite a livello di ambiente. 

Le risorse digitali e le funzionalità di gestione si trovano all'interno di Creazione di siti Web sotto **Libreria multimediale** nel riquadro di spostamento a sinistra all'interno di un sito.
