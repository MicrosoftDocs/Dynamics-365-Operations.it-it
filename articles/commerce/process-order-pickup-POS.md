[!include [banner](includes/banner.md)]

<span data-ttu-id="6443a-104">Quando un [ordine del cliente](customer-orders-overview.md) viene creato per il ritiro in negozio, un utente del negozio può utilizzare l'applicazione punto vendita (POS) per avviare il ritiro dell'articolo.</span><span class="sxs-lookup"><span data-stu-id="6443a-104">When a [customer order](customer-orders-overview.md) is created for store pickup, a store user can use the point of sale (POS) application to start the pickup of inventory.</span></span> <span data-ttu-id="6443a-105">Il POS eseguirà l'acquisizione del pagamento finale come richiesto.</span><span class="sxs-lookup"><span data-stu-id="6443a-105">POS will run the final payment capture as required.</span></span> <span data-ttu-id="6443a-106">Completerà inoltre l'inventario e la registrazione finanziaria per le quantità prelevate.</span><span class="sxs-lookup"><span data-stu-id="6443a-106">It will also complete the inventory and financial posting for the quantities that are picked up.</span></span>

<span data-ttu-id="6443a-107">Se sei un utente del negozio, puoi eseguire il ritiro utilizzando l'operazione **Richiama ordine** o l'operazione **Evasione ordini** in POS.</span><span class="sxs-lookup"><span data-stu-id="6443a-107">If you're a store user, you can perform the pickup by using either the **Recall order** operation or the **Order fulfillment** operation in POS.</span></span> <span data-ttu-id="6443a-108">Per rendere disponibile l'operazione **Preleva** è necessario prima seguire uno di questi passaggi:</span><span class="sxs-lookup"><span data-stu-id="6443a-108">To make the **Pick up** operation available, you must first follow one of these steps:</span></span>

<span data-ttu-id="6443a-111">Se l'ordine o le righe d'ordine selezionati non sono configurati per il ritiro in quel negozio specifico, o se l'ordine è già stato completamente ritirato, l'operazione **Preleva** non sarà disponibile.</span><span class="sxs-lookup"><span data-stu-id="6443a-111">If the selected order or order lines aren't configured for pickup at that specific store, or if the order has already been fully picked up, the **Pick up** operation will be unavailable.</span></span>

<span data-ttu-id="6443a-113">In Microsoft Dynamics 365 Commerce versione 10.0.17 e successive, la funzionalità **Esperienza utente migliorata per l'elaborazione degli ordini di ritiro nel punto vendita** può essere attivata tramite la gestione delle funzionalità in Commerce headquarters.</span><span class="sxs-lookup"><span data-stu-id="6443a-113">In Microsoft Dynamics 365 Commerce version 10.0.17 and later, the **Improved user experience for pick up order processing in Point of Sale** feature can be turned on through Feature management in Commerce headquarters.</span></span> <span data-ttu-id="6443a-114">Se questa funzione è disattivata, gli utenti non possono selezionare le quantità di prelievo.</span><span class="sxs-lookup"><span data-stu-id="6443a-114">If this feature is turned off, users can't select pickup quantities.</span></span> <span data-ttu-id="6443a-115">Per impostazione predefinita, la quantità completa ordinata per la riga è la quantità che verrà prelevata.</span><span class="sxs-lookup"><span data-stu-id="6443a-115">By default, the full quantity that was ordered for the line is the quantity that will be picked up.</span></span> <span data-ttu-id="6443a-116">Questa esperienza può essere problematica, perché gli utenti potrebbero dimenticare di selezionare alcuni articoli per il ritiro quando effettuano il ritiro tramite l'evasione dell'ordine.</span><span class="sxs-lookup"><span data-stu-id="6443a-116">This experience can be problematic, because users might forget to select some items for pickup when they perform the pickup through order fulfillment.</span></span>

<span data-ttu-id="6443a-117">La funzionalità **Esperienza utente migliorata per l'elaborazione degli ordini di ritiro nel punto vendita** offre agli utenti un maggiore controllo sulla selezione dei prodotti che verranno prelevati e sulla quantità di tali prodotti che verranno ritirati.</span><span class="sxs-lookup"><span data-stu-id="6443a-117">The **Improved user experience for pick up order processing in Point of Sale** feature gives users more control over the selection of products that will be picked up and the quantity of those products that will be picked up.</span></span> <span data-ttu-id="6443a-118">Gli utenti non devono selezionare ogni riga dell'ordine di vendita nella pagina di evasione dell'ordine prima di selezionare **Preleva**.</span><span class="sxs-lookup"><span data-stu-id="6443a-118">Users don't have to select every line of the sales order on the order fulfillment page before they select **Pick up**.</span></span> <span data-ttu-id="6443a-119">Verranno visualizzati tutti gli articoli che possono essere ritirati.</span><span class="sxs-lookup"><span data-stu-id="6443a-119">All the items that can be picked up will be shown.</span></span> <span data-ttu-id="6443a-120">Gli utenti possono specificare più righe per il ritiro anche se è selezionata una sola riga di prodotto.</span><span class="sxs-lookup"><span data-stu-id="6443a-120">Users can specify multiple lines for pickup even if only one product line is selected.</span></span>

<span data-ttu-id="6443a-121">Quando la funzionalità **Esperienza utente migliorata per l'elaborazione degli ordini di ritiro nel punto vendita** è attivata e selezioni l'operazione **Preleva**, la finestra di dialogo **Preleva** viene visualizzata.</span><span class="sxs-lookup"><span data-stu-id="6443a-121">When the **Improve user experience for pick up order processing in Point of Sale** feature is turned on, and you select the **Pick up** operation, the **Pick up** dialog box appears.</span></span> <span data-ttu-id="6443a-122">Qui puoi selezionare gli articoli e le quantità che verranno prelevate.</span><span class="sxs-lookup"><span data-stu-id="6443a-122">There, you can select the items and quantities that will be picked up.</span></span> <span data-ttu-id="6443a-123">Per impostazione predefinita, qualsiasi quantità ordinata con inventario in uno stato prelevato o imballato è considerata idonea per il ritiro.</span><span class="sxs-lookup"><span data-stu-id="6443a-123">By default, any ordered quantity that has inventory in a picked or packed state is considered eligible for pickup.</span></span> <span data-ttu-id="6443a-124">Per impostazione predefinita, tale quantità è impostata come quantità di prelievo.</span><span class="sxs-lookup"><span data-stu-id="6443a-124">By default, that quantity is set as the pickup quantity.</span></span> <span data-ttu-id="6443a-125">È possibile modificare la quantità immessa, a condizione che la quantità non sia 0 (zero) e non superi la quantità totale aperta (ovvero non fatturata) per la riga selezionata.</span><span class="sxs-lookup"><span data-stu-id="6443a-125">You can change the quantity that was entered, provided that the quantity isn't 0 (zero) and doesn't exceed the total open (that is, non-invoiced) quantity for the selected line.</span></span>

<span data-ttu-id="6443a-129">Nella pagina della transazione, il sistema calcola gli importi dovuti calcolando il totale dovuto per gli articoli di ritiro selezionati e quindi sottraendo eventuali depositi precedentemente applicati o pagamenti con carta di credito autorizzati.</span><span class="sxs-lookup"><span data-stu-id="6443a-129">On the transaction page, the system calculates the amounts that are due by calculating the total that is due for the selected pickup items and then subtracting any previously applied deposits or authorized credit card payments.</span></span> <span data-ttu-id="6443a-130">È necessario elaborare il pagamento per completare la transazione di ritiro.</span><span class="sxs-lookup"><span data-stu-id="6443a-130">You must process payment to complete the pickup transaction.</span></span> <span data-ttu-id="6443a-131">Se il [layout dello schermo](pos-screen-layouts.md) della pagina della transazione è configurato in modo da includere l'operazione **Concludi transazione** e nessun importo è dovuto, è possibile completare la transazione senza selezionare un metodo di pagamento.</span><span class="sxs-lookup"><span data-stu-id="6443a-131">If the [screen layout](pos-screen-layouts.md) of the transaction page is configured so that it includes the **Conclude transaction** operation, and no amount is due, you can complete the transaction without selecting a payment method.</span></span> <span data-ttu-id="6443a-132">Se l'operazione **Concludi transazione** non è disponibile, è possibile selezionare il collegamento **Importo di $ 0,00 dovuto** nel riquadro **Totali** per concludere la transazione senza dover selezionare un metodo di pagamento.</span><span class="sxs-lookup"><span data-stu-id="6443a-132">If the **Conclude transaction** operation isn't available, you can select the **$0.00 amount due** link in the **Totals** pane to conclude the transaction without having to select a payment method.</span></span>
