## <a name="serial-number-configurations"></a><span data-ttu-id="dc495-109">Configurazioni dei numeri di serie</span><span class="sxs-lookup"><span data-stu-id="dc495-109">Serial number configurations</span></span>

<span data-ttu-id="dc495-110">Un articolo viene considerato serializzato se dispone dell'assegnazione di un gruppo di dimensioni di tracciabilità impostato per consentire i numeri di serie.</span><span class="sxs-lookup"><span data-stu-id="dc495-110">An item is considered a serialized item if it's assigned a tracking dimension group that is set up to allow for serial numbers.</span></span> <span data-ttu-id="dc495-111">In Commerce headquarters selezionare l'opzione **Attiva** nella pagina **Gruppi di dimensioni di tracciabilità** per abilitare i numeri di serie per il processo di inventario.</span><span class="sxs-lookup"><span data-stu-id="dc495-111">In Commerce headquarters, on the **Tracking dimension groups** page, select the **Active** option to enable serial numbers for the inventory process.</span></span> <span data-ttu-id="dc495-112">Per abilitare i numeri di serie per il processo di vendita, selezionare l'opzione **Attiva in processo di vendita**.</span><span class="sxs-lookup"><span data-stu-id="dc495-112">To enable serial numbers for the sales process, select the **Active in sales process** option.</span></span>

<span data-ttu-id="dc495-113">Nella Scheda dettaglio **Dimensioni di tracciabilità**, se l'opzione **Entrata non specificata consentita** è attivata, il numero di serie è un input opzionale durante il processo di entrata in magazzino.</span><span class="sxs-lookup"><span data-stu-id="dc495-113">On the **Tracking dimensions** FastTab, if the **Blank receipt allowed** option is turned on, the serial number is an optional input during the inventory receipt process.</span></span> <span data-ttu-id="dc495-114">Se l'opzione è disattivata, è richiesto il numero di serie.</span><span class="sxs-lookup"><span data-stu-id="dc495-114">If the option is turned off, the serial number is required.</span></span>

<span data-ttu-id="dc495-128">Il campo **Stato** nella pagina **Gestione numeri di serie** fornisce informazioni sulla fase corrente in cui si trova ciascun numero di serie:</span><span class="sxs-lookup"><span data-stu-id="dc495-128">The **Status** field on the **Serial number management** page provides information about the current stage that each serial number is in:</span></span>

- <span data-ttu-id="dc495-130">**In registrazione**: il numero di serie è stato registrato e salvato in locale nel database del canale del negozio oppure il numero di serie preregistrato è stato convalidato.</span><span class="sxs-lookup"><span data-stu-id="dc495-130">**Registering** – The serial number has been registered and saved locally to the store's channel database, or the preregistered serial number has been validated.</span></span> <span data-ttu-id="dc495-131">Solo i numeri di serie con stato **In registrazione** verranno inviati a Commerce headquarters al termine della ricezione.</span><span class="sxs-lookup"><span data-stu-id="dc495-131">Only serial numbers that have a status of **Registering** will be submitted to Commerce headquarters when you finish receiving.</span></span>
