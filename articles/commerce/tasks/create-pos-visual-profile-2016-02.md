# <a name="create-point-of-sale-pos-visual-profiles"></a>Creare profili di visivi di POS

[!include [banner](../includes/banner.md)]

In questa procedura vengono descritti i passaggi per creare un profilo visivo di un nuovo punto vendita (POS). Il profilo visivo contiene informazioni di base che determinano l'aspetto dei registratori POS. È possibile creare diversi profili visivi e assegnare quali profili specifici eseguire su registri specifici. Questa procedura utilizza la società di dati dimostrativi USRT.

1. Vai a **Retail e Commerce > Impostazione canale > Impostazione POS > Profili POS > Profili visivi**.
2. Fare clic su **Nuovo**.
3. Nel campo **Numero profilo**, digita un valore.
4. Digitare un valore nel campo **Descrizione**
5. Nel campo **Tipo di applicazione** fai clic sul pulsante a discesa per aprire la ricerca.
6. Nell'elenco fare clic sul collegamento nella riga selezionata.
7. Nel campo **Tema** fai clic sul pulsante a discesa per aprire la ricerca.
8. Nell'elenco fare clic sul collegamento nella riga selezionata.
9. Nel campo **Colore accento** fai clic sul pulsante a discesa per aprire la ricerca.
10. Nell'elenco trovare e selezionare il record desiderato.
11. Nell'elenco fare clic sul collegamento nella riga selezionata.
12. Attiva l'espansione della sezione **Sfondo accesso**.
13. Nel campo **ID immagine orizzontale**, seleziona o immetti un ID immagine.
14. Nel campo **ID immagine verticale**, seleziona o immetti un ID immagine.
15. Attiva l'espansione della sezione **Sfondo**.
16. RequestPopup a ID immagine.
17. Nell'elenco fare clic sul collegamento nella riga selezionata.
18. Fare clic su **Salva**.
