# <a name="crop-images"></a><span data-ttu-id="bfead-103">Tagliare immagini</span><span class="sxs-lookup"><span data-stu-id="bfead-103">Crop images</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="bfead-104">In questo argomento viene descritto come tagliare immagini in Creazione di siti Web Microsoft Dynamics 365 Commerce.</span><span class="sxs-lookup"><span data-stu-id="bfead-104">This topic describes how to crop images in Microsoft Dynamics 365 Commerce site builder.</span></span>

<span data-ttu-id="bfead-105">La libreria multimediale di Creazione di siti Web Commerce consente di ritagliare le immagini per ottimizzarle per diversi tipi di riquadri di visualizzazione e finestre.</span><span class="sxs-lookup"><span data-stu-id="bfead-105">The Commerce site builder Media Library allows you to crop images to optimize them for different module types and viewports.</span></span>

## <a name="crop-an-image"></a><span data-ttu-id="bfead-106">Ritagliare un'immagine</span><span class="sxs-lookup"><span data-stu-id="bfead-106">Crop an image</span></span>

<span data-ttu-id="bfead-107">Per ritagliare un'immagine in Creazione di siti Web, attenersi alla seguente procedura.</span><span class="sxs-lookup"><span data-stu-id="bfead-107">To crop an image in site builder, follow these steps.</span></span>

1. <span data-ttu-id="bfead-108">Nel riquadro di navigazione a sinistra di Creazione di siti Web Commerce, selezionare **Libreria multimediale**.</span><span class="sxs-lookup"><span data-stu-id="bfead-108">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="bfead-109">Nella finestra principale, selezionare l'immagine da modificare.</span><span class="sxs-lookup"><span data-stu-id="bfead-109">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="bfead-110">Nella barra dei comandi, selezionare **Modifica**.</span><span class="sxs-lookup"><span data-stu-id="bfead-110">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="bfead-111">Selezionare l'immagine per abilitare la **Modalità di modifica**.</span><span class="sxs-lookup"><span data-stu-id="bfead-111">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="bfead-112">Nella **Modalità di modifica**, selezionare **Modifica visualizzazione per modulo**.</span><span class="sxs-lookup"><span data-stu-id="bfead-112">Under **Edit Mode**, select **Edit View by Module**.</span></span>
1. <span data-ttu-id="bfead-113">Dal menu a discesa **Modulo**, selezionare il tipo di modulo.</span><span class="sxs-lookup"><span data-stu-id="bfead-113">From the **Module** drop-down menu, select the module type.</span></span>
1. <span data-ttu-id="bfead-114">Dal menu a discesa **Tipo di visualizzazione**, selezionare il tipo di visualizzazione.</span><span class="sxs-lookup"><span data-stu-id="bfead-114">From the **View type** drop-down menu, select the view type.</span></span>
1. <span data-ttu-id="bfead-115">Dal menu a discesa **Posizione**, selezionare la posizione dell'immagine.</span><span class="sxs-lookup"><span data-stu-id="bfead-115">From the **Placement** drop-down menu, select the image placement.</span></span>
1. <span data-ttu-id="bfead-116">Dal menu a discesa **Riquadro di visualizzazione**, selezionare la dimensione del riquadro di visualizzazione.</span><span class="sxs-lookup"><span data-stu-id="bfead-116">From the **Viewport** drop-down menu, select the viewport size.</span></span>
1. <span data-ttu-id="bfead-117">L'immagine viene sovrapposta all'area che rappresenta l'area di ritaglio.</span><span class="sxs-lookup"><span data-stu-id="bfead-117">The image is overlaid with the area representing the crop region.</span></span> <span data-ttu-id="bfead-118">Spostare e ridimensionare l'area di ritaglio secondo necessità.</span><span class="sxs-lookup"><span data-stu-id="bfead-118">Move and resize the crop region as needed.</span></span> <span data-ttu-id="bfead-119">Le proporzioni verranno mantenute automaticamente.</span><span class="sxs-lookup"><span data-stu-id="bfead-119">The aspect ratio will be maintained automatically.</span></span>
1. <span data-ttu-id="bfead-120">Al termine, selezionare nella barra dei comandi **Salva**, quindi selezionare **Fine modifica**.</span><span class="sxs-lookup"><span data-stu-id="bfead-120">When you're done, on the command bar, select **Save**, and then select **Finish editing**.</span></span> 

<span data-ttu-id="bfead-121">Al termine del ritaglio personalizzato, le modifiche alle immagini avranno effetto quasi immediatamente.</span><span class="sxs-lookup"><span data-stu-id="bfead-121">After custom cropping is completed, image modifications will take effect almost immediately.</span></span>
