# <a name="customize-image-focal-points"></a><span data-ttu-id="50623-103">Personalizzare i punti focali dell'immagine</span><span class="sxs-lookup"><span data-stu-id="50623-103">Customize image focal points</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="50623-104">In questo argomento viene descritto come personalizzare i punti focali dell'immagine in Creazione di siti Web Microsoft Dynamics 365 Commerce.</span><span class="sxs-lookup"><span data-stu-id="50623-104">This topic describes how to customize image focal points in Microsoft Dynamics 365 Commerce site builder.</span></span>

## <a name="overview"></a><span data-ttu-id="50623-105">Panoramica</span><span class="sxs-lookup"><span data-stu-id="50623-105">Overview</span></span>

<span data-ttu-id="50623-106">Quando un'immagine viene caricata nella libreria multimediale di Creazione di siti Web Commerce, il sistema tenta di determinare il punto focale dell'immagine.</span><span class="sxs-lookup"><span data-stu-id="50623-106">When an image is uploaded to the Commerce site builder Media Library, the system attempts to determine the focal point of the image.</span></span> <span data-ttu-id="50623-107">Ad esempio, se l'immagine contiene una persona, il sistema imposterà il punto focale sul viso della persona per impostazione predefinita.</span><span class="sxs-lookup"><span data-stu-id="50623-107">For example, if the image has a person on it, the system will set the focal point to the face of the person by default.</span></span> <span data-ttu-id="50623-108">Nella maggior parte dei casi il punto focale impostato automaticamente funziona bene per tutti i riquadri di visualizzazione, ma a volte si potrebbe voler regolare il punto focale per garantire che una parte specifica dell'immagine sia sempre visibile.</span><span class="sxs-lookup"><span data-stu-id="50623-108">In most cases the automatically set focal point works well for all viewports, but sometimes you may want to adjust the focal point to ensure that a specific part of the image is always visible.</span></span>

### <a name="define-a-custom-focal-point-for-an-image"></a><span data-ttu-id="50623-109">Definire un punto focale personalizzato per un'immagine</span><span class="sxs-lookup"><span data-stu-id="50623-109">Define a custom focal point for an image</span></span>

<span data-ttu-id="50623-110">Per definire un punto focale personalizzato per un'immagine, attenersi alla seguente procedura.</span><span class="sxs-lookup"><span data-stu-id="50623-110">To define a custom focal point for an image, follow these steps.</span></span>

1. <span data-ttu-id="50623-111">Nel riquadro di navigazione a sinistra di Creazione di siti Web Commerce, selezionare **Libreria multimediale**.</span><span class="sxs-lookup"><span data-stu-id="50623-111">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="50623-112">Nella finestra principale, selezionare l'immagine da modificare.</span><span class="sxs-lookup"><span data-stu-id="50623-112">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="50623-113">Nella barra dei comandi, selezionare **Modifica**.</span><span class="sxs-lookup"><span data-stu-id="50623-113">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="50623-114">Selezionare l'immagine per abilitare la **Modalità di modifica**.</span><span class="sxs-lookup"><span data-stu-id="50623-114">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="50623-115">Sotto **Modalità di modifica**, selezionare **Cambia punto focale**.</span><span class="sxs-lookup"><span data-stu-id="50623-115">Under **Edit Mode**, select **Change Focal Point**.</span></span> <span data-ttu-id="50623-116">Un controllo circolare del punto focale appare sull'immagine.</span><span class="sxs-lookup"><span data-stu-id="50623-116">A circular focal point control appears over the image.</span></span>
1. <span data-ttu-id="50623-117">Selezionare il controllo del punto focale per spostarlo sul punto focale desiderato.</span><span class="sxs-lookup"><span data-stu-id="50623-117">Select the focal point control to move it over the desired focal point.</span></span>
1. <span data-ttu-id="50623-118">Al termine, selezionare nella barra dei comandi **Salva**, quindi selezionare **Fine modifica**.</span><span class="sxs-lookup"><span data-stu-id="50623-118">When you're done, on the command bar select **Save**, and then select **Finish editing**.</span></span>
