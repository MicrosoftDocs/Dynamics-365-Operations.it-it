# <a name="add-product-recommendation-lists-to-e-commerce-pages"></a>Aggiungere elenchi di suggerimenti sul prodotto alle pagine di e-Commerce

[!include [banner](includes/preview-banner.md)]
[!include [banner](includes/banner.md)]

In questo argomento viene descritto come aggiungere un elenco di suggerimenti sul prodotto a una pagina di e-Commerce.

1. Da una pagina di creazione di e-Commerce esistente, selezionare **Aggiungi modulo**.
2. Cercare il modulo **Raccolta prodotti**.
3. Nella procedura guidata del modulo, selezionare l'elenco da aggiungere alla pagina.
4. Salvare le modifiche.
