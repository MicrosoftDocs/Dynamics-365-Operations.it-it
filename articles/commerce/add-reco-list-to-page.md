# <a name="add-recommendation-lists-to-a-site"></a>Aggiungere elenchi di suggerimenti a un sito


[!include [banner](includes/banner.md)]

In questo argomento viene descritto come aggiungere un elenco di suggerimenti sul prodotto a una pagina di e-Commerce.

1. Da una pagina di Creazione di siti di e-Commerce esistente, selezionare **Aggiungi modulo**.
2. Cercare il modulo **Raccolta prodotti**.
3. Nella procedura guidata del modulo, selezionare l'elenco da aggiungere alla pagina.
4. Salvare le modifiche.
