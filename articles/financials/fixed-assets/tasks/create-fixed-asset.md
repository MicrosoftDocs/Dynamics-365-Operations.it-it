<span data-ttu-id="f988d-104">Questa guida attività utilizza la società dimostrativa USMF.</span><span class="sxs-lookup"><span data-stu-id="f988d-104">This task guide uses the USMF demo company.</span></span>  <span data-ttu-id="f988d-105">Consentirà di creare un nuovo cespite utilizzando la pagina elenco Cespite.</span><span class="sxs-lookup"><span data-stu-id="f988d-105">It will create a new fixed asset using the Fixed asset list page.</span></span>

1. <span data-ttu-id="f988d-106">Passare a Cespiti > Cespiti > Cespiti.</span><span class="sxs-lookup"><span data-stu-id="f988d-106">Go to Fixed assets > Fixed assets > Fixed assets.</span></span>
2. <span data-ttu-id="f988d-107">Fare clic su Nuovo.</span><span class="sxs-lookup"><span data-stu-id="f988d-107">Click New.</span></span>

    * <span data-ttu-id="f988d-117">Immettere le informazioni aggiuntive relative ai libri rimanenti necessarie per l'azienda.</span><span class="sxs-lookup"><span data-stu-id="f988d-117">Enter the additional information that your business needs for the remaining books.</span></span>
