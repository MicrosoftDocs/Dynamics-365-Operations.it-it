# <a name="change-depreciation-conventions-for-multiple-fixed-assets"></a>Cambiare le convenzioni di ammortamento per più cespiti

[!include [task guide banner](../../includes/task-guide-banner.md)]

Tale attività consente di aggiornare la convenzione di ammortamento per un gruppo di cespiti specificato. Questa guida attività utilizza la società dimostrativa USMF.

1. Andare a Cespiti > Attività periodiche > Aggiornamento di massa
2. Nel campo Registro beni ammortizzabili fare clic sul pulsante a discesa per aprire la ricerca.
3. Nell'elenco fare clic sul collegamento nella riga selezionata.
4. Nel campo Inizio messa in servizio immettere una data.
5. Nel campo Fine messa in servizio immettere una data.
    * Solo i cespiti appartenenti al registro beni ammortizzabili selezionato e messi in servizio nel periodo compreso tra queste date verranno aggiornati.  
6. Nel campo Convenzione di ammortamento corrente selezionare un'opzione.
    * Solo i cespiti con la convenzione di ammortamento corrente verranno aggiornati.  
7. Nel campo Nuova convenzione di ammortamento selezionare un'opzione.
    * Verificare che il report venga stampato nella destinazione selezionata.  
8. Espandere la sezione Record da includere.
9. Fare clic su Filtro.
10. Selezionare il gruppo cespite nell'elenco.
11. Nel campo Criteri fare clic sul pulsante a discesa per aprire la ricerca.
12. Selezionare il gruppo cespite desiderato.
13. Nell'elenco fare clic sul collegamento nella riga selezionata.
14. Fare clic su OK.
15. Fare clic su OK.
    *  I risultati del processo vengono visualizzati nel report Aggiornamento di massa.
