# <a name="create-ledger-accrual-transactions"></a><span data-ttu-id="568cd-103">Creare transazioni di attribuzione per competenza in contabilità generale</span><span class="sxs-lookup"><span data-stu-id="568cd-103">Create ledger accrual transactions</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="568cd-104">Questa guida di attività descrive la generazione di transazioni di attribuzione per competenza in contabilità generale basate su schemi di attribuzione per competenza.</span><span class="sxs-lookup"><span data-stu-id="568cd-104">This task guide steps through generating ledger accrual transactions that are based on accrual schemes</span></span>

1. <span data-ttu-id="568cd-105">Fare clic su Contabilità generale > Scritture contabili > Giornali di registrazione generali.</span><span class="sxs-lookup"><span data-stu-id="568cd-105">Go to General ledger > Journal entries > General journals.</span></span>
2. <span data-ttu-id="568cd-106">Nell'elenco, trovare e selezionare il giornale di registrazione desiderato o crearne uno nuovo.</span><span class="sxs-lookup"><span data-stu-id="568cd-106">In the list, find and select the desired journal or create a new one.</span></span>
3. <span data-ttu-id="568cd-107">Fare clic per seguire il collegamento nel campo Numero batch giornale di registrazione.</span><span class="sxs-lookup"><span data-stu-id="568cd-107">Click to follow the link in the Journal batch number field.</span></span>
4. <span data-ttu-id="568cd-108">Nell'elenco contrassegnare la riga selezionata.</span><span class="sxs-lookup"><span data-stu-id="568cd-108">In the list, mark the selected row.</span></span>
5. <span data-ttu-id="568cd-109">Nel campo Conto, specificare i valori desiderati.</span><span class="sxs-lookup"><span data-stu-id="568cd-109">In the Account field, specify the desired values.</span></span>
    * <span data-ttu-id="568cd-110">In questo esempio viene definita la spesa dell'assicurazione.</span><span class="sxs-lookup"><span data-stu-id="568cd-110">In this example, we are defining the expense for the insurance.</span></span> <span data-ttu-id="568cd-111">Diventerà un importo di spesa periodico.</span><span class="sxs-lookup"><span data-stu-id="568cd-111">It will become a periodic expense amount.</span></span>  
6. <span data-ttu-id="568cd-112">Nel campo Descrizione digitare un valore.</span><span class="sxs-lookup"><span data-stu-id="568cd-112">In the Description field, type a value.</span></span>
7. <span data-ttu-id="568cd-113">Nel campo Dare immettere un numero.</span><span class="sxs-lookup"><span data-stu-id="568cd-113">In the Debit field, enter a number.</span></span>
8. <span data-ttu-id="568cd-114">Nel campo Conto di contropartita, specificare i valori desiderati.</span><span class="sxs-lookup"><span data-stu-id="568cd-114">In the Offset account field, specify the desired values.</span></span>
9. <span data-ttu-id="568cd-115">Fare clic su Funzioni.</span><span class="sxs-lookup"><span data-stu-id="568cd-115">Click Functions.</span></span>
10. <span data-ttu-id="568cd-116">Fare clic su Ratei in contabilità generale.</span><span class="sxs-lookup"><span data-stu-id="568cd-116">Click Ledger accruals.</span></span>
11. <span data-ttu-id="568cd-117">Nel campo Identificazione attribuzione per competenza fare clic sul pulsante a discesa per aprire la ricerca.</span><span class="sxs-lookup"><span data-stu-id="568cd-117">In the Accrual identification field, click the drop-down button to open the lookup.</span></span>
12. <span data-ttu-id="568cd-118">Nell'elenco, trovare e selezionare lo schema di attribuzione per competenza che si desidera applicare.</span><span class="sxs-lookup"><span data-stu-id="568cd-118">In the list, find and select the accrual scheme you want to apply.</span></span>
13. <span data-ttu-id="568cd-119">Nell'elenco fare clic sul collegamento nella riga selezionata.</span><span class="sxs-lookup"><span data-stu-id="568cd-119">In the list, click the link in the selected row.</span></span>
14. <span data-ttu-id="568cd-120">Nel campo Data di inizio, immettere una data.</span><span class="sxs-lookup"><span data-stu-id="568cd-120">In the Start date field, enter a date.</span></span>
15. <span data-ttu-id="568cd-121">Fare clic su Transazioni.</span><span class="sxs-lookup"><span data-stu-id="568cd-121">Click Transactions.</span></span>
16. <span data-ttu-id="568cd-122">Chiudere la pagina.</span><span class="sxs-lookup"><span data-stu-id="568cd-122">Close the page.</span></span>
17. <span data-ttu-id="568cd-123">Fare clic su OK.</span><span class="sxs-lookup"><span data-stu-id="568cd-123">Click OK.</span></span>
18. <span data-ttu-id="568cd-124">Fare clic su Registra.</span><span class="sxs-lookup"><span data-stu-id="568cd-124">Click Post.</span></span>
