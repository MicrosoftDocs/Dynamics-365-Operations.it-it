[!include [task guide banner](../../includes/task-guide-banner.md)]

3. <span data-ttu-id="69fd6-107">Fare clic per seguire il collegamento nel campo Numero batch giornale di registrazione.</span><span class="sxs-lookup"><span data-stu-id="69fd6-107">Click to follow the link in the Journal batch number field.</span></span>
4. <span data-ttu-id="69fd6-108">Nell'elenco contrassegnare la riga selezionata.</span><span class="sxs-lookup"><span data-stu-id="69fd6-108">In the list, mark the selected row.</span></span>
5. <span data-ttu-id="69fd6-109">Nel campo Conto, specificare i valori desiderati.</span><span class="sxs-lookup"><span data-stu-id="69fd6-109">In the Account field, specify the desired values.</span></span>
    * <span data-ttu-id="69fd6-110">In questo esempio viene definita la spesa dell'assicurazione.</span><span class="sxs-lookup"><span data-stu-id="69fd6-110">In this example, we are defining the expense for the insurance.</span></span> <span data-ttu-id="69fd6-111">Verrà impostata come importo di spesa periodico.</span><span class="sxs-lookup"><span data-stu-id="69fd6-111">It will be come periodic expense amount.</span></span>  
6. <span data-ttu-id="69fd6-112">Nel campo Descrizione digitare un valore.</span><span class="sxs-lookup"><span data-stu-id="69fd6-112">In the Description field, type a value.</span></span>
7. <span data-ttu-id="69fd6-113">Nel campo Dare immettere un numero.</span><span class="sxs-lookup"><span data-stu-id="69fd6-113">In the Debit field, enter a number.</span></span>
8. <span data-ttu-id="69fd6-114">Nel campo Conto di contropartita, specificare i valori desiderati.</span><span class="sxs-lookup"><span data-stu-id="69fd6-114">In the Offset account field, specify the desired values.</span></span>
9. <span data-ttu-id="69fd6-115">Fare clic su Funzioni.</span><span class="sxs-lookup"><span data-stu-id="69fd6-115">Click Functions.</span></span>
10. <span data-ttu-id="69fd6-116">Fare clic su Ratei in contabilità generale.</span><span class="sxs-lookup"><span data-stu-id="69fd6-116">Click Ledger accruals.</span></span>
11. <span data-ttu-id="69fd6-117">Nel campo Identificazione attribuzione per competenza fare clic sul pulsante a discesa per aprire la ricerca.</span><span class="sxs-lookup"><span data-stu-id="69fd6-117">In the Accrual identification field, click the drop-down button to open the lookup.</span></span>

14. <span data-ttu-id="69fd6-120">Nel campo Data di inizio, immettere una data.</span><span class="sxs-lookup"><span data-stu-id="69fd6-120">In the Start date field, enter a date.</span></span>

16. <span data-ttu-id="69fd6-122">Chiudere la pagina.</span><span class="sxs-lookup"><span data-stu-id="69fd6-122">Close the page.</span></span>
17. <span data-ttu-id="69fd6-123">Fare clic su OK.</span><span class="sxs-lookup"><span data-stu-id="69fd6-123">Click OK.</span></span>
18. <span data-ttu-id="69fd6-124">Fare clic su Registra.</span><span class="sxs-lookup"><span data-stu-id="69fd6-124">Click Post.</span></span>
