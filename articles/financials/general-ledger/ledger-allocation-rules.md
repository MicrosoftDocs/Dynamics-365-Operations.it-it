# <a name="ledger-allocation-rules"></a><span data-ttu-id="3ee6b-104">Regole di allocazione contabile</span><span class="sxs-lookup"><span data-stu-id="3ee6b-104">Ledger allocation rules</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="3ee6b-105">Questo articolo fornisce informazioni sulle regole di allocazione della contabilità generale.</span><span class="sxs-lookup"><span data-stu-id="3ee6b-105">This article provides information about ledger allocation rules.</span></span> <span data-ttu-id="3ee6b-106">Descrive i vari componenti delle regole di allocazione e i metodi di allocazione che possono essere utilizzati per tali regole.</span><span class="sxs-lookup"><span data-stu-id="3ee6b-106">It describes the various components of these allocation rules and the allocation methods that can be used for them.</span></span>

<span data-ttu-id="3ee6b-107">Le regole di allocazione contabile vengono utilizzate per calcolare e generare automaticamente i giornali di registrazione e le voci contabili per l'allocazione dei saldi contabili o degli importi fissi.</span><span class="sxs-lookup"><span data-stu-id="3ee6b-107">Ledger allocation rules are used to automatically calculate and generate allocation journals and account entries for the allocation of ledger balances or fixed amounts.</span></span> <span data-ttu-id="3ee6b-108">I metodi di allocazione possono essere variabili o fissi.</span><span class="sxs-lookup"><span data-stu-id="3ee6b-108">Allocation methods can be variable or fixed.</span></span> <span data-ttu-id="3ee6b-109">I seguenti metodi di allocazione possono essere utilizzati per le regole di allocazione contabile:</span><span class="sxs-lookup"><span data-stu-id="3ee6b-109">The following allocation methods can be used for ledger allocation rules:</span></span>

-   <span data-ttu-id="3ee6b-110">**Base**: il metodo variabile viene utilizzato quando l'allocazione dipende dal saldo contabile, in base ai criteri di filtro.</span><span class="sxs-lookup"><span data-stu-id="3ee6b-110">**Basis** – This variable method is used when the allocation depends on the actual ledger balance, based on filter criteria.</span></span> <span data-ttu-id="3ee6b-111">È ad esempio possibile allocare le spese relative alla pubblicità in base alle vendite di ciascun reparto in rapporto alle vendite totali di tutti i reparti.</span><span class="sxs-lookup"><span data-stu-id="3ee6b-111">For example, advertising expenses can be allocated based on each department's sales in proportion to the total departmental sales.</span></span>
-   <span data-ttu-id="3ee6b-112">**Percentuale fissa** e **Peso fisso**: per questi metodi la percentuale o il peso di allocazione è definito direttamente per la regola.</span><span class="sxs-lookup"><span data-stu-id="3ee6b-112">**Fixed percentage** and **Fixed weight** – For these methods, the allocation percentage or weight is defined directly for the rule.</span></span> <span data-ttu-id="3ee6b-113">Ad esempio, le spese di pubblicità possono essere allocate in modo che il reparto A ne riceva il 70 percento e il reparto B il 30 percento.</span><span class="sxs-lookup"><span data-stu-id="3ee6b-113">For example, advertising expenses can be allocated so that Department A receives 70 percent of the advertising expense and Department B receives 30 percent.</span></span>
-   <span data-ttu-id="3ee6b-114">**Uniformemente**: il metodo distribuisce equamente l'importo a ogni destinazione definita.</span><span class="sxs-lookup"><span data-stu-id="3ee6b-114">**Equally** – This method distributes the amount equally to each defined destination.</span></span> <span data-ttu-id="3ee6b-115">Ad esempio, se vengono definite destinazioni per il reparto A e il reparto B, le spese di pubblicità possono essere allocate in modo che sia il reparto A che il reparto B ricevano il 50 percento delle spese di pubblicità.</span><span class="sxs-lookup"><span data-stu-id="3ee6b-115">For example, if destinations are defined for Department A and Department B, advertising expenses can be allocated so that both Department A and Department B receive 50 percent of the advertising expense.</span></span>

<span data-ttu-id="3ee6b-116">Se Base è il metodo di allocazione utilizzato per una regola di allocazione, è necessario definire anche regole di base separate di allocazione contabile.</span><span class="sxs-lookup"><span data-stu-id="3ee6b-116">If Basis is used as the allocation method for an allocation rule, you must also define separate ledger allocation basis rules.</span></span> <span data-ttu-id="3ee6b-117">Il processo "Elabora richiesta di allocazione" consente agli utenti di elaborare la regola di allocazione contabile e di visualizzare in anteprima gli inserimenti risultanti nel giornale di registrazione allocazioni prima di registrare o eliminare le allocazioni calcolate.</span><span class="sxs-lookup"><span data-stu-id="3ee6b-117">The "Process allocation request" process lets users process the ledger allocation rule and preview the resulting allocation journal entries before they either post or delete the calculated allocations.</span></span>

## <a name="components-of-ledger-allocation-rules"></a><span data-ttu-id="3ee6b-118">Componenti delle regole di allocazione contabile</span><span class="sxs-lookup"><span data-stu-id="3ee6b-118">Components of ledger allocation rules</span></span>
<span data-ttu-id="3ee6b-119">Le regole di allocazione sono composte da quattro componenti: generale, origine, destinazione e contropartita.</span><span class="sxs-lookup"><span data-stu-id="3ee6b-119">Each allocation rule has four components: general, source, destination, and offset.</span></span> <span data-ttu-id="3ee6b-120">Un componente aggiuntivo, regole di base dell'allocazione contabile, è necessario se Base viene utilizzato come metodo di allocazione.</span><span class="sxs-lookup"><span data-stu-id="3ee6b-120">An additional component, ledger allocation bases rules, is required if Basis is used as the allocation method.</span></span> <span data-ttu-id="3ee6b-121">Ciascun componente corrisponde a un'informazione critica per l'elaborazione delle allocazioni.</span><span class="sxs-lookup"><span data-stu-id="3ee6b-121">Each component provides a critical piece of the information that is required in order to process allocations.</span></span>

-   <span data-ttu-id="3ee6b-122">**Generale**: in questo componente vengono specificate alcune opzioni, quali il metodo di allocazione, le impostazioni delle regole interaziendali e l'impostazione della regola come attiva o inattiva.</span><span class="sxs-lookup"><span data-stu-id="3ee6b-122">**General** – This component is where the user specifies options such as the allocation method, intercompany rule settings, and whether the rule is active.</span></span>
-   <span data-ttu-id="3ee6b-123">**Origine**: in questo componente l'utente specifica i dati di origine per l'allocazione.</span><span class="sxs-lookup"><span data-stu-id="3ee6b-123">**Source** – This component is where the user specifies the source data for the allocation.</span></span> <span data-ttu-id="3ee6b-124">L'allocazione può essere basata sui saldi contabili (**Origine dati** =  **Contabilità generale**) o gli importi fissi (**Origine dati** =  **Valore fisso**).</span><span class="sxs-lookup"><span data-stu-id="3ee6b-124">Allocation can be based on ledger balances (**Data source** = **Ledger**) or fixed amounts (**Data source** = **Fixed value**).</span></span> <span data-ttu-id="3ee6b-125">Quando l'opzione **Origine dati** è impostata su **Contabilità generale**, i criteri di filtro dell'origine devono essere definiti per la regola di allocazione contabile (ad esempio, per le spese di pubblicità).</span><span class="sxs-lookup"><span data-stu-id="3ee6b-125">When **Data source** is set to **Ledger**, source filter criteria must be defined for the ledger allocation rule (for example, for the advertising expenses).</span></span>
-   <span data-ttu-id="3ee6b-126">**Destinazione**: in questo componente vengono definiti la distribuzione e l'incidenza del risultato del calcolo di allocazione.</span><span class="sxs-lookup"><span data-stu-id="3ee6b-126">**Destination** – This component defines how the result of the allocation calculation should be distributed and accounted for.</span></span> <span data-ttu-id="3ee6b-127">Ad esempio, vi può essere una riga di destinazione per ciascun reparto.</span><span class="sxs-lookup"><span data-stu-id="3ee6b-127">For example, there can be one destination line for each department.</span></span>
-   <span data-ttu-id="3ee6b-128">**Contropartita**: questo componente stabilisce come i conti principali e le dimensioni debbano essere determinati per le voci di contropartita che sono a saldo delle voci di destinazione.</span><span class="sxs-lookup"><span data-stu-id="3ee6b-128">**Offset** – This component defines how main accounts and dimensions should be determined for the offset entries that balance the destination entries.</span></span> <span data-ttu-id="3ee6b-129">Le opzioni definite dall'utente vengono solitamente utilizzate al posto dei conti e delle dimensioni basati sull'origine.</span><span class="sxs-lookup"><span data-stu-id="3ee6b-129">User-defined options are typically used instead of accounts and dimensions that are based on the source.</span></span> <span data-ttu-id="3ee6b-130">Quando l'opzione **Origine dati** è impostata su **Valore fisso**, **Origine** non può essere utilizzata come opzione.</span><span class="sxs-lookup"><span data-stu-id="3ee6b-130">When **Data source** is set to **Fixed value**, **Source** can't be used as an option.</span></span>
-   <span data-ttu-id="3ee6b-131">**Regole di base di allocazione contabile**: queste regole utilizzano i propri i criteri di filtro di origine per determinare i saldi contabili che devono essere utilizzati per l'allocazione (ad esempio, i ricavi per reparto).</span><span class="sxs-lookup"><span data-stu-id="3ee6b-131">**Ledger allocation basis rules** – These rules use their own source filter criteria to determine which ledger balances should be used for allocation (for example, the revenue per department).</span></span> <span data-ttu-id="3ee6b-132">È possibile utilizzare ciascuna regola base di allocazione con più regole di allocazione.</span><span class="sxs-lookup"><span data-stu-id="3ee6b-132">Each allocation basis rule can be used with multiple allocation rules.</span></span>
