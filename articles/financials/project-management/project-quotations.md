# <a name="project-quotations"></a><span data-ttu-id="4e7d8-104">Offerte di progetto</span><span class="sxs-lookup"><span data-stu-id="4e7d8-104">Project quotations</span></span>

[!include[banner](../includes/banner.md)]


<span data-ttu-id="4e7d8-105">Viene introdotto il concetto delle offerte di progetto da utilizzare per fare un'offerta attraente a un cliente come primo passaggio della fase di progetto.</span><span class="sxs-lookup"><span data-stu-id="4e7d8-105">This article introduces the concept of project quotations, which you can use to make an attractive offer to a customer as the first step of the project phase.</span></span> <span data-ttu-id="4e7d8-106">Un'offerta di progetto può comprendere articoli e servizi inclusi, informazioni di base sul contatto, accordi commerciali e sconti speciali e possibili imposte e supplementi.</span><span class="sxs-lookup"><span data-stu-id="4e7d8-106">A project quotation might include the items and services that are quoted, basic contact information, special trade agreements and discounts, and possible taxes and surcharges.</span></span> 

<span data-ttu-id="4e7d8-107">La possibilità di monitore, analizzare e controllare la pipeline delle offerte di progetto e degli ordini rappresenta una parte importante della gestione del progetto.</span><span class="sxs-lookup"><span data-stu-id="4e7d8-107">The ability to monitor, review, and control the pipeline of project quotations and orders is an important part of project management.</span></span> <span data-ttu-id="4e7d8-108">I vari strumenti in Microsoft Dynamics 365 for Finance and Operations, Enterprise Edition possono fornire assistenza alle attività.</span><span class="sxs-lookup"><span data-stu-id="4e7d8-108">Various tools in Microsoft Dynamics 365 for Finance and Operations, Enterprise edition can help with these tasks.</span></span> <span data-ttu-id="4e7d8-109">Ad esempio, le definizioni dei corretti dati di riferimento (tipi di offerta, origine dell'offerta, previsione e probabilità) consentono di analizzare la pipeline.</span><span class="sxs-lookup"><span data-stu-id="4e7d8-109">For example, correct reference data definitions (quotation types, quotation origin, and prognosis and probability) help you analyze the pipeline.</span></span> <span data-ttu-id="4e7d8-110">È possibile utilizzare questi strumenti per classificare i motivi per cui un'offerta di progetto è stata acquisita o persa e determinare il potenziale valore di un'offerta.</span><span class="sxs-lookup"><span data-stu-id="4e7d8-110">You can use these tools to categorize the reasons why a project quotation was won or lost, and to determine the potential value of a quotation.</span></span> 

<span data-ttu-id="4e7d8-111">In un'offerta di progetto, immettere i servizi, le informazioni di base sul contatto, accordi commerciali e sconti speciali nonché imposte e supplementi stimati per un progetto.</span><span class="sxs-lookup"><span data-stu-id="4e7d8-111">In a project quotation, you enter the services, basic contact information, special trade agreements and discounts, and estimated taxes and surcharges for a project.</span></span> <span data-ttu-id="4e7d8-112">È anche possibile selezionare le attività o le operazioni per un progetto e creare una gerarchia di attività e attività secondarie.</span><span class="sxs-lookup"><span data-stu-id="4e7d8-112">You can also select the activities or tasks for a project, and create a hierarchy of tasks and subtasks.</span></span> <span data-ttu-id="4e7d8-113">Per ogni attività, è possibile immettere dettagli sui tempi e la durata relativi e sulle competenze e l'esperienza richieste ai lavoratori per questa attività.</span><span class="sxs-lookup"><span data-stu-id="4e7d8-113">For each activity, you can enter details about the timing and duration of the activity, and about the skills and experience that are required for workers who perform the activity.</span></span> 

<span data-ttu-id="4e7d8-114">L'offerta di progetto è una stima non vincolante del lavoro da svolgere.</span><span class="sxs-lookup"><span data-stu-id="4e7d8-114">The project quotation is a non-binding estimate of the work that must be performed.</span></span> <span data-ttu-id="4e7d8-115">Tuttavia, quando le informazioni nell'offerta vengono copiate in un progetto che viene associato a un contratto di progetto, le informazioni diventano parte di un contratto vincolante tra le due parti.</span><span class="sxs-lookup"><span data-stu-id="4e7d8-115">However, when the information in the quotation is copied to a project that is associated with a project contract, that information becomes part of a binding agreement between two parties.</span></span> 

<span data-ttu-id="4e7d8-116">Se il cliente approva l'offerta di progetto, è possibile copiare le informazioni dell'offerta di progetto in un contratto di progetto.</span><span class="sxs-lookup"><span data-stu-id="4e7d8-116">If the customer approves the project quotation, you can copy the information in the project quotation to a project.</span></span> <span data-ttu-id="4e7d8-117">È inoltre possibile copiare contemporaneamente le informazioni dell'offerta di progetto in una previsione di progetto.</span><span class="sxs-lookup"><span data-stu-id="4e7d8-117">You can also copy the project quotation information to a project forecast at the same time.</span></span>
