# <a name="view-an-expense-report"></a>Visualizza nota spese

[!include [banner](../includes/banner.md)]

Quando si esegue una query per visualizzare le transazioni in Contabilità generale o Approvvigionamento, la query potrebbe restituire troppe informazioni. Di conseguenza, potrebbe essere necessario limitare l'ambito della ricerca. Per ottenere le informazioni necessarie, è possibile eseguire il drill-down in una transazione per visualizzare la nota spese con le informazioni desiderate.

Per le note spese è possibile eseguire il drill-down da una transazione fino alla nota spese originale archiviata in Gestione spese. Oltre all'importo totale delle spese, è possibile visualizzare i dettagli delle transazioni, le transazioni idonee al recupero dell'imposta sul valore aggiunto (IVA) e le ricevute collegate alle spese.

Per eseguire il drill-down fino alla nota spese originale in cui una transazione è riportata, selezionare la transazione, quindi l'opzione **Documento originale**.

Quando si visualizzano le singole transazioni, è possibile visualizzare anche tutti i dati aggiuntivi collegati a una transazione. Per i dettagli delle spese di albergo è possibile ad esempio visualizzare l'elenco dettagliato di tutte le spese quali accesso a Internet, chiamate telefoniche e servizio in camera.
