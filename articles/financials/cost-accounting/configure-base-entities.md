# <a name="configure-base-entities"></a>Configura entità di base 

[!include[banner](../includes/banner.md)]


La contabilità industriale è un modulo indipendente che richiede la corretta configurazione delle seguenti entità di base in Dynamics 365 for Finance and Operations, Enterprise edition.

-  Valuta
-  Tasso di cambio
-  Calendario fiscale  

Prima di utilizzare il modulo Contabilità industriale, è necessario che una valuta, un tasso di cambio e un calendario fiscale siano definiti per la società. 

Per ulteriori informazioni su queste entità, effettuare il corso di formazione eLearning [Modulo 2 Impostazione della valuta e del tasso di cambio](https://mbspartner.microsoft.com/AX/CourseModules/1215) (è necessario un account CustomerSource).
