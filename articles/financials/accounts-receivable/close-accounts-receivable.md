# <a name="close-accounts-receivable"></a>Chiusura della contabilità clienti

[!include [banner](../includes/banner.md)]

Nella tabella riportata di seguito sono elencate le pagine di supporto al processo aziendale relativo alla chiusura della contabilità clienti.

> [!NOTE] 
> Per aprire alcune pagine riportate nella tabella è necessario immettere informazioni o specificare impostazioni di parametri.

**Attività componente processo aziendale**                   

Chiudere i periodi nella contabilità generale

| Nome pagina                            | Uso                                                                                      |
|--------------------------------------|--------------------------------------------------------------------------------------------|
|Processo batch                             | Consente di visualizzare o creare processi batch. I processi batch non possono essere completati e si desidera assicurare che tutte le registrazioni siano completate.                                                                                                               |
|Conferma ordine cliente                   | Consente di aggiornare gli ordini cliente.                                                                       |
|Rivalutazione valuta estera          | Consente di generare transazioni per aggiornare il valore delle transazioni cliente aperte in valuta estera.                                                                                                                         |
| Giornale di registrazione                              | Consente di registrare fatture, pagamenti ed effetti passivi.                                             |
| Giustificativo giornale di registrazione                      |<ul><li>**Giornale di registrazione pagamenti**: consente di generare, elaborare e registrare pagamenti.</li><li>**Giornale di registrazione effetti attivi emessi**: consente di registrare gli effetti attivi.</li><li>**Giornale di registrazione effetti attivi protestati**: consente di registrare gli effetti attivi protestati.</li><li>**Giornale di registrazione effetti attivi riemessi**: consente di registrare gli effetti attivi riemessi.</li><li>**Giornale di registrazione rimesse**: consente di registrare le rimesse.</li><li>**Giornale di registrazione effetti attivi liquidati**: consente di registrare gli effetti attivi liquidati</li></ul>                   |
| Registrazione documento di trasporto                 | Consente di aggiornare i documenti di trasporto per gli ordini cliente.                                                     |
| Registra fattura a testo libero               | Consente di registrare le fatture a testo libero.                                                                   |
| Registrazione fattura                      | Consente di registrare le fatture per gli ordini cliente.                                                            |
| Registrazione distinta di prelievo                 |Consente di aggiornare le distinte di prelievo per gli ordini cliente.                                                      |

**Attività componente processo aziendale**   

Creare e inviare l'elenco vendite UE

| Nome pagina                            | Uso                                                                                      |
|--------------------------------------|--------------------------------------------------------------------------------------------|
|Elenco vendite UE                         | Consente di creare report sulle vendite UE (Unione Europea) da inviare agli uffici IVA ai fini della dichiarazione IVA.                                                                                                                           |
