# <a name="set-up-i-9-document-types"></a>Impostare i tipi di documento I-9

[!include[task guide banner](../../../includes/task-guide-banner.md)]

Questa procedura illustra come visualizzare e impostare i tipi di documento utilizzati per la verifica I-9. Prima di impostare i tipi di documento I-9, è necessario impostare gli enti emittenti e i tipi di identificazione. La società di dati dimostrativi utilizzata per creare questa procedura è USMF, che include esempi degli enti emittenti e dei tipi di identificazione necessari per completare la procedura.

1. Andare a Risorse umane > Lavoratori > I-9 > Tipi di documento I-9.
2. Fare clic su Nuovo.
3. Nel campo Tipo di documento I-9 digitare un valore.
    * Esempio: ID scuola.  
4. Selezionare il tipo di identificazione.  Esempio: ID scuola
    * Esempi di tipi di identificazione includono il numero SSN (Social Security Number), il numero del visto, l'ID passaporto o la patente di guida.  
5. Selezionare l'elenco di documento I-9 utilizzato per il tipo di documento.
    * Durante il processo I-9, i dipendenti devono presentare la documentazione che dimostri al datore di lavoro l'identità e l'autorizzazione all'impiego. Il sito Web sui servizi per la cittadinanza e l'immigrazione negli Stati Uniti contiene informazioni sui documenti accettabili e sull'elenco a cui appartengono.  Vedere il sito http://www.uscis.gov  
6. Nel campo Modulo immettere il numero del documento ufficiale per il tipo di documento. Esempio: ID scuola.
    * I numeri dei documenti ufficiali sono disponibili nel sito Web sui servizi per la cittadinanza e l'immigrazione negli Stati Uniti.  Vedere il sito http://www.uscis.gov  
7. Selezionare o deselezionare la casella di controllo Attivo.
8. Nel campo Data di scadenza impostare la data su "27-10-2019".
    * La data di scadenza è facoltativa.  
9. Selezionare l'ente che ha emesso il tipo di documento. Esempio: Regione/provincia/territorio
10. Fare clic su Salva.
