# <a name="develop-a-succession-plan"></a>Sviluppare un piano di successione

[!include[task guide banner](../../includes/task-guide-banner.md)]

Se l'organizzazione si sviluppa e si prende in considerazione la pianificazione della successione, potrebbe essere necessario cercare qualcuno che abbia competenze simili a un'altra persona.  Il mapping delle competenze consente di analizziate i dipendenti esistenti e i candidati per verificare se soddisfano il set di competenze di un dipendente stimato. La società di dati dimostrativi utilizzata per creare questa procedura è USMF.

1. Andare a Risorse umane > Competenze > Analisi competenze > Profili mapping competenze.
2. Fare clic su Nuovo.
3. Nel campo Mapping competenze, immettere un nome per il mapping delle competenze.  Esempio: Dipendente.
4. Nel campo Descrizione digitare un valore.
5. Nel campo Data immettere una data.
6. Fare clic su Recupera profilo.
7. Fare clic su Persona.
8. Nel campo Persona, digitare un nome oppure selezionare il menu a discesa.  Esempio: Cassie Hicks.
9. Fare clic su OK.
10. Espandere la Scheda dettaglio Certificati per visualizzare o modificare i certificati inclusi nel mapping delle competenze.
11. Espandere la Scheda dettaglio Competenze per visualizzare o modificare le competenze da includere.
12. Nell'elenco contrassegnare la prima riga.  Esempio: Contabilità
13. Fare clic sulla casella di controllo Facoltativo.
14. Nel campo Importanza selezionare un'opzione.
    * Quando si contrassegna una competenza come facoltativa, verrà chiesto di selezionare il livello di importanza della competenza.  
15. Nell'elenco selezionare la riga 2.
16. Fare clic sulla casella di controllo Facoltativo.
17. Nel campo Importanza selezionare un'opzione.
18. Nell'elenco selezionare la riga 3.
19. Fare clic sulla casella di controllo Facoltativo.
20. Nel campo Importanza selezionare un'opzione.
21. Nell'elenco selezionare la riga 4.
22. Fare clic sulla casella di controllo Facoltativo.
23. Nel campo Importanza selezionare un'opzione.
24. Espandere la Scheda dettaglio Istruzione per visualizzare o modificare le competenze da includere nel mapping delle competenze.
25. Fare clic su Esegui.
26. Fare clic su OK.
27. Fare clic su Risultato.
28. Fare clic su Report.
    * Le corrispondenze migliori verranno elencate nella parte superiore del report.  È possibile vedere che viene elencato un elemento di lacuna.  È la differenza tra il livello elencato nel mapping competenze e il livello di competenza assegnato alla persona.
