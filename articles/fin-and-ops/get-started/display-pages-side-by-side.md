# <a name="show-pages-side-by-side-by-using-the-open-in-new-window-feature"></a><span data-ttu-id="1424a-103">Mostrare le pagine affiancate utilizzando la funzionalità Apri in nuova finestra</span><span class="sxs-lookup"><span data-stu-id="1424a-103">Show pages side by side by using the Open in new window feature</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="1424a-104">Questo articolo illustra come visualizzare pagine affiancate in Microsoft Dynamics 365 for Finance and Operations.</span><span class="sxs-lookup"><span data-stu-id="1424a-104">This article explains how to display pages side-by-side in Microsoft Dynamics 365 for Finance and Operations.</span></span>

<span data-ttu-id="1424a-105">Microsoft Dynamics 365 for Finance and Operations consente di eseguire le attività efficientemente.</span><span class="sxs-lookup"><span data-stu-id="1424a-105">Microsoft Dynamics 365 for Finance and Operations helps you perform tasks efficiently.</span></span> <span data-ttu-id="1424a-106">In alcuni casi potrebbe essere utile visualizzare più pagine affiancate per eseguire rapidamente le attività.</span><span class="sxs-lookup"><span data-stu-id="1424a-106">In some cases, you may want to view multiple pages side-by-side to complete tasks quickly.</span></span> <span data-ttu-id="1424a-107">Come esempio, potrebbe essere utile convalidare o immettere righe in più di un giornale di registrazione.</span><span class="sxs-lookup"><span data-stu-id="1424a-107">As an example, you might want to validate or enter lines in more than one journal.</span></span> <span data-ttu-id="1424a-108">In genere, a tale scopo è necessario andare avanti e indietro tra la pagina che visualizza un elenco di giornali di registrazione e la pagina che visualizza righe di un giornale di registrazione specifico.</span><span class="sxs-lookup"><span data-stu-id="1424a-108">Typically, to do this you would have to go back and forth between the page that displays a list of journals, and the page that displays lines for a given journal.</span></span> <span data-ttu-id="1424a-109">Tuttavia, la funzionalità **Apri in nuova finestra** consente di visualizzare queste pagine affiancate in modo da poter eseguire rapidamente le attività.</span><span class="sxs-lookup"><span data-stu-id="1424a-109">However, the **Open in new window** feature enables you to display these pages side-by-side so that you can perform your tasks quickly.</span></span>

<span data-ttu-id="1424a-110">Continuando con l'esempio citato sopra, quando vengono visualizzate le righe, è possibile fare clic sull'icona **Apri in nuova finestra**.</span><span class="sxs-lookup"><span data-stu-id="1424a-110">Continuing with the example mentioned above, when viewing the lines, you can click the **Open in new window** icon.</span></span>

<span data-ttu-id="1424a-111">[![icona Apri in nuova finestra](./media/open-in-new-window-icon.png)](./media/open-in-new-window-icon.png)</span><span class="sxs-lookup"><span data-stu-id="1424a-111">[![open-in-new-window-icon](./media/open-in-new-window-icon.png)](./media/open-in-new-window-icon.png)</span></span>

<span data-ttu-id="1424a-112">Se si fa clic su **Apri in nuova finestra** si apre la pagina delle righe in un nuovo browser popup e il browser originale torna indietro nello storico alla pagina in cui era visualizzato l'elenco dei giornali di registrazione.</span><span class="sxs-lookup"><span data-stu-id="1424a-112">Clicking the **Open in new window** icon opens the lines page in a new, pop-up browser, and then navigates the original browser back in history to the page that displayed the list of journals.</span></span> <span data-ttu-id="1424a-113">È possibile quindi visualizzare entrambe le pagine in modalità affiancata.</span><span class="sxs-lookup"><span data-stu-id="1424a-113">You can then display both pages side-by-side.</span></span> <span data-ttu-id="1424a-114">Al termine della visualizzazione di un giornale di registrazione, sarà possibile modificare il giornale di registrazione selezionato nella pagina con l'elenco dei giornali di registrazione e nella pagina delle righe della finestra popup verranno automaticamente visualizzate le righe del giornale di registrazione appena selezionato.</span><span class="sxs-lookup"><span data-stu-id="1424a-114">When you are done viewing a journal, you can change the selected journal on the journal list page, and the lines page in the pop-up window will automatically display the lines of the newly selected journal.</span></span>

<span data-ttu-id="1424a-115">[![visualizzazione di pagine affiancate](./media/pages-show-side-by-side.png)](./media/pages-show-side-by-side.png)</span><span class="sxs-lookup"><span data-stu-id="1424a-115">[![pages-show-side-by-side](./media/pages-show-side-by-side.png)](./media/pages-show-side-by-side.png)</span></span>

<span data-ttu-id="1424a-116">Il collegamento e l'aggiornamento dinamici si verificano a causa delle relazioni esistenti tra i dati dietro a queste pagine.</span><span class="sxs-lookup"><span data-stu-id="1424a-116">The dynamic linking and refreshing happens due to the relations that exist between the data that is backing these pages.</span></span> <span data-ttu-id="1424a-117">Se il sistema non è informato della relazione tra i dati, la finestra popup non si aggiornerà automaticamente in risposta a una modifica nella finestra da cui è stata originata.</span><span class="sxs-lookup"><span data-stu-id="1424a-117">If the system is not aware of the relation between the data, the pop-up window will not refresh automatically in response to a change in the window it originated from.</span></span>

<span data-ttu-id="1424a-118">Alcune pagine presentano più visualizzazioni, ad esempio le visualizzazioni Griglia, Intestazione e Dettagli.</span><span class="sxs-lookup"><span data-stu-id="1424a-118">Some pages have multiple views such as the Grid view, Header view, and Details view.</span></span> <span data-ttu-id="1424a-119">L'icona **Apri in nuova finestra** causa l'apertura dell'intera pagina nella nuova finestra del browser.</span><span class="sxs-lookup"><span data-stu-id="1424a-119">The **Open in new window** icon causes the entire page to be opened in the new browser window.</span></span> <span data-ttu-id="1424a-120">Di conseguenza, non è possibile mantenere due visualizzazioni nella stessa pagina affiancate utilizzando la funzionalità **Apri in nuova finestra**.</span><span class="sxs-lookup"><span data-stu-id="1424a-120">Therefore, you cannot keep two views of the same page side-by-side using the **Open in new window** feature.</span></span> <span data-ttu-id="1424a-121">Tuttavia, quasi tutte le pagine di questo tipo hanno un elenco di navigazione da utilizzare per passare tra i record e raggiungere un simile esperienza.</span><span class="sxs-lookup"><span data-stu-id="1424a-121">However, almost all such pages have a navigation list that you can use to switch between records and achieve a similar experience.</span></span>

<span data-ttu-id="1424a-122">Prima di utilizzare la funzionalità **Apri in nuova finestra**, è necessario configurare il blocco popup del browser per consentire i popup provenienti dall'URL del sito Finance and Operations.</span><span class="sxs-lookup"><span data-stu-id="1424a-122">Before using the **Open in new window** feature, you should configure your browser's pop-up blocker to allow pop-ups from the URL of the Finance and Operations site.</span></span> <span data-ttu-id="1424a-123">Ad esempio, è possibile consentire i popup provenienti da  "\*.dynamics.com".</span><span class="sxs-lookup"><span data-stu-id="1424a-123">As an example, you could allow pop-ups from "\*.dynamics.com".</span></span>

<span data-ttu-id="1424a-124">La funzionalità **Apri in nuova finestra** è disponibile solo quando sono presenti più pagine aperte nella finestra.</span><span class="sxs-lookup"><span data-stu-id="1424a-124">The **Open in new window** feature is only available when there is more than one page open in the window.</span></span> <span data-ttu-id="1424a-125">Inoltre, la finestra popup si chiude automaticamente quando non sono presenti più pagine aperte (ovvero quando viene chiusa l'ultima pagina nella finestra).</span><span class="sxs-lookup"><span data-stu-id="1424a-125">Also, the pop-up window automatically closes when there are no more pages open (that is, when the last page in that window is closed).</span></span> <span data-ttu-id="1424a-126">Finance and Operations chiude anche le pagine aperte quando si esplora un'area diversa dell'applicazione.</span><span class="sxs-lookup"><span data-stu-id="1424a-126">Finance and Operations also closes open pages when you navigate to a different area in the application.</span></span> <span data-ttu-id="1424a-127">Di conseguenza, se le finestre popup sono aperte e si esplora un'area diversa dell'applicazione, le finestre popup sono automaticamente chiuse perché le pagine in quelle finestre sono state chiuse dal sistema.</span><span class="sxs-lookup"><span data-stu-id="1424a-127">Therefore, if you have pop-up windows open and navigate to a different area in the application, the pop-up windows are automatically closed because the pages in those windows were closed by the system.</span></span>

<span data-ttu-id="1424a-128">La barra superiore nelle finestre popup visualizza le informazioni sulla società in cui è stata aperta la pagina ed è di sola lettura.</span><span class="sxs-lookup"><span data-stu-id="1424a-128">The top bar in the pop-up windows displays information about the company the page was opened in and is read-only.</span></span> <span data-ttu-id="1424a-129">Le finestre popup si basano anche sulla finestra principale del browser Finance and Operations.</span><span class="sxs-lookup"><span data-stu-id="1424a-129">The pop-up windows also rely on the main Finance and Operations browser window.</span></span> <span data-ttu-id="1424a-130">Se la finestra principale è chiusa o aggiornata, tutte le finestre popup aperte diventeranno di sola lettura.</span><span class="sxs-lookup"><span data-stu-id="1424a-130">If the main window is closed or refreshed, all open pop-up windows will become read only.</span></span> <span data-ttu-id="1424a-131">Ciò significa che è comunque possibile visualizzare le informazioni in queste finestre, ma non sarà possibile interagire con esse.</span><span class="sxs-lookup"><span data-stu-id="1424a-131">This means that you can still view the information in these windows, but you will not be able to interact with it.</span></span>
