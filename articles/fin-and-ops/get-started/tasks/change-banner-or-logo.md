[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="d9b0f-104">La procedura riportata di seguito elenca i passaggi che gli amministratori di sistema possono utilizzare per aggiornare l'immagine di logo o banner visualizzata per una persona giuridica.</span><span class="sxs-lookup"><span data-stu-id="d9b0f-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="d9b0f-105">La società di dati dimostrativi utilizzata per creare questa procedura è USMF.</span><span class="sxs-lookup"><span data-stu-id="d9b0f-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="d9b0f-106">Andare ad Amministrazione organizzazione > Organizzazioni > Persone giuridiche.</span><span class="sxs-lookup"><span data-stu-id="d9b0f-106">Go to Organization administration > Organizations > Legal entities.</span></span>

4. <span data-ttu-id="d9b0f-111">Fare clic su Modifica.</span><span class="sxs-lookup"><span data-stu-id="d9b0f-111">Click Edit.</span></span>

6. <span data-ttu-id="d9b0f-113">Fare clic su Modifica.</span><span class="sxs-lookup"><span data-stu-id="d9b0f-113">Click Change.</span></span>
    * <span data-ttu-id="d9b0f-114">Suggerimento: la risoluzione ideale per un'immagine di tipo banner è 1920 x 281 pixel.</span><span class="sxs-lookup"><span data-stu-id="d9b0f-114">Tip: The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="d9b0f-115">La larghezza ideale per un'immagine di tipo logo è 350 pixel.</span><span class="sxs-lookup"><span data-stu-id="d9b0f-115">The ideal width for a logo image is 350 pixels.</span></span>
