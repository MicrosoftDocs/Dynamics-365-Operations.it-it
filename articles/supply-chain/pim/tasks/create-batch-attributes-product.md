# <a name="create-batch-attributes-for-a-product"></a>Creare gli attributi batch per un prodotto

[!include [banner](../../includes/banner.md)]

Questa procedura illustra come creare un attributo batch, come assegnare intervalli di valori predefiniti e come includere l'attributo in un gruppo. La società di dati dimostrativi utilizzata per creare questa procedura è USP2.

1. Andare a Gestione articoli > Impostazioni > Batch > Attributi batch.
2. Fare clic su Nuovo.
3. Digitare un valore nel campo Attributo.
4. Nel campo Descrizione digitare un valore.
5. Nel campo Tipo di attributo selezionare "Frazione".
    * Questa procedura utilizza il tipo Frazione per consentire i valori decimali. È possibile selezionare altri tipi di attributo. Se si seleziona il tipo di enumerazione, è necessario immettere i valori nell'elenco di enumerazione prima di immettere un valore nel campo di destinazione.  
6. Nel campo Minimo immettere un numero.
7. Nel campo Massimo immettere un numero.
8. Nel campo Incremento immettere un numero.
9. Nel campo Destinazione digitare un valore.
10. Fare clic su Salva.
11. Chiudere la pagina.
12. Andare a Gestione articoli > Impostazioni > Batch > Gruppi di attributi batch.
13. Fare clic su Nuovo.
14. Digitare un valore nel campo Gruppo di attributi.
15. Nel campo Descrizione digitare un valore.
16. Fare clic su Salva.
17. Fare clic su Attributi del gruppo.
18. Fare clic su Nuovo.
19. Nel campo Attributo fare clic sul pulsante a discesa per aprire la ricerca.
20. Nell'elenco trovare e selezionare il record desiderato.
21. Nell'elenco fare clic sul collegamento nella riga selezionata.
    * Un attributo può essere incluso in uno qualsiasi dei gruppi.  
22. Fare clic su Salva.
23. Chiudere la pagina.
