[!include [banner](../../includes/banner.md)]

<span data-ttu-id="a83cf-104">In questa argomento viene illustrato come creare un codice a barre manualmente usando il numero di articolo M0001 come esempio.</span><span class="sxs-lookup"><span data-stu-id="a83cf-104">This topic shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="a83cf-105">La società di dati dimostrativi utilizzata per creare questa procedura è USMF.</span><span class="sxs-lookup"><span data-stu-id="a83cf-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="a83cf-106">Selezionare **Gestione prodotti rilasciati** in home page.</span><span class="sxs-lookup"><span data-stu-id="a83cf-106">Select **Released product maintenance** on the homepage.</span></span>

6. <span data-ttu-id="a83cf-112">Selezionare **Nuovo**.</span><span class="sxs-lookup"><span data-stu-id="a83cf-112">Select **New**.</span></span>

10. <span data-ttu-id="a83cf-116">Digitare un valore nel campo **Codice a barre**.</span><span class="sxs-lookup"><span data-stu-id="a83cf-116">In the **Bar code** field, type a value.</span></span>

12. <span data-ttu-id="a83cf-118">Selezionare **Salva**.</span><span class="sxs-lookup"><span data-stu-id="a83cf-118">Select **Save**.</span></span>
13. <span data-ttu-id="a83cf-119">Chiudere la pagina.</span><span class="sxs-lookup"><span data-stu-id="a83cf-119">Close the page.</span></span>
