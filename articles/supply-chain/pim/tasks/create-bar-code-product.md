[!include [banner](../../includes/banner.md)]

<span data-ttu-id="8f7ce-104">In questa argomento viene illustrato come creare un codice a barre manualmente usando il numero di articolo M0001 come esempio.</span><span class="sxs-lookup"><span data-stu-id="8f7ce-104">This topic shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="8f7ce-105">La società di dati dimostrativi utilizzata per creare questa procedura è USMF.</span><span class="sxs-lookup"><span data-stu-id="8f7ce-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="8f7ce-106">Selezionare **Gestione prodotti rilasciati** in home page.</span><span class="sxs-lookup"><span data-stu-id="8f7ce-106">Select **Released product maintenance** on the homepage.</span></span>
2. <span data-ttu-id="8f7ce-107">Passare a **Prodotti > Prodotti rilasciati** nella sezione **Collegamenti**.</span><span class="sxs-lookup"><span data-stu-id="8f7ce-107">Go to **Products > Released products** under the **Links** section.</span></span>
3. <span data-ttu-id="8f7ce-108">Nell'elenco trovare e selezionare il record desiderato.</span><span class="sxs-lookup"><span data-stu-id="8f7ce-108">In the list, find and select the desired record.</span></span> <span data-ttu-id="8f7ce-109">Per questo esempio, selezionare il numero articolo **M0001**.</span><span class="sxs-lookup"><span data-stu-id="8f7ce-109">For this example, select item number **M0001**.</span></span>

5. <span data-ttu-id="8f7ce-111">Selezionare **Codici a barre**.</span><span class="sxs-lookup"><span data-stu-id="8f7ce-111">Select **Bar codes**.</span></span>
6. <span data-ttu-id="8f7ce-112">Selezionare **Nuovo**.</span><span class="sxs-lookup"><span data-stu-id="8f7ce-112">Select **New**.</span></span>
7. <span data-ttu-id="8f7ce-113">Contrassegnare la riga selezionata creata nell'elenco seguente.</span><span class="sxs-lookup"><span data-stu-id="8f7ce-113">Mark the selected row that is created in the list below.</span></span>

9. <span data-ttu-id="8f7ce-115">Nel campo **Codice a barre** immettere o selezionare un valore.</span><span class="sxs-lookup"><span data-stu-id="8f7ce-115">In the **Bar code** field, enter or select a value.</span></span>
10. <span data-ttu-id="8f7ce-116">Digitare un valore nel campo **Codice a barre**.</span><span class="sxs-lookup"><span data-stu-id="8f7ce-116">In the **Bar code** field, type a value.</span></span>  
11. <span data-ttu-id="8f7ce-117">Nel campo **Quantità** immettere un numero.</span><span class="sxs-lookup"><span data-stu-id="8f7ce-117">In the **Quantity** field, enter a number.</span></span>
12. <span data-ttu-id="8f7ce-118">Selezionare **Salva**.</span><span class="sxs-lookup"><span data-stu-id="8f7ce-118">Select **Save**.</span></span>
13. <span data-ttu-id="8f7ce-119">Chiudere la pagina.</span><span class="sxs-lookup"><span data-stu-id="8f7ce-119">Close the page.</span></span>
