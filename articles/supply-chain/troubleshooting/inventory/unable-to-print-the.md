# <a name="picking-list-report-doesnt-show-new-notes-that-are-attached-to-a-journal"></a>Il report Distinta di prelievo non mostra le nuove note associate a un giornale di registrazione

Numero KB: 4611660

## <a name="symptoms"></a>Sintomi

Il report Distinta di prelievo non mostra le nuove note associate a un giornale di registrazione distinte di prelievo.

## <a name="resolution"></a>Risoluzione

Il report Distinta di prelievo mostra solo le note associate alla riga dell'ordine cliente e all'intestazione dell'ordine cliente. Non mostra le eventuali nuove note aggiunte nel giornale di registrazione fatture.
