# <a name="cant-end-a-production-order-because-of-a-bom-consumptioncost-value-error"></a>Impossibile terminare un ordine di produzione a causa di un errore del valore consumptionCost della distinta base

## <a name="symptoms"></a>Sintomi

Quando si tenta di terminare un ordine di produzione, viene visualizzato il seguente messaggio di errore:

> Il calcolo del valore consumptionCost della distinta base deve essere negativo all'emissione dall'inventario.

## <a name="resolution"></a>Risoluzione

Questo problema è stato risolto nella versione 10.0.15.
