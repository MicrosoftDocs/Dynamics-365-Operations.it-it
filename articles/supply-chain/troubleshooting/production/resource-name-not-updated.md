# <a name="resource-name-isnt-updated-after-changing-a-worker-name-in-the-address-book"></a>Il nome della risorsa non viene aggiornato dopo aver modificato il nome di un lavoratore nella rubrica

## <a name="symptoms"></a>Sintomi

Se si modifica il nome di un lavoratore nella rubrica globale, il nome della risorsa corrispondente non viene aggiornato nei dati master del gruppo di risorse.

## <a name="resolution"></a>Risoluzione

Il nome della risorsa corrispondente non viene attualmente aggiornato automaticamente. Per risolvere il problema, è necessario aggiornare manualmente il nome della risorsa.
