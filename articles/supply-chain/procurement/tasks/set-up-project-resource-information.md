# <a name="set-up-project-resource-information"></a>Impostare le informazioni sulle risorse di progetto

[!include[task guide banner](../../includes/task-guide-banner.md)]

Questa procedura mostra come impostare una risorsa del progetto. Questa attività utilizza il set di dati dimostrativi USMF.

1. Passare a Gestione progetti e contabilità > Risorse di progetto > Elenco di risorse.
2. Fare clic su Nuova risorsa.
3. Nel campo Lavoratore immettere o selezionare un valore.
4. Nel campo Calendario immettere o selezionare un valore.
5. Nel campo Codice periodo immettere o selezionare un valore.
6. Fare clic su Crea.
7. Fare clic su Prezzi di costo.
8. Fare clic su Prezzo di costo - Ora.
9. Fare clic su Nuovo.
10. Nel campo Data di validità, immettere una data.
11. Immettere un numero nel campo Prezzo di costo.
12. Fare clic su Salva.
13. Chiudere la pagina.
14. Fare clic su Prezzi di vendita.
15. Fare clic su Prezzo di vendita - Ora.
16. Fare clic su Nuovo.
17. Nel campo Data di validità, immettere una data.
18. Immettere un numero nel campo Determinazione prezzo.
19. Selezionare un'opzione nel campo Valido per.
20. Fare clic su Salva.
21. Chiudere la pagina.
