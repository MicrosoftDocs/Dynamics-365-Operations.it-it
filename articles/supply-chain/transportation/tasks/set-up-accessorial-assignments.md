# <a name="set-up-accessorial-assignments"></a>Impostare assegnazioni spese accessorie

[!include [banner](../../includes/banner.md)]

Questa procedura mostra come impostare un'assegnazione delle spese accessorie. La procedura viene in genere eseguita dal coordinatore dei trasporti. Prima di utilizzare questa guida, è necessario eseguire la procedura "Impostare le spese accessorie per l'hub e le spese accessorie principali".


## <a name="set-up-accessorial-assignment"></a>Impostare un'assegnazione di spese accessorie
1. Andare a Gestione trasporto > Impostazioni > Valutazione > Assegnazioni spese accessorie.
2. Fare clic su Nuovo.
3. Digitare un valore nel campo Nome.
4. Attivare/disattivare l'espansione della sezione Dettagli.
5. Nel campo Hub fare clic sul pulsante a discesa per aprire la ricerca.
6. Nell'elenco selezionare l'hub per cui sono state create spese accessorie principali quando è stata eseguita la guida "Impostare le spese accessorie per l'hub e le spese accessorie principali". 
7. Nel campo ID spese accessorie hub fare clic sul pulsante a discesa per aprire la ricerca.
8. Nell'elenco fare clic sul collegamento nella riga selezionata.
9. Attivare/disattivare l'espansione della sezione Criteri.
    * Nella sezione Criteri è possibile selezionare i criteri esatti per il momento in cui applicare la spesa, in base ai valori diversi offerti in questo campo.  
10. Impostare l'opzione Applica sempre su Sì.
11. Selezionare un'opzione nel campo Livello di assegnazione spese accessorie.
12. Attivare/disattivare l'espansione della sezione Calcolo.
13. Nel campo per il tipo di commissioni per le spese accessorie selezionare "Forfettario".
    * Il tipo di commissioni per le spese accessorie determina la modalità di calcolo dell'addebito effettivo. In questo esempio si tratta di un addebito forfettario.  
14. Nel campo Commissioni accessorie immettere un numero.
15. Fare clic su Salva.
