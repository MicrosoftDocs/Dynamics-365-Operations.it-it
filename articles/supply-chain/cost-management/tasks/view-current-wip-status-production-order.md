# <a name="view-current-wip-status-on-a-production-order"></a>Visualizzare lo stato WIP corrente su un ordine di produzione

[!include [banner](../../includes/banner.md)]

Questa procedura mostra come visualizzare un rendiconto WIP in un ordine di produzione. La società di dati dimostrativi utilizzata per creare questa procedura è USMF. Questa procedura è destinata al controllore costi.

1. Fare clic su Amministrazione costi.
2. Fare clic su Ordini di produzione.
3. Utilizzare il filtro rapido per filtrare il campo Produzione in base a un valore di 'p000153'.
4. Nel riquadro azioni, fare clic su Gestisci costi.
5. Fare clic su Rendiconto WIP produzione.
6. Nel campo Data iniziale impostare la data su '2012-12-01'.
7. Nel campo Data finale impostare la data su '2012-12-31'.
