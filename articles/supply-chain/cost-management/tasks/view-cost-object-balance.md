# <a name="view-cost-object-balance"></a>Visualizzare il saldo oggetto di costo

[!include [banner](../../includes/banner.md)]

La società di dati dimostrativi utilizzata per creare questa procedura è USMF. Questa procedura è destinata al controllore costi.

1. Selezionare **Amministrazione costi**.
2. Selezionare **Prodotti rilasciati**.
3. Utilizzare il filtro rapido per filtrare il campo **Numero articolo** in base al valore *m0004*.
4. Nel riquadro Azioni, fare clic su **Gestione costi**.
5. Selezionare **Oggetti di costo**.
6. Selezionare **Quantità fisica**.
7. Selezionare **Visualizzazione dimensioni**.
8. Deselezionare la casella di controllo **Ubicazione**.
9. Selezionare **OK**.
