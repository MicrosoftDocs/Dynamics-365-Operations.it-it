# <a name="weighted-average-date"></a><span data-ttu-id="ab73a-103">Data media ponderata</span><span class="sxs-lookup"><span data-stu-id="ab73a-103">Weighted average date</span></span>

<span data-ttu-id="ab73a-104">La data media ponderata è un modello inventariale basato sul principio di media ponderate.</span><span class="sxs-lookup"><span data-stu-id="ab73a-104">Weighted average date is an inventory model that is based on the weighted average principle.</span></span> <span data-ttu-id="ab73a-105">Per tale principio, le uscite da magazzino vengono valutate al valore medio degli articoli ricevuti in magazzino per ogni giorno del periodo di chiusura inventario.</span><span class="sxs-lookup"><span data-stu-id="ab73a-105">For the weighted average principle, issues from inventory are valued at the average value of the items that are received into inventory for each day in the inventory closing period.</span></span> 

<span data-ttu-id="ab73a-106">Quando si esegue una chiusura inventario tramite la data media ponderata, tutte le entrate giornaliere vengono liquidate a fronte di un'uscita virtuale.</span><span class="sxs-lookup"><span data-stu-id="ab73a-106">When you run an inventory closing by using weighted average date, all daily receipts are settled against a virtual issue.</span></span> <span data-ttu-id="ab73a-107">Quest'ultima comprende la quantità totale ricevuta e il valore del giorno.</span><span class="sxs-lookup"><span data-stu-id="ab73a-107">This virtual issue holds the total received quantity and value for that day.</span></span> <span data-ttu-id="ab73a-108">Questa uscita virtuale è associata a un'entrata virtuale corrispondente rispetto a cui verrà liquidata.</span><span class="sxs-lookup"><span data-stu-id="ab73a-108">The virtual issue has a corresponding virtual receipt that the issues will be settled against.</span></span> <span data-ttu-id="ab73a-109">In questo modo a tutte le uscite viene assegnato lo stesso costo medio.</span><span class="sxs-lookup"><span data-stu-id="ab73a-109">Therefore, all issues receive the same average cost.</span></span> <span data-ttu-id="ab73a-110">L'uscita e l'entrata virtuali possono essere considerate come un trasferimento virtuale, denominato *trasferimento di chiusura inventario media ponderata*.</span><span class="sxs-lookup"><span data-stu-id="ab73a-110">The virtual issue and virtual receipt can be considered a virtual transfer that is known as the *weighted average inventory closing transfer*.</span></span>
