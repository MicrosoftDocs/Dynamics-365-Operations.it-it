# <a name="inventory-close"></a><span data-ttu-id="9d771-103">Chiusura inventario</span><span class="sxs-lookup"><span data-stu-id="9d771-103">Inventory close</span></span>

[!INCLUDE [banner](../includes/banner.md)]

<span data-ttu-id="9d771-104">Durante il processo per liquidare le transazioni in uscita con le transazioni in entrata, è possibile scegliere di aggiornare la contabilità generale in base alle rettifiche effettuate.</span><span class="sxs-lookup"><span data-stu-id="9d771-104">As part of the process to settle issue transactions with receipt transactions, you can also choose to have the general ledger updated to reflect the adjustments that have been made.</span></span>

<span data-ttu-id="9d771-105">Il processo di chiusura dell'inventario  consente di compensare le transazioni in uscita a fronte delle transazioni in entrata in base al metodo di valutazione del magazzino selezionato nel gruppo di modelli di articoli dell'articolo.</span><span class="sxs-lookup"><span data-stu-id="9d771-105">The inventory close process settles issue transactions to receipt transactions, based on the inventory valuation method that is selected in the item’s item model group.</span></span> <span data-ttu-id="9d771-106">Nel processo di liquidazione, è possibile specificare la necessità di aggiornare la contabilità generale, in modo che rifletta le rettifiche effettuate.</span><span class="sxs-lookup"><span data-stu-id="9d771-106">As part of the settlement process, you can specify that the general ledger should be updated, so that it reflects the adjustments that have been made.</span></span> <span data-ttu-id="9d771-107">Finché tuttavia non si effettuerà la chiusura o il ricalcolo dell'inventario, le transazioni in uscita vengono registrate con il prezzo di costo medio corrente calcolato.</span><span class="sxs-lookup"><span data-stu-id="9d771-107">However, until inventory close or recalculation has been run, issue transactions are posted at the calculated running average cost price.</span></span> 

<span data-ttu-id="9d771-108">Dopo la chiusura dell'inventario, non è più possibile effettuare la registrazione in periodi precedenti alla data di chiusura impostata, a meno che non si decida di stornare un processo di chiusura inventario completato.</span><span class="sxs-lookup"><span data-stu-id="9d771-108">After inventory close, you can no longer post in periods that are before the inventory closing date that you set, unless you reverse a completed inventory close process.</span></span> <span data-ttu-id="9d771-109">Ad esempio, se la chiusura dell'inventario viene eseguita per il periodo che termina il 31 gennaio, non è possibile registrare le transazioni con una data precedente a il 31 gennaio.</span><span class="sxs-lookup"><span data-stu-id="9d771-109">For example, if inventory close is run for the period that ends on January 31, you can't post transactions that have a date that is earlier than January 31.</span></span> 

<span data-ttu-id="9d771-110">Gli articoli del magazzino vengono assegnati a uno dei due tipi di inventario seguenti: articolo o assistenza.</span><span class="sxs-lookup"><span data-stu-id="9d771-110">Items in inventory are assigned to one of two inventory types: item or service.</span></span> <span data-ttu-id="9d771-111">La chiusura dell'inventario esegue le stesse funzioni per entrambi i tipi.</span><span class="sxs-lookup"><span data-stu-id="9d771-111">Inventory close performs the same functions for both types.</span></span> <span data-ttu-id="9d771-112">Tuttavia, per gli articoli di tipo Assistenza, le uscite verranno comunque compensate con le entrate.</span><span class="sxs-lookup"><span data-stu-id="9d771-112">However, for service items, inventory close still settles issues to receipts.</span></span> 

<span data-ttu-id="9d771-113">La frequenza di esecuzione del processo di chiusura dell'inventario varia a seconda della società.</span><span class="sxs-lookup"><span data-stu-id="9d771-113">How often the inventory close process is run varies by company.</span></span> <span data-ttu-id="9d771-114">Tuttavia, il volume di transazioni dovrebbe agevolare la determinazione della frequenza di esecuzione della chiusura dell'inventario.</span><span class="sxs-lookup"><span data-stu-id="9d771-114">However, transaction volume should help determine how often you decide to run inventory close.</span></span> <span data-ttu-id="9d771-115">La maggior parte delle società esegue di solito la chiusura dell'inventario nell'ambito delle procedure di chiusura e riconciliazione di fine mese.</span><span class="sxs-lookup"><span data-stu-id="9d771-115">In general, most companies run inventory close as part of their month-end close and reconciliation procedures.</span></span>

<span data-ttu-id="9d771-119">Durante il ricalcolo dell'inventario, le scorte disponibili vengono rettificate, le operazioni di magazzino vengono rettificate e i ricalcoli e le chiusure dell'inventario vengono eseguiti.</span><span class="sxs-lookup"><span data-stu-id="9d771-119">During inventory recalculation, on-hand inventory is adjusted, inventory transactions are adjusted, and inventory recalculations and inventory closes are run.</span></span> <span data-ttu-id="9d771-120">Queste attività influiscono su qualsiasi conto CoGe che è collegato all'operazione di magazzino originale.</span><span class="sxs-lookup"><span data-stu-id="9d771-120">These tasks affect any ledger account that is linked to the original inventory transaction.</span></span>

<span data-ttu-id="9d771-125">Al termine dell'aggiornamento, è possibile esaminare il giustificativo contabile che è stato registrato a seguito dell'esecuzione di una di queste attività.</span><span class="sxs-lookup"><span data-stu-id="9d771-125">After the update is completed, you can review the ledger voucher that is posted because of one of these tasks.</span></span>
