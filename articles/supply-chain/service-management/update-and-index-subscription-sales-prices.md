# <a name="update-and-index-subscription-sales-prices"></a>Aggiornare e indicizzare i prezzi di vendita di sottoscrizione 

[!include [banner](../includes/banner.md)]


1.  Fare clic su **Gestione assistenza** \> **Comune** \> **Sottoscrizioni assistenza** \> **Tutte le sottoscrizioni assistenza**.

2.  Selezionare la sottoscrizione per la quale si desidera aggiornare l'indice o il prezzo di base.

3.  Fare clic su **Indice**. Nel campo **Modifica** indicare se si desidera aggiornare il prezzo di base o l'indice e se l'aggiornamento deve utilizzare una percentuale o un nuovo valore. Ãˆ possibile selezionare tra le opzioni seguenti:
    
      - **Indice per percentuale**
    
      - **Indice per**
    
      - **Prezzo base per percentuale**
    
      - **Prezzo base per**

4.  Fare clic su **OK**.

L'indice o il prezzo di base della sottoscrizione vengono aggiornati con il valore specificato.
