[!include [banner](../includes/banner.md)]


<span data-ttu-id="8b0d0-104">È possibile creare una sottoscrizione assistenza da un progetto in **Gestione progetti e contabilità**.</span><span class="sxs-lookup"><span data-stu-id="8b0d0-104">You can create a service subscription from a project in **Project management and accounting**.</span></span>

1.  <span data-ttu-id="8b0d0-105">Fare clic su **Gestione progetti e contabilità** \> **Comune** \> **Progetti** \> **Tutti i progetti**.</span><span class="sxs-lookup"><span data-stu-id="8b0d0-105">Click **Project management and accounting** \> **Common** \> **Projects** \> **All projects**.</span></span> <span data-ttu-id="8b0d0-106">Aprire il progetto per il quale si desidera creare una sottoscrizione.</span><span class="sxs-lookup"><span data-stu-id="8b0d0-106">Open the project that you want to create a subscription for.</span></span>

2.  <span data-ttu-id="8b0d0-107">Nel modulo **Progetti**, nella scheda **Gestisci** del **riquadro azioni** fare clic su **Assistenza** \> **Sottoscrizione assistenza**.</span><span class="sxs-lookup"><span data-stu-id="8b0d0-107">On the **Projects** form, on the **Action Pane**, on the **Manage** tab, click **Service** \> **Service subscription**.</span></span>

4.  <span data-ttu-id="8b0d0-110">Immettere nel modulo le restanti informazioni.</span><span class="sxs-lookup"><span data-stu-id="8b0d0-110">Enter the remaining information in the form.</span></span>

<span data-ttu-id="8b0d0-111">È ora possibile creare transazioni per la sottoscrizione.</span><span class="sxs-lookup"><span data-stu-id="8b0d0-111">You can now create subscription transactions for the subscription.</span></span>


> [!NOTE]
> <P><span data-ttu-id="8b0d0-112">Se è stata impostata una sequenza numerica per il gruppo di sottoscrizioni, quando si specifica tale gruppo viene generato automaticamente un ID per la sottoscrizione.</span><span class="sxs-lookup"><span data-stu-id="8b0d0-112">If you have set up a number sequence for the subscription group, a subscription ID is generated automatically for the subscription when you specify the subscription group.</span></span></P>
