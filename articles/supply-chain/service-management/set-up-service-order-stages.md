# <a name="set-up-service-order-stages"></a>Imposta le fasi dell'ordine di assistenza 

[!include [banner](../includes/banner.md)]


1.  Fare clic su **Gestione assistenza** \> **Impostazione** \> **Ordini di assistenza** \> **Fasi di assistenza**.

2.  Premere CTRL+N per creare un nuovo record.

3.  Nei campi **Fase di assistenza** e **Descrizione** immettere un ID e una descrizione per la fase di assistenza.

4.  Selezionare i parametri appropriati per la fase.

5.  Selezionare la fase padre della fase corrente oppure lasciare vuoto il campo **Padre** se si tratta della fase iniziale del processo di impostazione.


> [!NOTE]
> <P>Dopo aver salvato la fase, non è più possibile modificare il campo <STRONG>Padre</STRONG>. Sarà invece possibile eliminare il record e crearne uno nuovo selezionando un'opzione diversa nel campo <STRONG>Padre</STRONG>.</P>
> <P>Inoltre, è possibile creare una sola fase con un campo <STRONG>Padre</STRONG> vuoto. È consentita, pertanto, una sola fase iniziale.</P>
