# <a name="set-up-service-order-stages"></a><span data-ttu-id="f99c4-103">Imposta le fasi dell'ordine di assistenza</span><span class="sxs-lookup"><span data-stu-id="f99c4-103">Set up service order stages</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="f99c4-104">Fare clic su **Gestione assistenza** \> **Impostazione** \> **Ordini di assistenza** \> **Fasi di assistenza**.</span><span class="sxs-lookup"><span data-stu-id="f99c4-104">Click **Service management** \> **Setup** \> **Service orders** \> **Service stages**.</span></span>

2.  <span data-ttu-id="f99c4-105">Premere CTRL+N per creare un nuovo record.</span><span class="sxs-lookup"><span data-stu-id="f99c4-105">Press CTRL+N to create a new record.</span></span>

3.  <span data-ttu-id="f99c4-106">Nei campi **Fase di assistenza** e **Descrizione** immettere un ID e una descrizione per la fase di assistenza.</span><span class="sxs-lookup"><span data-stu-id="f99c4-106">In the **Service stage** and **Description** fields, specify a service stage ID and description.</span></span>

4.  <span data-ttu-id="f99c4-107">Selezionare i parametri appropriati per la fase.</span><span class="sxs-lookup"><span data-stu-id="f99c4-107">Select the appropriate parameters for the stage.</span></span>

5.  <span data-ttu-id="f99c4-108">Selezionare la fase padre della fase corrente oppure lasciare vuoto il campo **Padre** se si tratta della fase iniziale del processo di impostazione.</span><span class="sxs-lookup"><span data-stu-id="f99c4-108">Select the parent stage for the current stage or leave the **Parent** field blank if the stage is the initial stage in the stage setup.</span></span>
