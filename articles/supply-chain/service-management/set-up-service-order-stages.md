# <a name="set-up-service-order-stages"></a><span data-ttu-id="401a2-103">Imposta le fasi dell'ordine di assistenza</span><span class="sxs-lookup"><span data-stu-id="401a2-103">Set up service order stages</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="401a2-104">Selezionare **Gestione assistenza** \> **Impostazione** \> **Ordini di assistenza** \> **Fasi di assistenza**.</span><span class="sxs-lookup"><span data-stu-id="401a2-104">Go to **Service management** \> **Setup** \> **Service orders** \> **Service stages**.</span></span>

2.  <span data-ttu-id="401a2-105">Selezionare **Nuovo** per creare un nuovo record.</span><span class="sxs-lookup"><span data-stu-id="401a2-105">Select **New** to create a new record.</span></span>

3.  <span data-ttu-id="401a2-106">Nei campi **Fase di assistenza** e **Descrizione** immettere un ID e una descrizione per la fase di assistenza.</span><span class="sxs-lookup"><span data-stu-id="401a2-106">In the **Service stage** and **Description** fields, specify a service stage ID and description.</span></span>

4.  <span data-ttu-id="401a2-107">Selezionare i parametri appropriati per la fase.</span><span class="sxs-lookup"><span data-stu-id="401a2-107">Select the appropriate parameters for the stage.</span></span>

5.  <span data-ttu-id="401a2-108">Selezionare la fase padre della fase corrente oppure lasciare vuoto il campo **Padre** se si tratta della fase iniziale del processo di impostazione.</span><span class="sxs-lookup"><span data-stu-id="401a2-108">Select the parent stage for the current stage or leave the **Parent** field blank if the stage is the initial stage in the stage setup.</span></span>


> [!NOTE]
> <P><span data-ttu-id="401a2-109">Dopo aver salvato la fase, non è più possibile modificare il campo <STRONG>Padre</STRONG>.</span><span class="sxs-lookup"><span data-stu-id="401a2-109">You cannot modify the <STRONG>Parent</STRONG> field after you save the stage.</span></span> <span data-ttu-id="401a2-110">Sarà invece possibile eliminare il record e crearne uno nuovo selezionando un'opzione diversa nel campo <STRONG>Padre</STRONG>.</span><span class="sxs-lookup"><span data-stu-id="401a2-110">Instead, you can delete the record and create the record again with a different selection in the <STRONG>Parent</STRONG> field.</span></span></P>
> <P><span data-ttu-id="401a2-111">Inoltre, è possibile creare una sola fase con un campo <STRONG>Padre</STRONG> vuoto.</span><span class="sxs-lookup"><span data-stu-id="401a2-111">Also, you can only create one stage with a blank <STRONG>Parent</STRONG> field.</span></span> <span data-ttu-id="401a2-112">È consentita, pertanto, una sola fase iniziale.</span><span class="sxs-lookup"><span data-stu-id="401a2-112">That is, only one initial stage is permitted.</span></span></P>
