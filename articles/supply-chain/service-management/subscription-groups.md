# <a name="subscription-groups"></a>Gruppi di sottoscrizioni    

[!include [banner](../includes/banner.md)]


Le informazioni riportate di seguito vengono definite per il gruppo di sottoscrizioni e sono condivise da tutte le sottoscrizioni.

  - Eventuale codice di sequenza numerica della sottoscrizione.

  - Periodo di fatturazione per il quale vengono create le transazioni di sottoscrizione. Ad esempio, se il codice periodo **Fatturazione** nel modulo **Gruppi di sottoscrizioni** è impostato su **Trimestrale**, le transazioni relative a tutte le sottoscrizioni collegate al gruppo vengono create con frequenza trimestrale.

  - Frequenza con cui la sottoscrizione deve essere accumulata (se applicabile).

È inoltre possibile effettuare le seguenti attività per tutte le sottoscrizioni collegate a un gruppo:

  - Creazione di transazioni di sottoscrizione

  - Aggiornare il prezzo di base o l'indice delle sottoscrizioni

  - Registrare i ricavi sospesi per le transazioni di sottoscrizione
