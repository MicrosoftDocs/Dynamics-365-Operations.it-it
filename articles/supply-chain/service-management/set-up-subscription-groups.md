# <a name="set-up-subscription-groups"></a>Imposta gruppi di sottoscrizioni 

[!include [banner](../includes/banner.md)]


1.  Fare clic su **Gestione assistenza** \> **Impostazione** \> **Sottoscrizioni assistenza** \> **Gruppo di sottoscrizioni**.

2.  Premere CTRL+N o fare clic su **Nuovo** per creare un nuovo gruppo di sottoscrizioni.

3.  Nel campo **Gruppo di sottoscrizioni**, immettere un nome per il nuovo gruppo di sottoscrizioni.

4.  Nel campo **Fatturazione**, selezionare una frequenza di fatturazione per la sottoscrizione. Tale frequenza Ã¨ indicata da un codice periodo impostato nel modulo **Tipi di periodo**.

5.  Per accumulare ricavi per le transazioni di sottoscrizione, selezionare la casella di controllo **Accumula ricavi**, quindi selezionare una frequenza nel campo **Accumulo**. La frequenza di attribuzione per competenza si basa su un codice periodo impostato nel modulo **Periodi**.
