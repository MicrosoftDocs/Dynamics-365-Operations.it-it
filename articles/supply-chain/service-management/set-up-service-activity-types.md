# <a name="set-up-service-activity-types"></a>Impostare tipi di attività di assistenza tecnica 

[!include [banner](../includes/banner.md)]


1.  Fare clic su **Vendite e marketing** \> **Impostazioni** \> **Attività** \> **Tipi di attività**.
    
    oppure
    
    Fare clic su **Amministrazione organizzazione** \> **Impostazioni** \> **Attività** \> **Tipi di attività**.

2.  Nel modulo **Tipi di attività** premere CTRL+N per aggiungere una nuova riga di tipo di attività.

3.  Nel campo **Tipo** immettere un nome per il tipo di attività.

4.  Nel campo **Descrizione** immettere il testo descrittivo del tipo di attività.

5.  Chiudere il modulo per salvare le modifiche.
