[!include [banner](../includes/banner.md)]


<span data-ttu-id="ef3ea-104">È possibile impostare i codici smaltimento per specificare la modalità di elaborazione di un articolo che viene restituito da un cliente.</span><span class="sxs-lookup"><span data-stu-id="ef3ea-104">You can set up disposition codes to specify how to process an item that is returned by a customer.</span></span> <span data-ttu-id="ef3ea-105">Ad esempio, è possibile creare un codice smaltimento denominato **Riparazione e reso** per indicare che il reso verrà riparato e verrà restituito al cliente.</span><span class="sxs-lookup"><span data-stu-id="ef3ea-105">For example, create a disposition code named **Repair and return** to indicate that the returned item will be repaired and then returned to the customer.</span></span> <span data-ttu-id="ef3ea-106">Per ulteriori esempi dei codici smaltimento che in genere vengono utilizzati per gli articoli restituiti dai clienti, vedere [Specificare la modalità di smaltimento dei resi](specify-how-to-dispose-of-returned-items.md).</span><span class="sxs-lookup"><span data-stu-id="ef3ea-106">For more examples of disposition codes that are typically used for items that are returned by customers, see [Specify how to dispose of returned items](specify-how-to-dispose-of-returned-items.md).</span></span>

<span data-ttu-id="ef3ea-107">È inoltre possibile impostare un codice motivo che consente di spiegare perché un articolo è stato restituito.</span><span class="sxs-lookup"><span data-stu-id="ef3ea-107">You can also set up a reason code to help explain why an item was returned.</span></span> <span data-ttu-id="ef3ea-108">Per ulteriori informazioni sull'impostazione di codici motivo, vedere [Impostare codici motivo reso](set-up-return-reason-code.md).</span><span class="sxs-lookup"><span data-stu-id="ef3ea-108">For more information about reason codes, see [Set up return reason codes](set-up-return-reason-code.md).</span></span>

1.  <span data-ttu-id="ef3ea-109">Selezionare **Vendite e marketing** \> **Impostazione** \> **Ordini cliente** \> **Resi** \> **Codici smaltimento**.</span><span class="sxs-lookup"><span data-stu-id="ef3ea-109">Go to **Sales and marketing** \> **Setup** \> **Sales orders** \> **Returns** \> **Disposition codes**.</span></span>

2.  <span data-ttu-id="ef3ea-110">Selezionare **Nuovo** per creare un nuovo codice smaltimento.</span><span class="sxs-lookup"><span data-stu-id="ef3ea-110">Select **New** to create a new disposition code.</span></span>

3.  <span data-ttu-id="ef3ea-111">Immettere un nome descrittivo univoco, selezionare un'azione e immettere una descrizione per il codice smaltimento.</span><span class="sxs-lookup"><span data-stu-id="ef3ea-111">Enter a unique, descriptive name, select an action, and enter a description for the disposition code.</span></span>

4.  <span data-ttu-id="ef3ea-112">Se si desidera associare eventuali spese cliente al codice smaltimento, selezionare il pulsante **Spese varie** per aprire il modulo **Imposta spese varie**.</span><span class="sxs-lookup"><span data-stu-id="ef3ea-112">If you want to associate any customer charges with this disposition code, select the **Charges** button to open the **Set up charges** form.</span></span>

5.  <span data-ttu-id="ef3ea-113">Se si desidera definire codici esterni da abbinare ai codici smaltimento della società, selezionare il pulsante **Codici esterni** per aprire il modulo **Codici esterni**.</span><span class="sxs-lookup"><span data-stu-id="ef3ea-113">If you want to define any external codes to match with the company's own disposition codes, select the **External codes** button to open the **External codes** form.</span></span>
