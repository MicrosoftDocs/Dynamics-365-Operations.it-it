# <a name="move-service-order-lines"></a>Sposta righe ordine di assistenza 

[!include [banner](../includes/banner.md)]


1.  Fare clic su **Gestione assistenza** \> **Comune** \> **Ordini di assistenza** \> **Ordini di assistenza**.

2.  Fare doppio clic sull'ordine di assistenza contenente le righe che si desidera spostare in un altro ordine di assistenza.

3.  Nel modulo **Ordini di assistenza**, nel riquadro azioni, fare clic sul pulsante **Sposta righe**.

4.  Nel modulo **Sposta righe ordine di assistenza**, nel campo **Ordine di assistenza**, selezionare l'ordine di assistenza in cui si desidera spostare le righe.

5.  Selezionare la casella di controllo **Elimina ordine di assistenza vuoto** se si desidera eliminare l'ordine di assistenza da cui le righe sono state spostate.

6.  Fare clic su **OK**.
