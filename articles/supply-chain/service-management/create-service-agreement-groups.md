# <a name="create-service-agreement-groups"></a><span data-ttu-id="79eb8-103">Creare gruppi di contratti di assistenza</span><span class="sxs-lookup"><span data-stu-id="79eb8-103">Create service agreement groups</span></span> 

[!include [banner](../includes/banner.md)]

<span data-ttu-id="79eb8-104">I gruppi di contratti di assistenza sono utili per ordinare e filtrare i contratti di assistenza.</span><span class="sxs-lookup"><span data-stu-id="79eb8-104">Service agreement groups are useful for sorting and filtering service agreements.</span></span>

1. <span data-ttu-id="79eb8-105">Fare clic su **Gestione assistenza** \> **Impostazione** \> **Contratti di assistenza** \> **Gruppi contratti di assistenza**.</span><span class="sxs-lookup"><span data-stu-id="79eb8-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Service agreement groups**.</span></span>
2. <span data-ttu-id="79eb8-106">Premere **Alt+N** per creare un nuovo gruppo di contratti di assistenza.</span><span class="sxs-lookup"><span data-stu-id="79eb8-106">Press **Alt+N** to create a new service agreement group.</span></span>
3. <span data-ttu-id="79eb8-107">Nel campo **Gruppo contratti di assistenza** immettere un ID per il gruppo di contratti di assistenza.</span><span class="sxs-lookup"><span data-stu-id="79eb8-107">In the **Service agreement group** field, enter an ID for the service agreement group.</span></span>
4. <span data-ttu-id="79eb8-108">Nel campo **Descrizione** immettere una descrizione del gruppo di contratti di assistenza.</span><span class="sxs-lookup"><span data-stu-id="79eb8-108">In the **Description** field, enter a description for the service agreement group.</span></span>
5. <span data-ttu-id="79eb8-109">Chiudere la pagina per salvare le modifiche.</span><span class="sxs-lookup"><span data-stu-id="79eb8-109">Close the page to save your changes.</span></span>

> [!NOTE]
> <span data-ttu-id="79eb8-110">Se nel campo **Gruppo contratti di assistenza** della pagina **Parametri di gestione assistenza** è definito un gruppo di contratti di assistenza, tale gruppo verrà suggerito per tutti i nuovi contratti di assistenza creati.</span><span class="sxs-lookup"><span data-stu-id="79eb8-110">If a service agreement group is defined in the **Service agreement group** field of the **Service management parameters** page, the program suggests this group for all new service agreements that you create.</span></span>
