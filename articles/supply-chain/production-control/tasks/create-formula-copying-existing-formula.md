# <a name="create-a-formula-by-copying-from-an-existing-formula"></a>Creare una formula effettuando la copia da una formula esistente

[!include[task guide banner](../../includes/task-guide-banner.md)]

La procedura consente di creare rapidamente una nuova formula copiando da una formula esistente. È possibile utilizzare le società di dati dimostrativi USMF, USP2 o USPI per creare questa procedura.


## <a name="create-a-new-formula"></a>Creare una nuova formula
1. Passare a Formule.
2. Fare clic su Nuovo.
3. Digitare un valore nel campo Formula.
4. Digitare un valore nel campo Nome.
5. Nel campo Sito immettere o selezionare un valore.
6. Nel campo Gruppo di articoli immettere o selezionare un valore.
7. Fare clic su Salva.

## <a name="copy-from-an-existing-formula"></a>Copiare da una formula esistente
1. Fare clic su Copia.
2. Nel campo Numero di articoli immettere o selezionare un valore.
3. Nel campo Versione formula immettere o selezionare un valore.
4. Fare clic su OK.

## <a name="revise-the-copied-formula"></a>Rivedere la formula copiata
1. Nell'elenco trovare e selezionare il record desiderato.
2. Fare clic su Elimina.
3. Fare clic su Sì.

## <a name="approve-formula"></a>Approva formula
1. Fare clic su Approva formula.
2. Nel campo Approvato da immettere o selezionare un valore.
3. Fare clic su OK.
4. Chiudere la pagina.
