# <a name="copy-a-formula"></a>Copiare una formula

[!include [banner](../../includes/banner.md)]

Questa procedura riguarda la creazione di una formula che include gli stessi ingredienti di una formula esistente, ma con differenze secondarie. Per creare le righe della formula, è possibile utilizzare la funzione di copia per copiare una formula esistente con la maggior parte degli ingredienti necessari. È quindi possibile apportare le modifiche necessarie alle singole righe nella nuova versione. Tramite la funzione di copia, non è necessario creare più formule quasi identiche. La società di dati dimostrativi utilizzata per creare questa attività è USP2.


## <a name="create-a-formula"></a>Creazione di una formula
1. Andare a Gestione informazioni sul prodotto > Distinte base e formule > Formule.
2. Fare clic su Nuovo.
3. Digitare un valore nel campo Formula.
4. Digitare un valore nel campo Nome.
    * Digitare un nome significativo per la formula.  
5. Nel campo Sito fare clic sul pulsante a discesa per aprire la ricerca.
6. Nell'elenco fare clic sul collegamento nella riga selezionata.
7. Nel campo Gruppo di articoli fare clic sul pulsante a discesa per aprire la ricerca.
8. Nell'elenco trovare e selezionare il record desiderato.
9. Nell'elenco fare clic sul collegamento nella riga selezionata.
10. Fare clic su Salva.

## <a name="copy-formula-lines"></a>Copiare le righe formula
1. Nel riquadro azioni fare clic su Formula.
2. Fare clic su Copia.
3. Nel campo Numero articolo fare clic sul pulsante a discesa per aprire la ricerca.
4. Nell'elenco fare clic sul collegamento nella riga selezionata.
5. Nel campo Versione formula fare clic sul pulsante a discesa per aprire la ricerca.
6. Nell'elenco fare clic sul collegamento nella riga selezionata.
7. Fare clic su OK.

## <a name="adjust-copied-formula-lines"></a>Modificare le righe formula copiate
1. Nell'elenco contrassegnare la riga selezionata.
2. Fare clic su Elimina.
3. Fare clic su Sì.

## <a name="approve-formula"></a>Approva formula
1. Nel riquadro azioni fare clic su Formula.
2. Fare clic su Approva formula.
3. Nel campo Approvata da fare clic sul pulsante a discesa per aprire la ricerca.
4. Nell'elenco fare clic sul collegamento nella riga selezionata.
5. Fare clic su Seleziona.
6. Fare clic su OK.
