# <a name="report-progress-on-a-mobile-job-device"></a>Dichiarare lo stato di avanzamento di un dispositivo di lavoro mobile

[!include [task guide banner](../../includes/task-guide-banner.md)]

Questa procedura consente di iniziare e dichiarare lo stato di avanzamento di un processo di produzione nel modulo di registrazione per il dispositivo di processo.



Per eseguire questa procedura è necessario disporre del ruolo di amministratore di sistema o di operatore macchina associato all'account utente.

1. Passare a Controllo produzione > Esecuzione produzione > Dispositivo schede processo.
2. Nel campo WorkerTextField, immettere il badge di un lavoratore. Nel tipo di dati dimostrativi USMF digitare '123' per Christina Portra.
3. Fare clic su Accesso.
4. Fare clic sul pulsante Filtro.
5. Selezionare o deselezionare la casella di controllo Applica filtro configurazione. Se si imposta un filtro è possibile utilizzare l'unità di produzione 110 in USMF.
6. Nel campo Unità di produzione, selezionare i processi di produzione del gruppo di risorse con cui il lavoratore può lavorare.
7. Nell'elenco fare clic sul collegamento nella riga selezionata.
8. Fare clic su OK.
9. Fare clic sul pulsante Inizio processo.
10. Fare clic su OK.
11. Fare clic sul pulsante Segnala stato.
12. Fare clic su OK.
13. Fare clic sul pulsante Processo successivo.
14. Fare clic sul pulsante Assegnato per visualizzare una panoramica di tutti i processi di produzione.
15. Chiudere la pagina.
16. Fare clic sul pulsante Pausa.
17. Nell'elenco trovare e selezionare il record desiderato.
18. Fare clic su OK.
19. Fare clic sul pulsante Uscita.
20. Selezionare per disconnettersi.
21. Fare clic su OK.
22. Nel campo WorkerTextField, eseguire di nuovo l'accesso. È possibile selezionare il lavoratore '123' se si utilizzano i dati dimostrativi di USMF.
23. Fare clic su Accesso.
24. Fare clic su Interrompi pausa.
25. Fare clic sul pulsante Attività.
26. Scegliere Annulla.
27. Fare clic sul pulsante Uscita.
28. Selezionare per registrare l'uscita.
29. Fare clic su OK.
30. Selezionare un motivo per cui si esce in anticipo.
