# <a name="create-a-value-stream"></a>Creare un flusso del valore

[!include [task guide banner](../../includes/task-guide-banner.md)]

I flussi del valore sono unità operative che raggruppano uno o più flussi di produzione. Per definire un flusso di produzione, è innanzitutto necessario definire un flusso del valore.


## <a name="open-the-form"></a>Aprire il modulo
1. Andare a Controllo produzione > Impostazioni > Flusso di produzione snella > Flussi del valore.

## <a name="create-a-new-value-stream"></a>Creare un nuovo flusso del valore
1. Fare clic su Nuovo per aprire la finestra di dialogo a discesa.
2. Digitare un valore nel campo Nome.

## <a name="select-a-value-stream-manager"></a>Selezionare un responsabile flusso del valore
1. Nel campo Responsabile fare clic sul pulsante a discesa per aprire la ricerca.
    * Il responsabile del flusso del valore viene definito per documentare l'organizzazione.  
2. Nell'elenco trovare e selezionare il record desiderato.
3. Nell'elenco fare clic sul collegamento nella riga selezionata.

## <a name="save-and-close"></a>Salva e chiudi
1. Fare clic su Salva.
2. Chiudere la pagina.
3. Aggiorna la pagina.
