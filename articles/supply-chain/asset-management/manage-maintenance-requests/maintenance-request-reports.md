# <a name="maintenance-request-reports"></a>Report sulle richieste di intervento di manutenzione

[!include [banner](../../includes/banner.md)]

 

In Gestione cespiti, è possibile generare due report correlati a richieste di intervento di manutenzione. Un report mostra i dettagli e l'altro report conterrà un elenco che può essere utilizzato per la pianificazione e il follow-up.

## <a name="create-a-maintenance-request-details-report"></a>Creare un report Dettagli richieste di intervento di manutenzione

Nel report **Dettagli richieste di intervento di manutenzione** vengono visualizzate le varie informazioni correlate alle richieste di intervento di manutenzione.

1. Selezionare **Gestione cespiti** \> **Report** \> **Richieste di intervento di manutenzione** \> **Dettagli richieste di intervento di manutenzione**.
2. Nella Scheda dettaglio **Record da includere**, è possibile selezionare specifiche richieste di intervento di manutenzione da includere nel report.
3. Nella Scheda dettaglio **Esecuzione in background**, è possibile impostare la generazione del report come processo batch, secondo le esigenze.
4. Scegliere **OK** per generare il report.

Nella figura seguente è illustrato un esempio del report **Dettagli richieste di intervento di manutenzione**.

![Report dettagli richieste di intervento di manutenzione.](media/09-manage-maintenance-requests.png)

## <a name="create-a-maintenance-request-list-report"></a>Creare un report Elenco richieste di intervento di manutenzione

Nel report **Elenco richieste di intervento di manutenzione** viene visualizzato un elenco di tutte le richieste di intervento di manutenzione dello stesso tipo.

1. Selezionare **Gestione cespiti** \> **Report** \> **Richieste di intervento di manutenzione** \> **Elenco richieste di intervento di manutenzione**.
2. Nella Scheda dettaglio **Record da includere**, è possibile selezionare per definire quali richieste di intervento di manutenzione includere nel report.
3. Nella Scheda dettaglio **Esecuzione in background**, è possibile impostare la generazione del report come processo batch, secondo le esigenze.
4. Scegliere **OK** per generare il report.

Nella figura seguente è illustrato un esempio del report **Elenco richieste di intervento di manutenzione** per tutte le richieste di intervento di manutenzione attive.

![Report elenco richieste di intervento di manutenzione.](media/10-manage-maintenance-requests.png)
