# <a name="introduction-to-functional-locations"></a><span data-ttu-id="90231-103">Introduzione alle unità funzionali</span><span class="sxs-lookup"><span data-stu-id="90231-103">Introduction to functional locations</span></span>

[!include [banner](../../includes/banner.md)]

[!include [banner](../../includes/preview-banner.md)]

<span data-ttu-id="90231-104">Questo argomento fornisce una panoramica sulle unità funzionali in Gestione cespiti.</span><span class="sxs-lookup"><span data-stu-id="90231-104">This topic provides an overview of functional locations in Asset Management.</span></span> <span data-ttu-id="90231-105">Le unità funzionali sono elementi di una struttura tecnica, ad esempio unità funzionali in un sistema.</span><span class="sxs-lookup"><span data-stu-id="90231-105">Functional locations are elements of a technical structure, such as the functional units in a system.</span></span> <span data-ttu-id="90231-106">Le unità funzionali vengono create in modo gerarchico e vi si installano cespiti.</span><span class="sxs-lookup"><span data-stu-id="90231-106">Functional locations are created hierarchically, and you install assets on them.</span></span> <span data-ttu-id="90231-107">L'impostazione delle unità funzionali della società dipende dalle esigenze della società.</span><span class="sxs-lookup"><span data-stu-id="90231-107">The setup of functional locations in your company depends on the company's requirements.</span></span>

<span data-ttu-id="90231-108">Di seguito sono riportati alcuni esempi di come utilizzare le unità funzionali:</span><span class="sxs-lookup"><span data-stu-id="90231-108">Here are some examples of how you can use functional locations:</span></span>

- <span data-ttu-id="90231-109">**Funzionale** - le unità funzionali possono essere orientate all'utente e usate per gestire i cespiti con comportamento simile.</span><span class="sxs-lookup"><span data-stu-id="90231-109">**Functional** – The functional locations can be user-oriented and used to manage assets that have similar behavior.</span></span>

<span data-ttu-id="90231-112">Ogni unità funzionale viene gestita in modo indipendente in Gestione cespiti.</span><span class="sxs-lookup"><span data-stu-id="90231-112">Each functional location is managed independently in Asset Management.</span></span> <span data-ttu-id="90231-113">Di seguito vengono descritte alcune utili funzionalità delle unità funzionali:</span><span class="sxs-lookup"><span data-stu-id="90231-113">Here are some of the useful features of functional locations:</span></span>

- <span data-ttu-id="90231-114">Impostare specifiche delle unità funzionali.</span><span class="sxs-lookup"><span data-stu-id="90231-114">Set up functional location specifications.</span></span>
- <span data-ttu-id="90231-115">Impostare requisiti di specifiche per i cespiti.</span><span class="sxs-lookup"><span data-stu-id="90231-115">Set up asset specification requirements.</span></span>
- <span data-ttu-id="90231-116">Impostare sequenze di manutenzione per la manutenzione preventiva e reattiva.</span><span class="sxs-lookup"><span data-stu-id="90231-116">Set up maintenance sequences for preventive and reactive maintenance.</span></span>
- <span data-ttu-id="90231-117">Gestire i cespiti installati.</span><span class="sxs-lookup"><span data-stu-id="90231-117">Manage installed assets.</span></span>
- <span data-ttu-id="90231-118">Tenere traccia di richieste e ordini di lavoro attivi correlati ai cespiti installati.</span><span class="sxs-lookup"><span data-stu-id="90231-118">Track active requests and work orders that are related to installed assets.</span></span>
- <span data-ttu-id="90231-119">Tenere traccia di problemi registrati nei cespiti.</span><span class="sxs-lookup"><span data-stu-id="90231-119">Track faults that are registered on assets.</span></span>
- <span data-ttu-id="90231-120">Tenere traccia dei costi della manutenzione dei cespiti correlati a una unità funzionale in qualsiasi dato momento.</span><span class="sxs-lookup"><span data-stu-id="90231-120">Track maintenance costs on the assets that are related to a functional location at any given time.</span></span>

<span data-ttu-id="90231-121">Le unità funzionali forniscono la tracciabilità dei cespiti in relazione a richieste, ordini di lavoro, alle registrazioni di problemi, valutazione di condizioni, registrazioni di interruzioni di produzione e registrazioni di contatori dei cespiti.</span><span class="sxs-lookup"><span data-stu-id="90231-121">Functional locations provide traceability of assets in relation to requests, work orders, fault registrations, condition assessments, production stop registrations, and asset counter registrations.</span></span>

> [!NOTE]
> <span data-ttu-id="90231-122">Anche se un cespite viene installato in varie unità funzionali durante la sua vita, i costi possono essere correlati a ogni ubicazione.</span><span class="sxs-lookup"><span data-stu-id="90231-122">Even if an asset is installed on various functional locations during its lifetime, the costs can be related to each location.</span></span> <span data-ttu-id="90231-123">In altre parole, i costi dei cespiti sono sempre correlati all'unità funzionale in cui il cespite è installato in un dato momento.</span><span class="sxs-lookup"><span data-stu-id="90231-123">In other words, asset costs are always related to the functional location that the asset was installed on at a given time.</span></span>

<span data-ttu-id="90231-124">Le unità funzionali **non** sono flessibili.</span><span class="sxs-lookup"><span data-stu-id="90231-124">Functional locations are **not** flexible.</span></span> <span data-ttu-id="90231-125">Di conseguenza, dopo aver impostato una gerarchia di unità funzionali, non è possibile spostare le ubicazioni al suo interno.</span><span class="sxs-lookup"><span data-stu-id="90231-125">Therefore, after you set up a functional location hierarchy, you can't move locations around in it.</span></span> 

<span data-ttu-id="90231-126">Dopo aver creato una gerarchia di unità funzionali, sarà necessario installarvi cespiti.</span><span class="sxs-lookup"><span data-stu-id="90231-126">After you create a functional location hierarchy, the next step is to install assets on it.</span></span> <span data-ttu-id="90231-127">Per ulteriori informazioni, vedere [Installare i cespiti nelle unità funzionali](../functional-locations/install-objects-on-functional-locations.md).</span><span class="sxs-lookup"><span data-stu-id="90231-127">For more information, see [Install assets on functional locations](../functional-locations/install-objects-on-functional-locations.md).</span></span>
