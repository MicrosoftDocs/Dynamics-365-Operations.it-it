# <a name="work-order-report"></a><span data-ttu-id="14063-103">Report di ordine di lavoro</span><span class="sxs-lookup"><span data-stu-id="14063-103">Work order report</span></span>

[!include [banner](../../includes/banner.md)]


<span data-ttu-id="14063-104">È possibile generare un report di ordine di lavoro che mostra informazioni dettagliate su uno o più ordini di lavoro selezionati.</span><span class="sxs-lookup"><span data-stu-id="14063-104">You can generate a work order report that shows detailed information about one or more work orders that you select.</span></span>

1. <span data-ttu-id="14063-105">Selezionare **Gestione cespiti** > **Report** > **Ordini di lavoro** > **Report di ordine di lavoro**.</span><span class="sxs-lookup"><span data-stu-id="14063-105">Select **Asset management** > **Reports** > **Work orders** > **Work order report**.</span></span>

2. <span data-ttu-id="14063-106">Impostare gli interruttori su "Sì" o "No" come necessario per determinare i dettagli da includere nel report.</span><span class="sxs-lookup"><span data-stu-id="14063-106">Select "Yes"/"No" on the toggle buttons as you require, to determine the details to be included in the report.</span></span>  
    - <span data-ttu-id="14063-107">Nella sezione **Impostazioni di stampa**, è possibile scegliere se includere allegati dalla configurazione dei tipi di processo di manutenzione correlata.</span><span class="sxs-lookup"><span data-stu-id="14063-107">In the **Print settings** section, you can select whether to include attachments from the setup of the related maintenance job type.</span></span>

3. <span data-ttu-id="14063-108">Nella Scheda dettaglio **Record da includere**, è possibile filtrare il contenuto del report per **Ordine di lavoro**.</span><span class="sxs-lookup"><span data-stu-id="14063-108">On the **Records to include** FastTab, you can filter the contents of the report by **Work order**.</span></span>

5. <span data-ttu-id="14063-110">Scegliere **OK** per generare il report.</span><span class="sxs-lookup"><span data-stu-id="14063-110">Select **OK** to generate the report.</span></span>
