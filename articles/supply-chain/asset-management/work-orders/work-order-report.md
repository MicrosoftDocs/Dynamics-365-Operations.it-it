<span data-ttu-id="d7165-104">È possibile generare un report di ordine di lavoro che mostra informazioni dettagliate su uno o più ordini di lavoro selezionati.</span><span class="sxs-lookup"><span data-stu-id="d7165-104">You can generate a work order report that shows detailed information about one or more work orders that you select.</span></span>
