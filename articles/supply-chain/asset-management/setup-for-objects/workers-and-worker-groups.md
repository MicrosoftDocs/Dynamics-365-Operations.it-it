## <a name="create-workers"></a><span data-ttu-id="41039-110">Crea lavoratori</span><span class="sxs-lookup"><span data-stu-id="41039-110">Create workers</span></span>

1. <span data-ttu-id="41039-111">Selezionare **Gestione cespiti** \> **Impostazione** \> **Lavoratori** \> **Lavoratori**.</span><span class="sxs-lookup"><span data-stu-id="41039-111">Select **Asset management** \> **Setup** \> **Workers** \> **Workers**.</span></span>
2. <span data-ttu-id="41039-112">Selezionare **Nuovo** per aggiungere un lavoratore all'elenco.</span><span class="sxs-lookup"><span data-stu-id="41039-112">Select **New** to add a worker to the list.</span></span>
3. <span data-ttu-id="41039-113">Selezionare il lavoratore nel campo **Lavoratore**.</span><span class="sxs-lookup"><span data-stu-id="41039-113">In the **Worker** field, select the worker.</span></span>
4. <span data-ttu-id="41039-114">Impostare **Attivo** su  **Sì** per la programmazione del lavoratore negli ordini di lavoro.</span><span class="sxs-lookup"><span data-stu-id="41039-114">Set the **Active** option to **Yes** to schedule the worker on work orders.</span></span>

    <span data-ttu-id="41039-115">Nella Scheda dettaglio **Generale**, i campi **Descrizione** e **Risorsa** vengono completati automaticamente se una risorsa è stata selezionata per il lavoratore.</span><span class="sxs-lookup"><span data-stu-id="41039-115">On the **General** FastTab, the **Resource** and **Description** fields are automatically filled in if a resource has been selected for the worker.</span></span> <span data-ttu-id="41039-116">Il campo **Calendario** verrà anche compilato automaticamente, a condizione di avere impostato il lavoratore come risorsa e allocato un calendario a tale risorsa nella pagina **Risorse** (**Amministrazione organizzazione** \> **Risorse** \> **Risorse**).</span><span class="sxs-lookup"><span data-stu-id="41039-116">The **Calendar** field is also automatically filled in, provided that you've set up the worker as a resource and allocated a calendar to that resource on the **Resources** page (**Organization administration** \> **Resources** \> **Resources**).</span></span>

5. <span data-ttu-id="41039-117">Nella Scheda dettaglio **Gruppi**, selezionare **Aggiungi** e quindi selezionare un gruppo di addetti alla manutenzione per il lavoratore.</span><span class="sxs-lookup"><span data-stu-id="41039-117">On the **Groups** FastTab, select **Add**, and then select a maintenance worker group for the worker.</span></span> <span data-ttu-id="41039-118">Un lavoratore può essere associato a più di un gruppo.</span><span class="sxs-lookup"><span data-stu-id="41039-118">A worker can be affiliated with more than one group.</span></span>
6. <span data-ttu-id="41039-119">Nell'impostazione standard, il rapporto di un lavoratore con un gruppo è valido a partire dalla data in cui è aggiunto al gruppo e non ha scadenza.</span><span class="sxs-lookup"><span data-stu-id="41039-119">In the standard setup, a worker's affiliation with a group is effective from the date when you add the group, and it never expires.</span></span> <span data-ttu-id="41039-120">Questa data viene visualizzata nel campo **Validità**.</span><span class="sxs-lookup"><span data-stu-id="41039-120">This date is shown in the **Effective** field.</span></span> <span data-ttu-id="41039-121">Per visualizzare il campo **Validità**, selezionare **Visualizza** \> **Tutti**.</span><span class="sxs-lookup"><span data-stu-id="41039-121">To see the **Effective** field, select **View** \> **All**.</span></span> <span data-ttu-id="41039-122">Se il rapporto del lavoratore con un gruppo deve essere limitato a un periodo specifico, utilizzare i campi **Validità** e **Scadenza** per definire il periodo.</span><span class="sxs-lookup"><span data-stu-id="41039-122">If the worker's affiliation with a group should be limited to a specific period, use the **Effective** and **Expiration** fields to define the period.</span></span>
7. <span data-ttu-id="41039-123">Nella Scheda dettaglio **Unità funzionali**, selezionare **Aggiungi** e quindi selezionare un'unità funzionale per il lavoratore.</span><span class="sxs-lookup"><span data-stu-id="41039-123">On the **Functional locations** FastTab, select **Add**, and then select a functional location for the maintenance worker.</span></span> <span data-ttu-id="41039-124">Inoltre specificare quale ubicazione rappresenta l'unità funzionale principale per il lavoratore.</span><span class="sxs-lookup"><span data-stu-id="41039-124">Also specify which location is the primary functional location for the worker.</span></span>

    > [!NOTE]
    > <span data-ttu-id="41039-125">Quando si aggiungono le unità funzionali al lavoratore, tutti i cespiti attivi correlati alle unità funzionali vengono visualizzati nelle diverse voci di menu, ad esempio **Cespiti attivi personali** e **Unità funzionali attive**.</span><span class="sxs-lookup"><span data-stu-id="41039-125">When you add functional locations to a worker, all active assets that are related to those functional locations appear on various menu items, such as **My active assets** and **My active functional locations**.</span></span> <span data-ttu-id="41039-126">Vengono inoltre visualizzati nelle ricerche cespiti che verranno visualizzate quando si crea un nuovo cespite, richiesta di intervento di manutenzione, o ordine di lavoro.</span><span class="sxs-lookup"><span data-stu-id="41039-126">They also appear in the asset lookups that are shown when you create a new asset, maintenance request, or work order.</span></span>

    <span data-ttu-id="41039-127">I campi nella Scheda dettaglio **Dettagli** indicano il numero di gruppi di addetti alla manutenzione e delle unità funzionali a cui l'addetto alla manutenzione è correlato.</span><span class="sxs-lookup"><span data-stu-id="41039-127">The fields on the **Details** FastTab show the number of maintenance worker groups and functional locations that the selected maintenance worker is related to.</span></span>

## <a name="create-worker-groups"></a><span data-ttu-id="41039-128">Creare gruppi di lavoratori</span><span class="sxs-lookup"><span data-stu-id="41039-128">Create worker groups</span></span>

1. <span data-ttu-id="41039-129">Selezionare **Gestione cespiti** \> **Impostazione** \> **Lavoratori** \> **Gruppi di addetti alla manutenzione**.</span><span class="sxs-lookup"><span data-stu-id="41039-129">Select **Asset management** \> **Setup** \> **Workers** \> **Maintenance worker groups**.</span></span>
2. <span data-ttu-id="41039-130">Selezionare **Nuovo** per aggiungere un gruppo di lavoratori all'elenco.</span><span class="sxs-lookup"><span data-stu-id="41039-130">Select **New** to add a worker group to the list.</span></span>
3. <span data-ttu-id="41039-131">Nel campo **Gruppo di addetti alla manutenzione** immettere un ID gruppo.</span><span class="sxs-lookup"><span data-stu-id="41039-131">In the **Maintenance worker group** field, enter a group ID.</span></span>
4. <span data-ttu-id="41039-132">Nel campo **Nome** immettere un nome.</span><span class="sxs-lookup"><span data-stu-id="41039-132">In the **Name** field, enter a name.</span></span>
5. <span data-ttu-id="41039-133">Nella Scheda dettaglio **Lavoratori**, selezionare **Aggiungi** e quindi selezionare un addetto alla manutenzione per il gruppo di lavoratori.</span><span class="sxs-lookup"><span data-stu-id="41039-133">On the **Workers** FastTab, select **Add**, and then select a maintenance worker for the worker group.</span></span> <span data-ttu-id="41039-134">Per informazioni sui campi **Validità** e **Scadenza**, vedere il passaggio 6 nella procedura precedente.</span><span class="sxs-lookup"><span data-stu-id="41039-134">For information about the **Effective** and **Expiration** fields, see step 6 in the previous procedure.</span></span>

    <span data-ttu-id="41039-139">Il campo nella Scheda dettaglio **Dettagli** mostra il numero di addetti alla manutenzione impostati nel gruppo di addetti alla manutenzione selezionato.</span><span class="sxs-lookup"><span data-stu-id="41039-139">The field on the **Details** FastTab shows the number of maintenance workers that have been set up on the selected maintenance worker group.</span></span>
