## <a name="create-counter-types-for-asset-counter-registrations"></a><span data-ttu-id="a95c8-119">Creare i tipi di contatore per le registrazioni dei contatori dei cespiti</span><span class="sxs-lookup"><span data-stu-id="a95c8-119">Create counter types for asset counter registrations</span></span>

6. <span data-ttu-id="a95c8-125">Selezionare "Sì" sull'interruttore **Eredita valori contatore** se i cespiti figlio in una struttura dei cespiti devono ereditare automaticamente le registrazioni di contatore effettuate sul cespite padre.</span><span class="sxs-lookup"><span data-stu-id="a95c8-125">Select "Yes" on the **Inherit counter values** toggle button if child assets in an asset structure should automatically inherit counter registrations made on the parent asset.</span></span>

8. <span data-ttu-id="a95c8-129">Nel campo **Scostamento sopra**, inserire il livello percentuale superiore per convalidare se le registrazioni manuali di contatore sono comprese in un determinato intervallo previsto.</span><span class="sxs-lookup"><span data-stu-id="a95c8-129">In the **Deviation over** field, insert the upper level in percent for validating if manual counter registrations are within an expected range.</span></span> <span data-ttu-id="a95c8-130">La convalida è basata su un aumento lineare nelle registrazioni di contatore esistenti.</span><span class="sxs-lookup"><span data-stu-id="a95c8-130">The validation is based on a linear increase in existing counter registrations.</span></span>
9. <span data-ttu-id="a95c8-131">Nel campo **Scostamento sotto**, inserire il livello percentuale inferiore per convalidare se le registrazioni manuali di contatore sono comprese in un determinato intervallo previsto.</span><span class="sxs-lookup"><span data-stu-id="a95c8-131">In the **Deviation under** field, insert the lower level in percent for validating if manual counter registrations are within an expected range.</span></span> <span data-ttu-id="a95c8-132">La convalida è basata su una riduzione lineare nelle registrazioni di contatore esistenti.</span><span class="sxs-lookup"><span data-stu-id="a95c8-132">The validation is based on a linear decrease in existing counter registrations.</span></span>
10. <span data-ttu-id="a95c8-133">Nel campo **Tipo**, selezionare il tipo di messaggio (informazioni, avviso, errore) da visualizzare se gli scostamenti dall'intervallo definito si verificano quando si eseguono registrazioni manuali di contatore.</span><span class="sxs-lookup"><span data-stu-id="a95c8-133">In the **Type** field, select the type of message (information, warning, error) to be shown if deviations outside the defined range occur when you make manual counter registrations.</span></span>

12. <span data-ttu-id="a95c8-135">Nella Scheda dettaglio **Contatori di cespiti correlati**, aggiungere il contatore che si desidera aggiornare automaticamente questa il contatore viene aggiornato.</span><span class="sxs-lookup"><span data-stu-id="a95c8-135">On the **Related asset counters** FastTab, add the counter that you want to be automatically updated when this counter is updated.</span></span>
