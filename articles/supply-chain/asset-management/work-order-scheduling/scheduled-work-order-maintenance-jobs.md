## <a name="view-scheduled-work-order-maintenance-jobs"></a><span data-ttu-id="c7991-107">Visualizzare processi di manutenzione di ordine di lavoro programmati</span><span class="sxs-lookup"><span data-stu-id="c7991-107">View scheduled work order maintenance jobs</span></span>

1. <span data-ttu-id="c7991-108">Fare clic su **Gestione cespiti** > **Comune** > **Ordini di lavoro** > **Processi di manutenzione ordine di lavoro programmati**.</span><span class="sxs-lookup"><span data-stu-id="c7991-108">Click **Asset management** > **Common** > **Work orders** > **Scheduled work order maintenance jobs**.</span></span> <span data-ttu-id="c7991-109">Viene visualizzato un elenco di tutti gli ordini di lavoro impostati il cui stato del ciclo di vita è "Programmato" o "In corso".</span><span class="sxs-lookup"><span data-stu-id="c7991-109">You see a list of all work orders set to work order lifecycle state "Scheduled" or "In progress".</span></span>

2. <span data-ttu-id="c7991-110">È possibile ordinare l'elenco, ad esempio, per addetto alla manutenzione.</span><span class="sxs-lookup"><span data-stu-id="c7991-110">You can sort the list, for example, by maintenance worker.</span></span> <span data-ttu-id="c7991-111">È inoltre possibile utilizzare il filtro per limitare l'elenco e visualizzare gli ordini di lavoro assegnati a una specifico addetto alla manutenzione o risorsa.</span><span class="sxs-lookup"><span data-stu-id="c7991-111">You can also use the filter to limit the list to display work orders allocated to a specific resource or maintenance worker.</span></span>

3. <span data-ttu-id="c7991-112">È possibile visualizzare le note nell'ordine di lavoro e, se necessario, aggiungere note selezionando il processo di ordine di lavoro e facendo clic su **Note**.</span><span class="sxs-lookup"><span data-stu-id="c7991-112">You can see notes on the work order and, if required, add new notes by selecting the work order job, and then click **Notes**.</span></span>

4. <span data-ttu-id="c7991-113">Se si desidera assegnare un addetto alla manutenzione a un ordine di lavoro, selezionare l'ordine di lavoro e fare clic su **Ordine di lavoro**.</span><span class="sxs-lookup"><span data-stu-id="c7991-113">If you want to allocate one maintenance worker to a work order, select the work order, and then click **Work order**.</span></span>
