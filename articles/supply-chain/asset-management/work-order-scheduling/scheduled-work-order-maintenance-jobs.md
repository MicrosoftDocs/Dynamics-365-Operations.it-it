## <a name="view-scheduled-work-order-maintenance-jobs"></a><span data-ttu-id="acccd-108">Visualizzare processi di manutenzione di ordine di lavoro programmati</span><span class="sxs-lookup"><span data-stu-id="acccd-108">View scheduled work order maintenance jobs</span></span>

2. <span data-ttu-id="acccd-111">È possibile ordinare l'elenco, ad esempio, per addetto alla manutenzione.</span><span class="sxs-lookup"><span data-stu-id="acccd-111">You can sort the list, for example, by maintenance worker.</span></span> <span data-ttu-id="acccd-112">È inoltre possibile utilizzare il filtro per limitare l'elenco e visualizzare gli ordini di lavoro assegnati a una specifico addetto alla manutenzione o risorsa.</span><span class="sxs-lookup"><span data-stu-id="acccd-112">You can also use the filter to limit the list to display work orders allocated to a specific resource or maintenance worker.</span></span>
