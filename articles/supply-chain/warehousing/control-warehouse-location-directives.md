<span data-ttu-id="dbe54-124">È possibile impostare una query per verificare se un determinato modello di lavoro è utilizzato.</span><span class="sxs-lookup"><span data-stu-id="dbe54-124">You can set ## <a name="location-directives"></a><span data-ttu-id="dbe54-134">Direttive ubicazione</span><span class="sxs-lookup"><span data-stu-id="dbe54-134">Location directives</span></span>
<span data-ttu-id="dbe54-135">Le direttive ubicazione sono regole che aiutano a identificare le ubicazioni di prelievo e stoccaggio per il movimento scorte.</span><span class="sxs-lookup"><span data-stu-id="dbe54-135">Location directives are rules that help identify pick and put locations for inventory movement.</span></span> <span data-ttu-id="dbe54-136">Ad esempio, in una transazione dell'ordine cliente, la direttiva ubicazione determina il punto di prelievo e il punto di stoccaggio degli articoli.</span><span class="sxs-lookup"><span data-stu-id="dbe54-136">For example, in a sales order transaction, a location directive determines where the items will be picked, and where the picked items will be put.</span></span> <span data-ttu-id="dbe54-137">Le direttive di ubicazione sono costituite dall'intestazione e dalle righe associate e possono essere create nella pagina **Direttive ubicazione**.</span><span class="sxs-lookup"><span data-stu-id="dbe54-137">Location directives consist of a header and associated lines, and you create them on the **Location directives** page.</span></span> 

<span data-ttu-id="dbe54-138">Nell'intestazione, ogni direttiva ubicazione deve essere associata con un *tipo ordine di lavoro* per specificare il tipo di transazione di magazzino per cui la direttiva verrà utilizzata, ad esempio ordini cliente, rifornimento o prelievo delle materie prime.</span><span class="sxs-lookup"><span data-stu-id="dbe54-138">On the header, each location directive must be associated with a *work order type* that specifies the type of inventory transaction that the directive will be used for, such as sales orders, replenishment, or raw material picking.</span></span> <span data-ttu-id="dbe54-139">Il *tipo di lavoro* specifica se la direttiva ubicazione verrà utilizzata per il prelievo, per la collocazione o per un altro processo di magazzino, ad esempio il conteggio o le modifiche di stato delle scorte.</span><span class="sxs-lookup"><span data-stu-id="dbe54-139">The *work type* specifies whether the location directive will be used for picking or putting work, or for some other warehouse process, such as counting or inventory status changes.</span></span> <span data-ttu-id="dbe54-140">È inoltre necessario specificare il *sito* e un *magazzino*.</span><span class="sxs-lookup"><span data-stu-id="dbe54-140">You must also specify a *site* and a *warehouse*.</span></span> <span data-ttu-id="dbe54-141">Un *codice della direttiva* specificato nell'intestazione può essere utilizzato per collegare la direttiva ubicazione a uno o più modelli di lavoro.</span><span class="sxs-lookup"><span data-stu-id="dbe54-141">A *directive code* that you specify on the header can be used to link the location directive to one or more work templates.</span></span>

<span data-ttu-id="dbe54-145">Le righe delle direttive ubicazione impostano restrizioni aggiuntive sull'applicazione delle regole di individuazione dell'ubicazione.</span><span class="sxs-lookup"><span data-stu-id="dbe54-145">The location directive lines set additional restrictions on the application of the location finding rules.</span></span> <span data-ttu-id="dbe54-146">È possibile specificare una quantità minima e una quantità massima a cui la direttiva deve essere applicata e specificare che la direttiva deve essere destinata a un'unità di magazzino specifica.</span><span class="sxs-lookup"><span data-stu-id="dbe54-146">You can specify a minimum quantity and a maximum quantity that the directive should apply to, and you can specify that the directive should be for a specific inventory unit.</span></span> <span data-ttu-id="dbe54-147">Ad esempio, se l'unità di misura sono i pallet, gli articoli in pallet potranno essere stoccati in un'ubicazione specifica.</span><span class="sxs-lookup"><span data-stu-id="dbe54-147">For example, if the unit of measure is pallets, the items in pallets can be put in a specific location.</span></span> <span data-ttu-id="dbe54-148">È inoltre possibile specificare se la quantità può essere suddivisa tra più ubicazioni.</span><span class="sxs-lookup"><span data-stu-id="dbe54-148">You can also specify whether the quantity can be split across multiple locations.</span></span> <span data-ttu-id="dbe54-149">Come l'intestazione della direttiva ubicazione, ciascuna riga direttiva ubicazione ha un numero progressivo che determina l'ordine in cui vengono valutate le righe.</span><span class="sxs-lookup"><span data-stu-id="dbe54-149">Like the location directive header, each location directive line has a sequence number that determines the order that the lines are assessed in.</span></span>

<span data-ttu-id="dbe54-156">Per questo esempio, considereremo un processo di ordine fornitore in cui la direttiva ubicazione deve individuare capacità libera all'interno di un magazzino per gli articoli di magazzino registrati immediatamente alla banchina di entrata.</span><span class="sxs-lookup"><span data-stu-id="dbe54-156">For this example, we will consider a purchase order process where the location directive must find free capacity within a warehouse for inventory items that have just been registered at the receiving dock.</span></span> <span data-ttu-id="dbe54-157">Innanzitutto, si cerca capacità libera all'interno del magazzino consolidando le scorte disponibili esistenti.</span><span class="sxs-lookup"><span data-stu-id="dbe54-157">First, we want to try to find free capacity within the warehouse by consolidating with existing on-hand inventory.</span></span> <span data-ttu-id="dbe54-158">Se il consolidamento non è possibile, si cerca di individuare un'ubicazione vuota.</span><span class="sxs-lookup"><span data-stu-id="dbe54-158">If consolidation isn't possible, we then want to find an empty location.</span></span> 

<span data-ttu-id="dbe54-159">Per questo scenario, dobbiamo definire due azioni di direttiva ubicazione.</span><span class="sxs-lookup"><span data-stu-id="dbe54-159">For this scenario, we must define two location directive actions.</span></span> <span data-ttu-id="dbe54-160">La prima azione nella sequenza deve utilizzare la strategia **Consolida** e la seconda deve utilizzare la strategia **Ubicazione vuota senza alcun lavoro in entrata**.</span><span class="sxs-lookup"><span data-stu-id="dbe54-160">The first action in the sequence must use the **Consolidate** strategy, and the second should use the **Empty location with no incoming work** strategy.</span></span> <span data-ttu-id="dbe54-161">A meno che definiamo una terza azione per gestire uno scenario di overflow, sono possibili due risultati in caso non vi sia più capacità in magazzino: il lavoro può essere creato anche se non è definita un'ubicazione o il processo di creazione di lavoro può interrompersi.</span><span class="sxs-lookup"><span data-stu-id="dbe54-161">Unless we define a third action to handle an overflow scenario, two outcomes are possible when there is no more capacity in the warehouse: work can be created even though no locations are defined, or the work creation process can fail.</span></span> <span data-ttu-id="dbe54-162">Il risultato viene determinato dall'impostazione nella pagina **Errori direttiva ubicazione**, in cui è possibile decidere se selezionare l'opzione **Interrompi lavoro in caso di errore direttiva ubicazione** per ogni tipo di ordine di lavoro.</span><span class="sxs-lookup"><span data-stu-id="dbe54-162">The outcome is determined by the setup on the **Location directive failures** page, where you can decide whether to select the **Stop work on location directive failure** option for each work order type.</span></span>
