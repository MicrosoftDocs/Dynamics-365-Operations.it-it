<span data-ttu-id="fa17f-123">Le righe di lavoro rappresentano le attività fisiche necessarie per elaborare il lavoro.</span><span class="sxs-lookup"><span data-stu-id="fa17f-123">The work lines represent the physical tasks that are required to process the work.</span></span> <span data-ttu-id="fa17f-124">Ad esempio, per un processo di magazzino in uscita, può essere presente una riga per il prelievo degli articoli dal magazzino e un'altra riga per la collocazione degli articoli in un'area di gestione temporanea.</span><span class="sxs-lookup"><span data-stu-id="fa17f-124">For example, for an outbound warehouse process, there might be one line for picking up the items in the warehouse and another line for putting those items into a staging area.</span></span> <span data-ttu-id="fa17f-125">Può essere presente quindi una riga aggiuntiva per il prelievo degli articoli dall'area di gestione temporanea e un'altra riga per la collocazione degli articoli in un camion come parte del processo di caricamento.</span><span class="sxs-lookup"><span data-stu-id="fa17f-125">There can then be an additional line for picking the items from staging and another line for putting the items into a truck as part of the loading process.</span></span> <span data-ttu-id="fa17f-126">È possibile impostare un *codice della direttiva* nelle righe del modello di lavoro.</span><span class="sxs-lookup"><span data-stu-id="fa17f-126">You can set a *directive code* on work template lines.</span></span> <span data-ttu-id="fa17f-127">Un codice della direttiva è collegato alla direttiva di ubicazione e quindi garantisce che il lavoro del magazzino venga elaborato nell'ubicazione corretta del magazzino.</span><span class="sxs-lookup"><span data-stu-id="fa17f-127">A directive code is linked to a location directive, and therefore helps ensure that the warehouse work is processed in the correct location in the warehouse.</span></span>

<span data-ttu-id="fa17f-128">È possibile impostare una query per verificare se un determinato modello di lavoro è utilizzato.</span><span class="sxs-lookup"><span data-stu-id="fa17f-128">You can set modello di lavoro solo in un magazzino specifico.</span><span class="sxs-lookup"><span data-stu-id="fa17f-129">For example, you can set a limitation so that a particular template can be used for work in.</span></span> <span data-ttu-id="fa17f-132">Di conseguenza, se si dispone di una query molto specifica per un determinato modello di lavoro, è necessario assegnare un numero progressivo ## <a name="location-directives"></a><span data-ttu-id="fa17f-139">Direttive ubicazione</span><span class="sxs-lookup"><span data-stu-id="fa17f-139">Location directives</span></span>

<span data-ttu-id="fa17f-140">Le direttive ubicazione sono regole che aiutano a identificare le ubicazioni di prelievo e stoccaggio per il movimento scorte.</span><span class="sxs-lookup"><span data-stu-id="fa17f-140">Location directives are rules that help identify pick and put locations for inventory movement.</span></span> <span data-ttu-id="fa17f-141">Ad esempio, in una transazione dell'ordine cliente, la direttiva ubicazione determina il punto di prelievo e il punto di stoccaggio degli articoli.</span><span class="sxs-lookup"><span data-stu-id="fa17f-141">For example, in a sales order transaction, a location directive determines where the items will be picked, and where the picked items will be put.</span></span> <span data-ttu-id="fa17f-142">Le direttive di ubicazione sono costituite dall'intestazione e dalle righe associate e possono essere create nella pagina **Direttive ubicazione**.</span><span class="sxs-lookup"><span data-stu-id="fa17f-142">Location directives consist of a header and associated lines, and you create them on the **Location directives** page.</span></span>

<span data-ttu-id="fa17f-143">Nell'intestazione, ogni direttiva ubicazione deve essere associata con un *tipo ordine di lavoro* per specificare il tipo di transazione di magazzino per cui la direttiva verrà utilizzata, ad esempio ordini cliente, rifornimento o prelievo delle materie prime.</span><span class="sxs-lookup"><span data-stu-id="fa17f-143">On the header, each location directive must be associated with a *work order type* that specifies the type of inventory transaction that the directive will be used for, such as sales orders, replenishment, or raw material picking.</span></span> <span data-ttu-id="fa17f-144">Il *tipo di lavoro* specifica se la direttiva ubicazione verrà utilizzata per il prelievo, per la collocazione o per un altro processo di magazzino, ad esempio il conteggio o le modifiche di stato delle scorte.</span><span class="sxs-lookup"><span data-stu-id="fa17f-144">The *work type* specifies whether the location directive will be used for picking or putting work, or for some other warehouse process, such as counting or inventory status changes.</span></span> <span data-ttu-id="fa17f-145">È inoltre necessario specificare il *sito* e un *magazzino*.</span><span class="sxs-lookup"><span data-stu-id="fa17f-145">You must also specify a *site* and a *warehouse*.</span></span> <span data-ttu-id="fa17f-146">Un *codice della direttiva* specificato nell'intestazione può essere utilizzato per collegare la direttiva ubicazione a uno o più modelli di lavoro.</span><span class="sxs-lookup"><span data-stu-id="fa17f-146">A *directive code* that you specify on the header can be used to link the location directive to one or more work templates.</span></span>

<span data-ttu-id="fa17f-150">Le righe delle direttive ubicazione impostano restrizioni aggiuntive sull'applicazione delle regole di individuazione dell'ubicazione.</span><span class="sxs-lookup"><span data-stu-id="fa17f-150">The location directive lines set additional restrictions on the application of the location finding rules.</span></span> <span data-ttu-id="fa17f-151">È possibile specificare una quantità minima e una quantità massima a cui la direttiva deve essere applicata e specificare che la direttiva deve essere destinata a un'unità di magazzino specifica.</span><span class="sxs-lookup"><span data-stu-id="fa17f-151">You can specify a minimum quantity and a maximum quantity that the directive should apply to, and you can specify that the directive should be for a specific inventory unit.</span></span> <span data-ttu-id="fa17f-152">Ad esempio, se l'unità di misura sono i pallet, gli articoli in pallet potranno essere stoccati in un'ubicazione specifica.</span><span class="sxs-lookup"><span data-stu-id="fa17f-152">For example, if the unit of measure is pallets, the items in pallets can be put in a specific location.</span></span> <span data-ttu-id="fa17f-153">È inoltre possibile specificare se la quantità può essere suddivisa tra più ubicazioni.</span><span class="sxs-lookup"><span data-stu-id="fa17f-153">You can also specify whether the quantity can be split across multiple locations.</span></span> <span data-ttu-id="fa17f-154">Come l'intestazione della direttiva ubicazione, ciascuna riga direttiva ubicazione ha un numero progressivo che determina l'ordine in cui vengono valutate le righe.</span><span class="sxs-lookup"><span data-stu-id="fa17f-154">Like the location directive header, each location directive line has a sequence number that determines the order that the lines are assessed in.</span></span>
