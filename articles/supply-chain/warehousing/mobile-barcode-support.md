[!include [banner](../includes/banner.md)]

<span data-ttu-id="79027-104">Poiché Android è un progetto open source, qualsiasi produttore di hardware per lettori di codici a barre per magazzino può creare un dispositivo compatibile con il sistema operativo Android.</span><span class="sxs-lookup"><span data-stu-id="79027-104">Because Android is an open source project, any manufacturer of hardware for warehouse bar code scanners can build a device to run the Android operating system.</span></span> <span data-ttu-id="79027-105">Un dispositivo è compatibile con Android se può eseguire app scritte per l'ambiente di esecuzione Android.</span><span class="sxs-lookup"><span data-stu-id="79027-105">A device is only Android-compatible if it can run apps that are written for the Android execution environment.</span></span>
<span data-ttu-id="79027-106">Tuttavia, un fornitore di hardware può modificare e creare overlay per la versione Android in esecuzione nel relativo hardware.</span><span class="sxs-lookup"><span data-stu-id="79027-106">However, a hardware vendor can modify and create overlays for the Android version that runs on their hardware.</span></span> <span data-ttu-id="79027-107">Non è responsabilità di Microsoft garantire la compatibilità di un app di scansione di codici a barre per dispositivi mobili Android con l'hardware di scansione di codici a barre del produttore e con la versione Android in esecuzione nello stesso.</span><span class="sxs-lookup"><span data-stu-id="79027-107">Microsoft cannot take any responsibility to ensure that a mobile bar code scanning app for Android is compatible with a manufacturer’s bar code scanning hardware and the Android version that runs on it.</span></span>

<span data-ttu-id="79027-110">Ai clienti si consiglia di testare l'app di scansione per dispositivi mobili Magazzino con l'hardware selezionato prima di decidere quale hardware acquistare.</span><span class="sxs-lookup"><span data-stu-id="79027-110">As a customer, we recommend that you test the Warehouse mobile scanning app on selected hardware before you decide on the hardware that you want to buy.</span></span>
