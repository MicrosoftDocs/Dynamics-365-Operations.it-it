<span data-ttu-id="f0efa-104">Questo argomento spiega come utilizzare le strategie della direttiva di ubicazione first in, first out (FIFO) e last in, first out (LIFO) durante il prelievo.</span><span class="sxs-lookup"><span data-stu-id="f0efa-104">This topic explains how to use first in, first out (FIFO) and last in, first out (LIFO) location directive strategies during picking.</span></span> <span data-ttu-id="f0efa-105">Queste strategie funzionano in combinazione con le date di aging registrate per le ubicazioni da tracciare quando le scorte sono entrate per la prima volta nel magazzino.</span><span class="sxs-lookup"><span data-stu-id="f0efa-105">These strategies work in conjunction with the aging dates that are recorded for locations to track when inventory first entered the warehouse.</span></span> <span data-ttu-id="f0efa-106">La funzionalità *Aging prelievo scorte per direttiva ubicazione* utilizza la data sull'ubicazione per determinare l'aging.</span><span class="sxs-lookup"><span data-stu-id="f0efa-106">The *Location directive inventory picking aging* feature uses the date on the location to determine aging.</span></span> <span data-ttu-id="f0efa-107">La funzionalità *Stato ubicazione magazzino* aggiorna la data sull'ubicazione, in base alla data della targa.</span><span class="sxs-lookup"><span data-stu-id="f0efa-107">The *Warehouse location status* feature updates the date on the location, based on the date from the license plate.</span></span>
