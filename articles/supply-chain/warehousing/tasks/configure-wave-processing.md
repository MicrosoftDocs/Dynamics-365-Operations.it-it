# <a name="configure-wave-processing"></a><span data-ttu-id="23f7e-103">Configurare l'elaborazione di ondate</span><span class="sxs-lookup"><span data-stu-id="23f7e-103">Configure wave processing</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="23f7e-104">In questo guida viene descritto come impostare i criteri che determinano quale lavoro viene generato per un magazzino quando viene elaborata un'ondata e se le ondate vengono elaborate manualmente o automaticamente.</span><span class="sxs-lookup"><span data-stu-id="23f7e-104">This guide describes how to set up the criteria that determine what work is generated for a warehouse when a wave is processed, and whether waves are processed manually or automatically.</span></span> <span data-ttu-id="23f7e-105">I criteri vengono specificati impostando i modelli ondata e le query che corrispondono a un'ondata con le righe rilasciate negli ordini cliente, negli ordini di produzione o negli ordini kanban.</span><span class="sxs-lookup"><span data-stu-id="23f7e-105">You specify the criteria by setting up wave templates and queries that match a wave with released lines in sales orders, production orders, or kanban orders.</span></span> <span data-ttu-id="23f7e-106">L'elaborazione ondata viene utilizzata in magazzini che utilizzano la funzionalità del modulo Gestione magazzino e non in quelli che utilizzano la funzionalità nel modulo Gestione inventario.</span><span class="sxs-lookup"><span data-stu-id="23f7e-106">Wave processing is used in warehouses that use the functionality in the Warehouse management module, and not those that use the functionality in the Inventory management module.</span></span> <span data-ttu-id="23f7e-107">È possibile eseguire questa procedura nella società di dati dimostrativi USMF.</span><span class="sxs-lookup"><span data-stu-id="23f7e-107">You can run this procedure in demo data company USMF.</span></span>

1. <span data-ttu-id="23f7e-108">Andare a Gestione magazzino > Impostazioni > Ondate > Modelli ondata.</span><span class="sxs-lookup"><span data-stu-id="23f7e-108">Go to Warehouse management > Setup > Waves > Wave templates.</span></span>

3. <span data-ttu-id="23f7e-110">Digitare un valore nel campo Nome modello ondata.</span><span class="sxs-lookup"><span data-stu-id="23f7e-110">In the Wave template name field, type a value.</span></span>
    * <span data-ttu-id="23f7e-111">Quando si imposta un modello di ondata, si specifica la sequenza in cui i modelli verranno associati alle righe rilasciate negli ordini cliente, negli ordini di produzione o sui kanban.</span><span class="sxs-lookup"><span data-stu-id="23f7e-111">When you set up a wave template, you specify the sequence in which the templates will be matched to released lines on sales orders, production orders, or kanbans.</span></span> <span data-ttu-id="23f7e-112">Quando una riga viene rilasciata al magazzino o alla produzione, viene utilizzato il primo modello di ondata che soddisfa i criteri.</span><span class="sxs-lookup"><span data-stu-id="23f7e-112">When a line is released to the warehouse or to production, it uses the first wave template that it meets the criteria for.</span></span> <span data-ttu-id="23f7e-113">Si consiglia di inserire i modelli con i criteri più specifici in cima all'elenco.</span><span class="sxs-lookup"><span data-stu-id="23f7e-113">We recommend that you put templates with the most specific criteria at the top of the list.</span></span> <span data-ttu-id="23f7e-114">Più ampi sono i criteri, maggiore è la possibilità che una riga soddisfi i criteri e ciò potrebbe comportare che le righe vengano assegnate a un'ondata errata.</span><span class="sxs-lookup"><span data-stu-id="23f7e-114">The broader the criteria, the more likely it is for a line to meet the criteria, and this could lead to lines being assigned to the wrong wave.</span></span>

5. <span data-ttu-id="23f7e-116">Nel campo Sito immettere o selezionare un valore.</span><span class="sxs-lookup"><span data-stu-id="23f7e-116">In the Site field, enter or select a value.</span></span>
    * <span data-ttu-id="23f7e-117">Se si utilizza USMF, è possibile selezionare il sito 2.</span><span class="sxs-lookup"><span data-stu-id="23f7e-117">If you’re using USMF, you can select site 2.</span></span>  
6. <span data-ttu-id="23f7e-118">Nel campo Magazzino immettere o selezionare un valore.</span><span class="sxs-lookup"><span data-stu-id="23f7e-118">In the Warehouse field, enter or select a value.</span></span>
    * <span data-ttu-id="23f7e-119">Se si utilizza USMF, è possibile selezionare il magazzino 24.</span><span class="sxs-lookup"><span data-stu-id="23f7e-119">If you’re using USMF, you can select warehouse 24.</span></span>  
7. <span data-ttu-id="23f7e-120">Impostare il campo Automatizza creazione ondata su Sì.</span><span class="sxs-lookup"><span data-stu-id="23f7e-120">Set the Automate wave creation field to Yes.</span></span>

    * <span data-ttu-id="23f7e-123">Selezionare questa opzione per elaborare automaticamente l'ondata e creare il lavoro quando una riga viene rilasciata nel magazzino.</span><span class="sxs-lookup"><span data-stu-id="23f7e-123">Select this option to automatically process the wave and create work when a line is released to the warehouse.</span></span>  
9. <span data-ttu-id="23f7e-124">Impostare l'opzione Automatizza rilascio ondata su Sì.</span><span class="sxs-lookup"><span data-stu-id="23f7e-124">Set the Automate wave release option to Yes.</span></span> 
    * <span data-ttu-id="23f7e-125">Selezionare questa opzione per rilasciare automaticamente l'ondata.</span><span class="sxs-lookup"><span data-stu-id="23f7e-125">Select this option to automatically release the wave.</span></span> <span data-ttu-id="23f7e-126">Il lavoro di prelievo viene creato e reso disponibile sui dispositivi mobili.</span><span class="sxs-lookup"><span data-stu-id="23f7e-126">The picking work is created and made available on mobile devices.</span></span>  
10. <span data-ttu-id="23f7e-127">Impostare l'opzione Assegna a ondate aperte su Sì.</span><span class="sxs-lookup"><span data-stu-id="23f7e-127">Set the Assign to open waves option to Yes.</span></span> 
    * <span data-ttu-id="23f7e-128">Le righe vengono assegnate a ondate sul filtro di query per il modello ondata.</span><span class="sxs-lookup"><span data-stu-id="23f7e-128">Lines are assigned to waves based on the query filter for the wave template.</span></span>  
11. <span data-ttu-id="23f7e-129">Impostare l'opzione Elabora ondata automaticamente alla soglia su Sì.</span><span class="sxs-lookup"><span data-stu-id="23f7e-129">Set the Process wave automatically at threshold option to Yes.</span></span> 
    * <span data-ttu-id="23f7e-130">Selezionare questa opzione per elaborare automaticamente l'ondata quando i relativi valori raggiungono le soglie di peso, la spedizione e le righe specificate nel gruppo di campi Soglie ondata.</span><span class="sxs-lookup"><span data-stu-id="23f7e-130">Select this option to automatically process the wave when its values reach the thresholds for weight, shipment, and lines specified in the Wave thresholds field group.</span></span> <span data-ttu-id="23f7e-131">Questa opzione è disponibile solo se nel campo Tipo di modello ondata è selezionata l'opzione Spedizione.</span><span class="sxs-lookup"><span data-stu-id="23f7e-131">This option is available only if Shipping is selected in the Wave template type field.</span></span>
