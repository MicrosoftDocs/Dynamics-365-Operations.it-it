# <a name="view-an-expense-report-from-general-ledger-or-procurement-and-sourcing"></a><span data-ttu-id="ebb8d-103">Visualizzare una nota spese da Contabilità generale o Approvvigionamento</span><span class="sxs-lookup"><span data-stu-id="ebb8d-103">View an expense report from General ledger or Procurement and sourcing</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="ebb8d-104">Quando si esegue una query per visualizzare le transazioni in Contabilità generale o Approvvigionamento, la query potrebbe restituire troppe informazioni.</span><span class="sxs-lookup"><span data-stu-id="ebb8d-104">When you run a query to view transactions in General ledger or Procurement and sourcing, the query might return too much information.</span></span> <span data-ttu-id="ebb8d-105">Di conseguenza, potrebbe essere necessario limitare l'ambito della ricerca.</span><span class="sxs-lookup"><span data-stu-id="ebb8d-105">Therefore, you might want to narrow your search.</span></span> <span data-ttu-id="ebb8d-106">Per ottenere le informazioni necessarie, è possibile eseguire il drill-down in una transazione per visualizzare la nota spese con le informazioni desiderate.</span><span class="sxs-lookup"><span data-stu-id="ebb8d-106">To get the information that you require, you can drill down into a transaction to see the expense report that it appeared on.</span></span>

<span data-ttu-id="ebb8d-107">Per le note spese è possibile eseguire il drill-down da una transazione fino alla nota spese originale archiviata in Gestione spese.</span><span class="sxs-lookup"><span data-stu-id="ebb8d-107">For expense reports, you can drill down from a transaction to the original report that is stored in Expense management.</span></span> <span data-ttu-id="ebb8d-108">Oltre all'importo totale delle spese, è possibile visualizzare i dettagli delle transazioni, le transazioni idonee al recupero dell'imposta sul valore aggiunto (IVA) e le ricevute collegate alle spese.</span><span class="sxs-lookup"><span data-stu-id="ebb8d-108">In addition to the total amount of expenses, you can view the itemized transactions, the transactions that are eligible for recovery of value-added tax (VAT), and the receipts that are attached to the expenses.</span></span>

<span data-ttu-id="ebb8d-109">Per eseguire il drill-down fino alla nota spese originale in cui una transazione è riportata, selezionare la transazione, quindi l'opzione **Documento originale**.</span><span class="sxs-lookup"><span data-stu-id="ebb8d-109">To drill down to the original expense report that a transaction appeared on, select the transaction, and then select **Original document**.</span></span>

<span data-ttu-id="ebb8d-110">Quando si visualizzano le singole transazioni, è possibile visualizzare anche tutti i dati aggiuntivi collegati a una transazione.</span><span class="sxs-lookup"><span data-stu-id="ebb8d-110">When you view individual transactions, you can also view any additional data that is attached to a transaction.</span></span> <span data-ttu-id="ebb8d-111">Per i dettagli delle spese di albergo è possibile ad esempio visualizzare l'elenco dettagliato di tutte le spese quali accesso a Internet, chiamate telefoniche e servizio in camera.</span><span class="sxs-lookup"><span data-stu-id="ebb8d-111">For example, for an itemized hotel expense, you can view a detailed list of expenses such as internet access, telephone calls, and room service.</span></span>
