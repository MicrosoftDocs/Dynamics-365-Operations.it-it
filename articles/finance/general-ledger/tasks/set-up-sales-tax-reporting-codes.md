# <a name="set-up-sales-tax-reporting-codes"></a><span data-ttu-id="5bba2-103">Imposta i codici reporting IVA</span><span class="sxs-lookup"><span data-stu-id="5bba2-103">Set up sales tax reporting codes</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="5bba2-104">Questi codici reporting IVA fanno riferimento a un numero di campo elencato in un report IVA.</span><span class="sxs-lookup"><span data-stu-id="5bba2-104">The Sales tax reporting codes refer to a field number that's listed on a sales tax report.</span></span> <span data-ttu-id="5bba2-105">Vengono utilizzati su layout di report specifici per paese/area geografica.</span><span class="sxs-lookup"><span data-stu-id="5bba2-105">They are used on country-specific report layouts.</span></span> <span data-ttu-id="5bba2-106">Vengono utilizzati anche nel pagamento IVA per codice (report).</span><span class="sxs-lookup"><span data-stu-id="5bba2-106">They're also used on the Sales tax payment by code report.</span></span> <span data-ttu-id="5bba2-107">Tale report mostra gli importi IVA per un periodo di liquidazione riepilogati per ogni codice di reporting.</span><span class="sxs-lookup"><span data-stu-id="5bba2-107">That report shows sales tax amounts for a settlement period summarized for each reporting code.</span></span> <span data-ttu-id="5bba2-108">Dopo aver creato i codici reporting IVA, è possibile utilizzarli come riferimenti nelle Schede dettaglio Impostazione report accessibili dalla pagina **Codice IVA**.</span><span class="sxs-lookup"><span data-stu-id="5bba2-108">After you create Sales tax reporting codes, you can refer to those codes on the Report setup FastTabs, which you can access from the **Sales tax code** page.</span></span> 

<span data-ttu-id="5bba2-109">Questa registrazione utilizza la società dimostrativa DEMF.</span><span class="sxs-lookup"><span data-stu-id="5bba2-109">This recording uses the DEMF demo company.</span></span>

1. <span data-ttu-id="5bba2-110">Nel **pannello di navigazione**, andare a **Imposta > Impostazione > IVA > Codici reporting IVA**.</span><span class="sxs-lookup"><span data-stu-id="5bba2-110">In the **Navigation pane**, go to **Tax > Setup > Sales tax > Sales tax reporting codes**.</span></span>
2. <span data-ttu-id="5bba2-111">Fare clic su **Nuovo**.</span><span class="sxs-lookup"><span data-stu-id="5bba2-111">Click **New**.</span></span>
3. <span data-ttu-id="5bba2-112">Selezionare il layout del report a cui appartiene il codice reporting.</span><span class="sxs-lookup"><span data-stu-id="5bba2-112">Select the report layout that the reporting code belongs to.</span></span> <span data-ttu-id="5bba2-113">Questo layout viene utilizzato per filtrare i codici reporting disponibili per un codice IVA.</span><span class="sxs-lookup"><span data-stu-id="5bba2-113">This layout is used to filter the available reporting codes for a sales tax code.</span></span> <span data-ttu-id="5bba2-114">Ogni codice IVA appartiene a un periodo di liquidazione appartenente a un ufficio IVA che utilizza un layout di report.</span><span class="sxs-lookup"><span data-stu-id="5bba2-114">Each sales tax code belongs to a settlement period, which belongs to a Sales tax authority, which uses a report layout.</span></span>  
4. <span data-ttu-id="5bba2-115">Nel campo **Codice del report** immettere un numero.</span><span class="sxs-lookup"><span data-stu-id="5bba2-115">In the **Reporting code** field, enter a number.</span></span>
5. <span data-ttu-id="5bba2-116">Nel campo **Testo report** immettere una descrizione da visualizzare nei report.</span><span class="sxs-lookup"><span data-stu-id="5bba2-116">In the **Report text** field, enter a description to display on reports.</span></span>
6. <span data-ttu-id="5bba2-117">Nel campo **Breve descrizione** immettere una descrizione per scopi interni.</span><span class="sxs-lookup"><span data-stu-id="5bba2-117">In the **Brief description** field, enter a description for internal purposes.</span></span>
7. <span data-ttu-id="5bba2-118">Fare clic su **Salva**.</span><span class="sxs-lookup"><span data-stu-id="5bba2-118">Click **Save**.</span></span>
