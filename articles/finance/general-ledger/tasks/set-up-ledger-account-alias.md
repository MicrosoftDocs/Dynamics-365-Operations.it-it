# <a name="set-up-a-ledger-account-alias"></a><span data-ttu-id="47d73-103">Impostare un alias conto CoGe</span><span class="sxs-lookup"><span data-stu-id="47d73-103">Set up a ledger account alias</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="47d73-104">Questa procedura mostra come creare un alias di conto che fornisce un collegamento per immettere un numero di conto.</span><span class="sxs-lookup"><span data-stu-id="47d73-104">This procedure shows how to create an account alias that provides a shortcut for entering an account number.</span></span> <span data-ttu-id="47d73-105">Questa procedura utilizza la società di dati dimostrativi USMF.</span><span class="sxs-lookup"><span data-stu-id="47d73-105">This procedure users demo data company USMF.</span></span>

1. <span data-ttu-id="47d73-106">Andare a Contabilità generale > Piano dei conti > Conti > Alias conto CoGe.</span><span class="sxs-lookup"><span data-stu-id="47d73-106">Go to General ledger > Chart of accounts > Accounts > Ledger account alias.</span></span>
2. <span data-ttu-id="47d73-107">Fare clic su Nuovo.</span><span class="sxs-lookup"><span data-stu-id="47d73-107">Click New.</span></span>
3. <span data-ttu-id="47d73-108">Digitare un valore nel campo Alias conto CoGe.</span><span class="sxs-lookup"><span data-stu-id="47d73-108">In the Ledger account alias field, type a value.</span></span>
4. <span data-ttu-id="47d73-109">Nel campo Struttura dei conti, selezionare la struttura a cui appartengono il conto e le dimensioni.</span><span class="sxs-lookup"><span data-stu-id="47d73-109">In the Account structure field, select the structure the account and dimensions belong to.</span></span>
5. <span data-ttu-id="47d73-110">Nel campo Società fare clic sul pulsante a discesa per aprire la ricerca.</span><span class="sxs-lookup"><span data-stu-id="47d73-110">In the Company field, click the drop-down button to open the lookup.</span></span>
6. <span data-ttu-id="47d73-111">Nell'elenco, individuare e selezionare la società a cui si applica l'alias.</span><span class="sxs-lookup"><span data-stu-id="47d73-111">In the list, find and select the company that the alias applies to.</span></span>
7. <span data-ttu-id="47d73-112">Nell'elenco fare clic sul collegamento nella riga selezionata.</span><span class="sxs-lookup"><span data-stu-id="47d73-112">In the list, click the link in the selected row.</span></span>
8. <span data-ttu-id="47d73-113">Nel campo Definizione alias conto CoGe, specificare il conto e le dimensioni.</span><span class="sxs-lookup"><span data-stu-id="47d73-113">In the Ledger account alias definition field, specify the account and dimensions.</span></span>
    * <span data-ttu-id="47d73-114">Il conto e le dimensioni saranno popolati quando si utilizza il collegamento.</span><span class="sxs-lookup"><span data-stu-id="47d73-114">The account and dimensions will be populated when using the shortcut.</span></span>  
9. <span data-ttu-id="47d73-115">Nel campo Stato attivo iniziale, selezionare la dimensione che avrà lo stato attivo quando l'alias viene utilizzato.</span><span class="sxs-lookup"><span data-stu-id="47d73-115">In the Initial focus field, select the dimension that will have focus when the alias is used.</span></span>
    * <span data-ttu-id="47d73-116">Dopo aver digitato il collegamento e il conto e le dimensioni sono popolati, il campo Stato attivo iniziale è dove si sposta il cursore o lo stato attivo.</span><span class="sxs-lookup"><span data-stu-id="47d73-116">After you type the shortcut, and the account and dimensions are populated, the Initial focus field is where the cursor or focus will move to.</span></span>
