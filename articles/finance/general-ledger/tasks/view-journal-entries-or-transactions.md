# <a name="view-journal-entries-or-transactions"></a><span data-ttu-id="2a979-103">Visualizzare le scritture contabili o le transazioni</span><span class="sxs-lookup"><span data-stu-id="2a979-103">View journal entries or transactions</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="2a979-104">In questa procedura viene illustrato l'utilizzo della Richiesta di informazioni su transazioni giustificativo per cercare gli inserimenti nel giornale di registrazione o le transazioni.</span><span class="sxs-lookup"><span data-stu-id="2a979-104">This procedure shows how to use the Voucher transactions inquiry to search for journal entries or transactions.</span></span>

1. <span data-ttu-id="2a979-105">Andare a **Pannello di navigazione >Moduli > Contabilità generale > Richieste di informazioni e report > Transazioni giustificativo**.</span><span class="sxs-lookup"><span data-stu-id="2a979-105">Go to **Navigation pane > Modules > General ledger > Inquiries and reports > Voucher transactions**.</span></span>
2. <span data-ttu-id="2a979-106">Selezionare il campo per il quale si desidera definire i criteri di filtro.</span><span class="sxs-lookup"><span data-stu-id="2a979-106">Select the field for which you want to define a filter criteria.</span></span>
3. <span data-ttu-id="2a979-107">Immettere i propri critieri di filtro per il campo selezionato.</span><span class="sxs-lookup"><span data-stu-id="2a979-107">Enter your filter critieria for the selected field.</span></span> <span data-ttu-id="2a979-108">È possibile applicare un filtro a un singolo valore o a un intervallo.</span><span class="sxs-lookup"><span data-stu-id="2a979-108">You could filter on a single value or a range.</span></span> <span data-ttu-id="2a979-109">Nel definire un intervallo, verificare che la sintassi utilizzata sia corretta.</span><span class="sxs-lookup"><span data-stu-id="2a979-109">When defining a range, make sure the correct syntax is used.</span></span> <span data-ttu-id="2a979-110">I valori devono essere separati da un doppio punto (..).</span><span class="sxs-lookup"><span data-stu-id="2a979-110">The values should be separated by a double period (..).</span></span>  
4. <span data-ttu-id="2a979-111">Fare clic sulla scheda **Join** per aggiungere tabelle aggiuntive dalle quali filtrare.</span><span class="sxs-lookup"><span data-stu-id="2a979-111">Click the **Joins** tab to add additional tables from which to filter.</span></span>
5. <span data-ttu-id="2a979-112">Nella struttura, selezionare **Tabelle/Voce giornale di registrazione generale**.</span><span class="sxs-lookup"><span data-stu-id="2a979-112">In the tree, select **Tables/General journal entry**.</span></span>
6. <span data-ttu-id="2a979-113">Fare clic su **Aggiungi join tabella**.</span><span class="sxs-lookup"><span data-stu-id="2a979-113">Click **Add table join**.</span></span>
7. <span data-ttu-id="2a979-114">Fare clic su **Annulla** se si decide di non aggiungere un'ulteriore tabella.</span><span class="sxs-lookup"><span data-stu-id="2a979-114">Click **Cancel** if you decide not to add an additional table.</span></span>
8. <span data-ttu-id="2a979-115">Fare clic sulla scheda **Intervallo**.</span><span class="sxs-lookup"><span data-stu-id="2a979-115">Click the **Range** tab.</span></span>
9. <span data-ttu-id="2a979-116">Fare clic su **OK** per eseguire la query.</span><span class="sxs-lookup"><span data-stu-id="2a979-116">Click **OK** to run the query.</span></span>
10. <span data-ttu-id="2a979-117">Nel riquadro azioni fare clic su **Origine transazione**.</span><span class="sxs-lookup"><span data-stu-id="2a979-117">On the Action pane, click **Transaction origin**.</span></span> <span data-ttu-id="2a979-118">I vari pulsanti relativi alla griglia possono essere utilizzati per cercare informazioni aggiuntive sul record di giustificativo selezionato.</span><span class="sxs-lookup"><span data-stu-id="2a979-118">Various buttons about the grid can be used to research additional information about the selected record of the voucher.</span></span> <span data-ttu-id="2a979-119">Alcuni pulsanti possono non essere disponibili, a seconda del tipo di transazione e le caratteristiche della transazione.</span><span class="sxs-lookup"><span data-stu-id="2a979-119">Some buttons may not be available, depending on the type of transaction and characteristics of the transaction.</span></span>
11. <span data-ttu-id="2a979-120">Chiudere la pagina.</span><span class="sxs-lookup"><span data-stu-id="2a979-120">Close the page.</span></span>
12. <span data-ttu-id="2a979-121">Nel riquadro azioni fare clic su **Documento originale**.</span><span class="sxs-lookup"><span data-stu-id="2a979-121">On the Action pane, Click **Original document**.</span></span>
13. <span data-ttu-id="2a979-122">Chiudere la pagina.</span><span class="sxs-lookup"><span data-stu-id="2a979-122">Close the page.</span></span>
