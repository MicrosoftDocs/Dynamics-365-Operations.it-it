<span data-ttu-id="ef196-109">Nelle procedure indicate di seguito viene descritto come impostare alcune competenze per una risorsa.</span><span class="sxs-lookup"><span data-stu-id="ef196-109">The following procedures explain how to set up some of the competencies for a resource.</span></span>

<span data-ttu-id="ef196-110">Per impostare le competenze di un lavoratore, è possibile utilizzare la pagina elenco **Lavoratori** in Risorse umane o la pagina elenco **Risorse** in Gestione progetti e contabilità.</span><span class="sxs-lookup"><span data-stu-id="ef196-110">To set up competencies for a worker, you can use either the **Workers** list page in Human resources or the **Resources** list page in Project management and accounting.</span></span> <span data-ttu-id="ef196-111">Per le procedure riportate di seguito, utilizzeremo la pagina elenco **Lavoratori** in Risorse umane.</span><span class="sxs-lookup"><span data-stu-id="ef196-111">For the following procedures, the **Workers** list page in Human resources is used.</span></span>

## <a name="set-up-competencies-certificates"></a><span data-ttu-id="ef196-112">Impostare le competenze: Certificati</span><span class="sxs-lookup"><span data-stu-id="ef196-112">Set up competencies: Certificates</span></span>

1. <span data-ttu-id="ef196-113">Nella pagina elenco **Lavoratori** selezionare la riga per il lavoratore per cui si desidera aggiungere i certificati.</span><span class="sxs-lookup"><span data-stu-id="ef196-113">On the **Workers** list page, select the line for the worker to add certificate information for.</span></span>
2. <span data-ttu-id="ef196-114">Nel riquadro azioni, scheda **Lavoratore**, gruppo **Competenze** selezionare **Certificati**.</span><span class="sxs-lookup"><span data-stu-id="ef196-114">On the Action Pane, on the **Worker** tab, in the **Competencies** group, select **Certificates**.</span></span>
3. <span data-ttu-id="ef196-115">Selezionare **Nuovo**, quindi nel campo **Tipo di certificato**, selezionare **PMP**.</span><span class="sxs-lookup"><span data-stu-id="ef196-115">Select **New**, and then, in the **Certificate type** field, select **PMP**.</span></span>
4. <span data-ttu-id="ef196-116">Nel campo **Data di inizio**, selezionare **10/1/2015** e quindi **Salva**.</span><span class="sxs-lookup"><span data-stu-id="ef196-116">In the **Start date** field, select **10/1/2015**, and select **Save**.</span></span>

## <a name="set-up-competencies-skills"></a><span data-ttu-id="ef196-117">Impostare le competenze: Competenze</span><span class="sxs-lookup"><span data-stu-id="ef196-117">Set up competencies: Skills</span></span>

1. <span data-ttu-id="ef196-118">Nella pagina elenco **Lavoratori** assicurarsi che il lavoratore utilizzato nella procedura precedente sia ancora selezionato.</span><span class="sxs-lookup"><span data-stu-id="ef196-118">On the **Workers** list page, make sure that the worker that you used in the previous procedure is still selected.</span></span> <span data-ttu-id="ef196-119">Nel riquadro azioni, scheda **Lavoratore**, gruppo **Competenze** selezionare **Competenze**.</span><span class="sxs-lookup"><span data-stu-id="ef196-119">Then, on the Action Pane, on the **Worker** tab, in the **Competencies** group, select **Skills**.</span></span>
2. <span data-ttu-id="ef196-120">Selezionare **Nuovo**.</span><span class="sxs-lookup"><span data-stu-id="ef196-120">Select **New**.</span></span>
3. <span data-ttu-id="ef196-121">Nel campo **Competenza** selezionare **Gestione progetti**.</span><span class="sxs-lookup"><span data-stu-id="ef196-121">In the **Skill** field, select **Project management**.</span></span>
4. <span data-ttu-id="ef196-122">Nel campo **Livello** selezionare **5 Esperto**.</span><span class="sxs-lookup"><span data-stu-id="ef196-122">In the **Level** field, select **5 Expert**.</span></span>
5. <span data-ttu-id="ef196-123">Nel campo **Data livello** selezionare **1/14/2014**.</span><span class="sxs-lookup"><span data-stu-id="ef196-123">In the **Level date** field, select **1-/14/2014**.</span></span>
6. <span data-ttu-id="ef196-124">Nel campo **Anni di esperienza** immettere **10**.</span><span class="sxs-lookup"><span data-stu-id="ef196-124">In the **Years of experience** field, enter **10**.</span></span>
7. <span data-ttu-id="ef196-125">Selezionare **Salva**, quindi chiudere la pagina.</span><span class="sxs-lookup"><span data-stu-id="ef196-125">Select **Save**, and then close the page.</span></span>
