# <a name="create-a-credit-note-on-project-invoices"></a><span data-ttu-id="7d8ca-103">Creare una nota di accredito per le fatture di progetto</span><span class="sxs-lookup"><span data-stu-id="7d8ca-103">Create a credit note on project invoices</span></span>

[!include [banner](../../includes/banner.md)]

1. <span data-ttu-id="7d8ca-104">Passare a Gestione progetti e contabilità > Progetti > Tutti i progetti.</span><span class="sxs-lookup"><span data-stu-id="7d8ca-104">Go to Project management and accounting > Projects > All projects.</span></span> 
2. <span data-ttu-id="7d8ca-105">Nell'elenco trovare e selezionare il record desiderato.</span><span class="sxs-lookup"><span data-stu-id="7d8ca-105">In the list, find and select the desired record.</span></span> 
3. <span data-ttu-id="7d8ca-106">Selezionare l'ID progetto utilizzato per la nota di accredito.</span><span class="sxs-lookup"><span data-stu-id="7d8ca-106">Select the Project ID that is used for the Credit Note.</span></span> <span data-ttu-id="7d8ca-107">Nell'elenco fare clic sul collegamento nella riga selezionata.</span><span class="sxs-lookup"><span data-stu-id="7d8ca-107">In the list, click the link in the selected row.</span></span> 
4. <span data-ttu-id="7d8ca-108">Nel riquadro azioni, fare clic su Gestisci.</span><span class="sxs-lookup"><span data-stu-id="7d8ca-108">On the Action Pane, click Manage.</span></span> 
5. <span data-ttu-id="7d8ca-109">Fare clic su Giornali di registrazione fatture.</span><span class="sxs-lookup"><span data-stu-id="7d8ca-109">Click Invoice journals.</span></span> 
6. <span data-ttu-id="7d8ca-110">Selezionare la transazione per la nota di accredito.</span><span class="sxs-lookup"><span data-stu-id="7d8ca-110">Select the transaction for credit note.</span></span> <span data-ttu-id="7d8ca-111">Nell'elenco contrassegnare la riga selezionata.</span><span class="sxs-lookup"><span data-stu-id="7d8ca-111">In the list, mark the selected row.</span></span> 
7. <span data-ttu-id="7d8ca-112">Fare clic su Funzioni.</span><span class="sxs-lookup"><span data-stu-id="7d8ca-112">Click Functions.</span></span> 
8. <span data-ttu-id="7d8ca-113">Fare clic su Seleziona per nota di accredito.</span><span class="sxs-lookup"><span data-stu-id="7d8ca-113">Click Select for credit note.</span></span> 
9. <span data-ttu-id="7d8ca-114">Selezionare la transazione per la nota di accredito spuntando la casella di controllo.</span><span class="sxs-lookup"><span data-stu-id="7d8ca-114">Select the transaction for credit note by marking the check box.</span></span>
10. <span data-ttu-id="7d8ca-115">Fare clic su OK.</span><span class="sxs-lookup"><span data-stu-id="7d8ca-115">Click OK.</span></span> 

<span data-ttu-id="7d8ca-116">Dopo questo passaggio, la nota di accredito sarà disponibile per la proposta di fatturazione.</span><span class="sxs-lookup"><span data-stu-id="7d8ca-116">After this step, the Credit Note will be available for Invoice Proposal.</span></span>
