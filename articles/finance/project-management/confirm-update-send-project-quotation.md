[!include [banner](../includes/banner.md)]

<span data-ttu-id="d79dd-104">Dopo aver creato e inviato un'offerta di progetto a un cliente, è necessario ottenere la conferma dal cliente prima di aggiornare lo stato dell'offerta a **Inviata**.</span><span class="sxs-lookup"><span data-stu-id="d79dd-104">After you create and send a project quotation to a customer, you must get confirmation from the customer before you update the quotation status to **Sent**.</span></span> <span data-ttu-id="d79dd-105">Eventuali modifiche richieste dal cliente possono essere aggiornate nell'offerta.</span><span class="sxs-lookup"><span data-stu-id="d79dd-105">Any modifications requested by the customer can be updated in the quotation.</span></span> <span data-ttu-id="d79dd-106">Dopo aver aggiornato lo stato dell'offerta a **Inviata**, non è possibile modificarlo.</span><span class="sxs-lookup"><span data-stu-id="d79dd-106">After you update the quotation status to **Sent**, you can’t modify it.</span></span> <span data-ttu-id="d79dd-107">La seguente procedura descrive le opzioni per inviare un'offerta per la conferma, effettuare aggiornamenti in base al feedback e quindi inviare l'offerta.</span><span class="sxs-lookup"><span data-stu-id="d79dd-107">The following procedure describes the options to send a quotation for confirmation, make updates based on feedback, and then send the quotation.</span></span>

## <a name="send-a-project-quotation-confirmation"></a><span data-ttu-id="d79dd-108">Inviare la conferma di un'offerta di progetto</span><span class="sxs-lookup"><span data-stu-id="d79dd-108">Send a project quotation confirmation</span></span>  

<span data-ttu-id="d79dd-109">È possibile inviare un'offerta di progetto esistente per la conferma tramite e-mail o come copia stampata.</span><span class="sxs-lookup"><span data-stu-id="d79dd-109">You can send an existing project quotation for confirmation through email or as a printed copy.</span></span> 

1. <span data-ttu-id="d79dd-110">Andare a **Gestione progetti e contabilità** > **Progetti** > **Tutti i progetti**.</span><span class="sxs-lookup"><span data-stu-id="d79dd-110">Go to **Project management and accounting** > **Quotations** > **Project quotation.**</span></span> 
2. <span data-ttu-id="d79dd-111">Nella pagina **Offerta di progetto**, selezionare l'offerta che si desidera inviare per la conferma.</span><span class="sxs-lookup"><span data-stu-id="d79dd-111">On the **Project quotation** page, select the quotation you want to send for confirmation.</span></span> 
3. <span data-ttu-id="d79dd-112">Nella scheda **Follow-up** nel gruppo **Genera**, selezionare **Conferma**.</span><span class="sxs-lookup"><span data-stu-id="d79dd-112">On the **Follow up** tab, in the **Generate** group, select **Confirm**.</span></span> 
4. <span data-ttu-id="d79dd-113">Nella pagina **Conferma offerta**, impostare i parametri richiesti.</span><span class="sxs-lookup"><span data-stu-id="d79dd-113">On the **Confirm quotation** page, set the required parameters.</span></span> <span data-ttu-id="d79dd-114">Ad esempio, per stampare la conferma, sotto le opzioni **Stampa**, attivare **Stampa conferma**, quindi selezionare **OK**.</span><span class="sxs-lookup"><span data-stu-id="d79dd-114">For example, to print the confirmation, under **Print** options, turn on **Print confirmation**, and then select **OK**.</span></span>
5. <span data-ttu-id="d79dd-115">Nella pagina **Invia offerta**, selezionare **Opzioni** e nella pagina **Impostazioni destinazione di stampa**, selezionare se si desidera inviare l'offerta a **Schermo**, **E-mail**, **File**, **Archivio di stampa** o **Stampante**, quindi apportare le modifiche desiderate nell'area **Specifica** per instradare l'offerta.</span><span class="sxs-lookup"><span data-stu-id="d79dd-115">On the **Send quotation** page, select **Options**, and on the **Print destination settings** page, select whether you want the quotation sent to **Screen**, **E-mail**, **File**, **Print archive**, or **Printer**, and then make any adjustments that you want in the **Specification** area to route the quotation.</span></span>
6. <span data-ttu-id="d79dd-116">Nella pagina **Impostazioni destinazione di stampa** selezionare **OK**.</span><span class="sxs-lookup"><span data-stu-id="d79dd-116">On the **Print destination settings** page, select **OK**.</span></span>

2. <span data-ttu-id="d79dd-121">Nella pagina **Offerte di progetto**, selezionare l'offerta che si desidera aggiornare e nel riquadro azioni selezionare **Modifica**.</span><span class="sxs-lookup"><span data-stu-id="d79dd-121">On the **Project quotations** page, select the quotation you want to update and on the Action Pane, select **Edit**.</span></span>
