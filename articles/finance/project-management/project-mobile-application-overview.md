# <a name="project-mobile-applications-overview"></a>Panoramica delle applicazioni per dispositivi mobili Project

[!include [banner](../includes/banner.md)]

Le tre applicazioni per dispositivi mobili correlate all'ora progetto sono disponibili per Microsoft Dynamics 365 Project Timesheet, Immissione ora progetto e Fogli presenze personali/Fogli presenze per revisione personale (ottimizzato per dispositivi mobili).

## <a name="project-timesheet-mobile-app"></a>App per dispositivi mobili Project Timesheet

L'app per dispositivi mobili Dynamics 365 Project Timesheet consente agli utenti di inviare e approvare fogli presenze per progetti sul loro dispositivo mobile. Questa app per dispositivi mobili utilizza la funzionalità di fogli presenze nell'area Gestione progetti e contabilità di Dynamics 365 Finance. Consente di migliorare la produttività e l'efficienza degli utenti e di facilitare l'immissione e l'approvazione di fogli presenze progetto.

## <a name="project-time-entry-workspace"></a>Area di lavoro Immissione ora progetto

L'area di lavoro Immissione ora progetto consente agli utenti di immettere e salvare l'ora di un progetto. Tuttavia, non consente agli utenti di inviare o approvare fogli presenze progetto. Può essere utilizzata solo per l'immissione di tempo. Questa area di lavoro mobile può essere utilizzata con l'app Finance and Operations per dispositivi mobili.

## <a name="my-timesheetstimesheets-for-my-review"></a>Fogli presenze personali/Fogli presenze per revisione personale

Fogli presenze personali (ottimizzato per dispositivi mobili) e Fogli presenze per revisione personale sono una soluzione che consente ai dipendenti di creare, inviare e approvare fogli presenze progetto su un dispositivo mobile. I lavoratori possono accedere alla funzionalità di Finance mediante l'URL dell'applicazione. Questa soluzione non richiede l'applicazione Finance and Operations per dispositivi mobili.
