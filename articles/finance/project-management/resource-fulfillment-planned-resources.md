# <a name="resource-fulfillment-for-planned-resources"></a><span data-ttu-id="52da8-103">Soddisfare la capacità delle risorse pianificate</span><span class="sxs-lookup"><span data-stu-id="52da8-103">Resource fulfillment for planned resources</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="52da8-104">Un manager di progetto può pianificare i ruoli risorse necessari per un progetto.</span><span class="sxs-lookup"><span data-stu-id="52da8-104">A project manager can plan required resource roles for a project.</span></span> <span data-ttu-id="52da8-105">Il responsabile delle risorse vedrà queste risorse pianificate come richieste nella pagina **Soddisfare la capacità delle risorse pianificate** e potrà assegnare le risorse effettive.</span><span class="sxs-lookup"><span data-stu-id="52da8-105">The resource manager will see these planned resources as requests on the **Resource fulfillment** page and can assign actual resources.</span></span>

1. <span data-ttu-id="52da8-106">Nella pagina **Tutti i progetti**, selezionare il progetto **Fase 2 di aggiornamento di XYZ**.</span><span class="sxs-lookup"><span data-stu-id="52da8-106">On the **All projects** page, select the **XYZ Upgrade Phase 2** project.</span></span>
2. <span data-ttu-id="52da8-107">Selezionare **Progetto**, quindi selezionare **Modifica**.</span><span class="sxs-lookup"><span data-stu-id="52da8-107">Select **Project**, and then select **Edit**.</span></span>
3. <span data-ttu-id="52da8-108">Nella scheda **Team progetto e programmazione**, selezionare **Aggiungi**.</span><span class="sxs-lookup"><span data-stu-id="52da8-108">On the **Project team and scheduling** tab, select **Add**.</span></span>
4. <span data-ttu-id="52da8-109">Nella finestra di dialogo **Aggiungi ruoli** selezionare il ruolo **Sviluppatore software**.</span><span class="sxs-lookup"><span data-stu-id="52da8-109">In the **Add roles** dialog box, select the **Software developer** role.</span></span>
5. <span data-ttu-id="52da8-110">Selezionare **Crea**, quindi chiudere la pagina del progetto.</span><span class="sxs-lookup"><span data-stu-id="52da8-110">Select **Create**, and then close the project page.</span></span>
6. <span data-ttu-id="52da8-111">Nella pagina **Possibilità di soddisfare la capacità delle risorse**, selezionare **Sviluppatore software 1** per il progetto **Fase 2 di aggiornamento di XYZ**.</span><span class="sxs-lookup"><span data-stu-id="52da8-111">On the **Resource fulfillment** page, select **Software developer 1** for the **XYZ Upgrade project Phase 2** project.</span></span>
7. <span data-ttu-id="52da8-112">Selezionare un lavoratore e quindi **Assegna**.</span><span class="sxs-lookup"><span data-stu-id="52da8-112">Select a worker, and then select **Assign**.</span></span>
8. <span data-ttu-id="52da8-113">Verificare che la riga per **Sviluppatore software 1** sia stata rimossa per il progetto **Fase 2 di aggiornamento di XYZ**.</span><span class="sxs-lookup"><span data-stu-id="52da8-113">Verify that the line for **Software developer 1** has been removed for the **XYZ Upgrade project Phase 2** project.</span></span>
9. <span data-ttu-id="52da8-114">Nella scheda **Team progetto e programmazione**, per il progetto **Fase 2 di aggiornamento di XYZ**, verificare che il lavoratore selezionato nel passaggio precedente sia stato aggiunto come **Sviluppatore software**.</span><span class="sxs-lookup"><span data-stu-id="52da8-114">On the **Project team and scheduling** tab, for the **XYZ Upgrade Phase 2** project, verify that the worker that you selected in the previous step has been added as **Software developer**.</span></span>
