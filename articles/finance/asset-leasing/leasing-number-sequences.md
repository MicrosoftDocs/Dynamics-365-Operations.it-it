# <a name="assign-number-sequences"></a>Assegnare sequenze numeriche

[!include [banner](../includes/banner.md)]

Questo argomento spiega come creare sequenze numeriche per ID leasing. Spiega inoltre come creare ID univoci che vengono utilizzati nel processo di rivalutazione dell'indice.

1. Vai a **Leasing cespite \> Imposta \> Parametri di leasing cespite**.
2. Seleziona la scheda laterale **Sequenze numeriche**.
3. Seleziona **Sequenze numeriche** nella barra laterale.
4. Seleziona la sequenza numerica per il riferimento **ID leasing**. Questa sequenza numerica verrà utilizzata per generare l'identificatore univoco per ogni leasing.
5. Seleziona la sequenza numerica per il riferimento **ID processo**. Questa sequenza numerica verrà utilizzata per generare l'identificatore univoco per ogni processo di rivalutazione dell'indice.
