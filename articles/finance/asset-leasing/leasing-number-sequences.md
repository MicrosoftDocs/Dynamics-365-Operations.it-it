[!include [banner](../includes/banner.md)]
[!include [preview banner](../includes/preview-banner.md)]

<span data-ttu-id="8eca2-105">Questo argomento spiega come creare sequenze numeriche per ID leasing.</span><span class="sxs-lookup"><span data-stu-id="8eca2-105">This topic explains how to create number sequences for lease IDs.</span></span> <span data-ttu-id="8eca2-106">Spiega inoltre come creare ID univoci che vengono utilizzati nel processo di rivalutazione dell'indice.</span><span class="sxs-lookup"><span data-stu-id="8eca2-106">It also explains how to create unique IDs that are used in the index revaluation process.</span></span>

1. <span data-ttu-id="8eca2-107">Vai a **Leasing cespite \> Imposta \> Parametri di leasing cespite**.</span><span class="sxs-lookup"><span data-stu-id="8eca2-107">Go to **Asset leasing \> Setup \> Asset leasing parameters**.</span></span>
2. <span data-ttu-id="8eca2-108">Seleziona la scheda laterale **Sequenze numeriche**.</span><span class="sxs-lookup"><span data-stu-id="8eca2-108">Select the **Number sequences** side tab.</span></span>
3. <span data-ttu-id="8eca2-109">Seleziona **Sequenze numeriche** nella barra laterale.</span><span class="sxs-lookup"><span data-stu-id="8eca2-109">Select **Number sequences** in the side bar.</span></span>

5. <span data-ttu-id="8eca2-112">Seleziona la sequenza numerica per il riferimento **ID processo**.</span><span class="sxs-lookup"><span data-stu-id="8eca2-112">Select the number sequence for the **Process ID** reference.</span></span> <span data-ttu-id="8eca2-113">Questa sequenza numerica verrà utilizzata per generare l'identificatore univoco per ogni processo di rivalutazione dell'indice.</span><span class="sxs-lookup"><span data-stu-id="8eca2-113">This number sequence will be used to generate the unique identifier for each index revaluation process.</span></span>
6. <span data-ttu-id="8eca2-114">Seleziona la sequenza numerica per il riferimento **ID proposta di risoluzione**.</span><span class="sxs-lookup"><span data-stu-id="8eca2-114">Select the number sequence for the **Termination Proposal ID** reference.</span></span> <span data-ttu-id="8eca2-115">Questa sequenza numerica verrà utilizzata per generare l'identificatore univoco per ciascuna proposta di risoluzione.</span><span class="sxs-lookup"><span data-stu-id="8eca2-115">This number sequence will be used to generate the unique identifier for each termination proposal.</span></span>
