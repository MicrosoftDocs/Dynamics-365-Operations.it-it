<span data-ttu-id="a54de-105">Questo argomento spiega come creare sequenze numeriche per ID leasing.</span><span class="sxs-lookup"><span data-stu-id="a54de-105">This topic explains how to create number sequences for lease IDs.</span></span> <span data-ttu-id="a54de-106">Spiega inoltre come creare ID univoci che vengono utilizzati nel processo di rivalutazione dell'indice.</span><span class="sxs-lookup"><span data-stu-id="a54de-106">It also explains how to create unique IDs that are used in the index revaluation process.</span></span>
