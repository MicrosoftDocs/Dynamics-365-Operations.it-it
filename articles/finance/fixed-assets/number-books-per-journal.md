# <a name="number-of-books-per-journal"></a><span data-ttu-id="866b7-104">Numero di libri per giornale di registrazione</span><span class="sxs-lookup"><span data-stu-id="866b7-104">Number of books per journal</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="866b7-105">Questo argomento descrive la relazione tra giornali e libri cespiti quando si crea un'acquisizione di cespiti o una proposta di ammortamento tramite un processo batch.</span><span class="sxs-lookup"><span data-stu-id="866b7-105">This topic describes the relationship between journals and asset books when you create a fixed asset acquisition or depreciation proposal through a batch job.</span></span> <span data-ttu-id="866b7-106">Puoi definire il numero massimo di libri inclusi per ciascuna acquisizione e per l'ammortamento utilizzando i campi nella sezione **Numero di libri per giornale di registrazione** nella scheda **Generale** della pagina **Parametri dei cespiti** (**Cespiti \> Imposta \> Parametri dei cespiti**).</span><span class="sxs-lookup"><span data-stu-id="866b7-106">You can define the maximum number of books that are included for each acquisition and for depreciation by using the fields in the **Number of books per journal** section on the **General** tab of the **Fixed assets parameters** page (**Fixed assets \> Setup \> Fixed assets parameters**).</span></span> <span data-ttu-id="866b7-107">Questi campi consentono di distribuire il numero di libri cespiti per giornale di registrazione acquisti e giornale di registrazione ammortamenti.</span><span class="sxs-lookup"><span data-stu-id="866b7-107">These fields let you distribute the number of asset books per acquisition journal and depreciation journal.</span></span>

<span data-ttu-id="866b7-108">Per una proposta di acquisizione, il valore predefinito è di almeno 10.000 libri.</span><span class="sxs-lookup"><span data-stu-id="866b7-108">For an acquisition proposal, the default value is at least 10,000 books.</span></span> <span data-ttu-id="866b7-109">Per una proposta di ammortamento, il valore predefinito è di almeno 2.000 libri.</span><span class="sxs-lookup"><span data-stu-id="866b7-109">For a depreciation proposal, the default value is at least 2,000 books.</span></span>

<span data-ttu-id="866b7-110">Ad esempio, se sono presenti 4.000 cespiti e due libri sono associati a ciascun cespite, un libro verrà registrato nel livello corrente e l'altro libro verrà registrato nel livello imposte.</span><span class="sxs-lookup"><span data-stu-id="866b7-110">For example, if there are 4,000 fixed assets, and two books are associated with each asset, one book will be posted to the current layer, and the other book will be posted to the tax layer.</span></span> <span data-ttu-id="866b7-111">Se si acquisiscono 4.000 cespiti tramite l'elaborazione batch, il processo batch che crea un giornale di registrazione acquisizione cespiti conterrà 4.000 libri cespiti.</span><span class="sxs-lookup"><span data-stu-id="866b7-111">If you acquire 4,000 fixed assets through batch processing, the batch job that creates one fixed asset acquisition journal will contain 4,000 asset books.</span></span>

> <span data-ttu-id="866b7-112">Il giornale di registrazione creato continuerà a essere utilizzato fino al completamento del processo batch.</span><span class="sxs-lookup"><span data-stu-id="866b7-112">The journal that is created will continue to be used until the batch job is completed.</span></span>

> <span data-ttu-id="866b7-113">I libri derivati non sono inclusi nel numero massimo di libri per giornale di registrazione.</span><span class="sxs-lookup"><span data-stu-id="866b7-113">The derived books aren't included in the maximum number of books per journal.</span></span>

<span data-ttu-id="866b7-118">Il processo di elaborazione batch esclude i libri chiusi.</span><span class="sxs-lookup"><span data-stu-id="866b7-118">The batch processing job excludes closed books.</span></span> <span data-ttu-id="866b7-119">Ad esempio, in un processo batch per l'ammortamento, vengono chiusi 10 dei primi 2.000 libri.</span><span class="sxs-lookup"><span data-stu-id="866b7-119">For example, in a batch job for depreciation, 10 of the first 2,000 books are closed.</span></span> <span data-ttu-id="866b7-120">In questo caso, il primo giornale di registrazione conterrà i libri associati ai cespiti numerati da 1 a 2.011.</span><span class="sxs-lookup"><span data-stu-id="866b7-120">In this case, the first journal will contain books that are associated with the fixed assets that are numbered 1 through 2,011.</span></span> <span data-ttu-id="866b7-121">Il secondo giornale di registrazione conterrà quindi i libri associati ai cespiti numerati da 2.012 a 4.000.</span><span class="sxs-lookup"><span data-stu-id="866b7-121">The second journal will then contain books that are associated with the fixed assets that are numbered 2,012 through 4,000.</span></span>
