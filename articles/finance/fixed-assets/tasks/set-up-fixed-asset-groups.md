# <a name="set-up-fixed-asset-groups"></a><span data-ttu-id="ac3ba-103">Impostare gruppi di cespiti</span><span class="sxs-lookup"><span data-stu-id="ac3ba-103">Set up fixed asset groups</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="ac3ba-104">In questo argomento viene illustrato come creare un nuovo gruppo cespite.</span><span class="sxs-lookup"><span data-stu-id="ac3ba-104">This topic explains how to create a new fixed asset group.</span></span> <span data-ttu-id="ac3ba-105">Utilizza il ruolo Ragioniere e i dati dimostrativi per la persona giuridica USMF.</span><span class="sxs-lookup"><span data-stu-id="ac3ba-105">It uses the Accountant role and demo data for the USMF legal entity.</span></span>

1. <span data-ttu-id="ac3ba-106">Nel pannello di navigazione, passare a **Moduli > Cespiti > Impostazione > Gruppi cespite**.</span><span class="sxs-lookup"><span data-stu-id="ac3ba-106">In the navigation pane, go to **Modules > Fixed assets > Setup > Fixed asset groups**.</span></span>
2. <span data-ttu-id="ac3ba-107">Selezionare **Nuovo**.</span><span class="sxs-lookup"><span data-stu-id="ac3ba-107">Select **New**.</span></span>
3. <span data-ttu-id="ac3ba-108">Nel campo **Gruppo cespite** digitare un valore.</span><span class="sxs-lookup"><span data-stu-id="ac3ba-108">In the **Fixed asset group** field, type a value.</span></span>
4. <span data-ttu-id="ac3ba-109">Digitare un valore nel campo **Nome**.</span><span class="sxs-lookup"><span data-stu-id="ac3ba-109">In the **Name** field, type a value.</span></span> <span data-ttu-id="ac3ba-110">Numerazione automatica cespiti e Codice sequenza numerica in **Gruppo cespite** sostituiranno le impostazioni di Parametri cespiti.</span><span class="sxs-lookup"><span data-stu-id="ac3ba-110">Autonumber fixed assets and Number sequence code on the **Fixed asset** group will override the settings on the Fixed assets parameters.</span></span> <span data-ttu-id="ac3ba-111">È possibile modificarlo in questo campo se i cespiti del gruppo cespite avranno una numerazione diversa da altri gruppi.</span><span class="sxs-lookup"><span data-stu-id="ac3ba-111">You can change it here if the assets in this fixed asset group will have different numbering from other groups.</span></span>  
5. <span data-ttu-id="ac3ba-112">Selezionare **Libri**.</span><span class="sxs-lookup"><span data-stu-id="ac3ba-112">Select **Books**.</span></span>
6. <span data-ttu-id="ac3ba-113">Nel campo **Libro** immettere o selezionare un valore.</span><span class="sxs-lookup"><span data-stu-id="ac3ba-113">In the **Book** field, enter or select a value.</span></span> <span data-ttu-id="ac3ba-114">Il campo **Calcola ammortamento** è impostato su **Sì**, quindi il libro del cespite verrà incluso nelle proposte di ammortamento.</span><span class="sxs-lookup"><span data-stu-id="ac3ba-114">The **Calculate depreciation** field is set to **Yes**, so the asset book will be included in depreciation proposals.</span></span> <span data-ttu-id="ac3ba-115">Se **Calcola ammortamento** è impostato su **No**, il cespite non verrà ammortizzato automaticamente.</span><span class="sxs-lookup"><span data-stu-id="ac3ba-115">If **Calculate depreciation** is set to **No**, the asset will not be automatically depreciated.</span></span>  
7. <span data-ttu-id="ac3ba-116">Impostare la vita utile del cespite, espressa in anni.</span><span class="sxs-lookup"><span data-stu-id="ac3ba-116">Set the Service life of the asset, in years.</span></span> <span data-ttu-id="ac3ba-117">Si noti che il valore del campo **Periodi di ammortamento** viene calcolato dopo aver impostato la Vita utile.</span><span class="sxs-lookup"><span data-stu-id="ac3ba-117">Note that the **Depreciation periods** field value is calculated after setting the Service life.</span></span>

9. <span data-ttu-id="ac3ba-119">Chiudere la pagina.</span><span class="sxs-lookup"><span data-stu-id="ac3ba-119">Close the page.</span></span>
