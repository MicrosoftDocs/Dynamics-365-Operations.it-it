# <a name="invoice-capture-received-files"></a>File ricevuti di Invoice Capture

[!include [banner](../includes/banner.md)]
[!include [preview banner](../includes/preview-banner.md)]

In Invoice Capture, la pagina **File ricevuti** è un luogo centrale in cui i file delle fatture vengono ricevuti da diverse origini.

## <a name="upload-invoice-files"></a>Carica file di fattura

Gli addetti alla contabilità fornitori possono caricare le immagini delle fatture selezionando **Carica file** per aprire la finestra di dialogo **Carica**. Dopo che un addetto della contabilità fornitori ha selezionato un file, il pulsante **Carica** diventa disponibile.

## <a name="include-or-obsolete-invoice-files"></a>Includere o rendere obsolete file di fattura

Gli utenti possono selezionare le fatture che contengono errori o avvisi, quindi includerle o renderle obsolete selezionando il pulsante corrispondente nel riquadro azioni. Una fattura contrassegnata come obsoleta non verrà visualizzata nella visualizzazione **File ricevuti (obsoleti)**.

## <a name="view-captured-invoices"></a>Visualizzare fatture acquisite

Dopo che un file ricevuto è stato riconosciuto con successo, le informazioni sulla fattura acquisita vengono restituite dal modello di intelligenza artificiale e inserite in Microsoft Dataverse. L'addetto alla contabilità fornitori può quindi verificare i dettagli della fattura selezionando **Visualizza fattura acquisita**. Gli utenti possono scaricare il file della fattura originale come richiesto.
