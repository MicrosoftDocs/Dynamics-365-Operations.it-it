# <a name="learning-catalog-for-dynamics-365-finance-functional-consultants"></a>Catalogo di apprendimento per i consulenti funzionali di Dynamics 365 Finance

Sei un esperto di implementazione per un dominio aziendale?

Il seguente catalogo è organizzato partendo dalla conoscenza di base a domini specifici e dalla più elementare alla più avanzata. Se il contenuto esiste in più formati, verrà reso noto in modo da poter scegliere il formato di formazione più adatto alle esigenze.

Per i consulenti funzionali, la ricerca sull'analisi delle mansioni ha dimostrato che tutti devono comprendere il set di contenuti di base e un'area di dominio.
