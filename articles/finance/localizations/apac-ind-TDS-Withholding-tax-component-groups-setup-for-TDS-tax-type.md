# <a name="set-up-withholding-tax-component-groups-for-the-tds-tax-type"></a><span data-ttu-id="ed6cf-103">Impostare i gruppi di componenti di ritenuta d'acconto per il tipo di imposta TDS</span><span class="sxs-lookup"><span data-stu-id="ed6cf-103">Set up withholding tax component groups for the TDS tax type</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="ed6cf-104">In questo argomento viene illustrato come impostare i gruppi di componenti di ritenuta d'acconto come **Affitto** e **Terzista** per il tipo di imposta TDS.</span><span class="sxs-lookup"><span data-stu-id="ed6cf-104">This topic explains how to set up withholding tax component groups, such as **Rent** and **Contractor**, for the Tax Deducted at Source (TDS) tax type.</span></span>

1. <span data-ttu-id="ed6cf-105">Seleziona **Imposta \> Impostazione \> Ritenuta d'acconto \> Gruppi componenti ritenuta d'acconto**.</span><span class="sxs-lookup"><span data-stu-id="ed6cf-105">Go to **Tax \> Setup \> Withholding tax \> Withholding tax component groups**.</span></span>

    <span data-ttu-id="ed6cf-106">[![Pagina Gruppi componenti ritenuta d'acconto](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span><span class="sxs-lookup"><span data-stu-id="ed6cf-106">[![Withholding tax component groups page](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span></span>

2. <span data-ttu-id="ed6cf-107">Nel campo **Tipo di imposta**, seleziona **TDS** per impostare i gruppi di componenti di ritenuta d'acconto per il tipo di imposta TDS.</span><span class="sxs-lookup"><span data-stu-id="ed6cf-107">In the **Tax type** field, select **TDS** to set up withholding tax component groups for the TDS tax type.</span></span>
3. <span data-ttu-id="ed6cf-108">Nel riquadro azioni seleziona **Nuova** per creare una riga.</span><span class="sxs-lookup"><span data-stu-id="ed6cf-108">On the Action Pane, select **New** to create a line.</span></span>
4. <span data-ttu-id="ed6cf-109">Nel campo **Gruppo componenti ritenuta d'acconto**, immetti un nome per il gruppo di componenti TDS.</span><span class="sxs-lookup"><span data-stu-id="ed6cf-109">In the **Withholding tax component group** field, enter a name for the TDS component group.</span></span>
5. <span data-ttu-id="ed6cf-110">Nel campo **Descrizione** immettere una descrizione.</span><span class="sxs-lookup"><span data-stu-id="ed6cf-110">In the **Description** field, enter a description.</span></span>
6. <span data-ttu-id="ed6cf-111">Nella Scheda dettaglio **Generale**, nel campo **Stato** seleziona lo stato residenziale del gruppo di componenti TDS.</span><span class="sxs-lookup"><span data-stu-id="ed6cf-111">On the **General** FastTab, in the **Status** field, select the residential status of the TDS component group.</span></span> <span data-ttu-id="ed6cf-112">Le opzioni sono **Residente** e **Non residente**.</span><span class="sxs-lookup"><span data-stu-id="ed6cf-112">The options are **Resident** and **Non-resident**.</span></span>
7. <span data-ttu-id="ed6cf-113">Nel campo **Codice sezione**, immetti il codice sezione applicato al gruppo di componenti TDS.</span><span class="sxs-lookup"><span data-stu-id="ed6cf-113">In the **Section code** field, enter the section code that is applied to the TDS component group.</span></span>
8. <span data-ttu-id="ed6cf-114">Chiudi la pagina.</span><span class="sxs-lookup"><span data-stu-id="ed6cf-114">Close the page.</span></span>
