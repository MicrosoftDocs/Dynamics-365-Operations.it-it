[!include [banner](../includes/banner.md)]

    <span data-ttu-id="25436-106">[![Pagina Periodi liquidazione ritenuta d'acconto](./media/apac-ind-TDS-50.png)](./media/apac-ind-TDS-50.png)</span><span class="sxs-lookup"><span data-stu-id="25436-106">[![Withholding tax settlement periods page](./media/apac-ind-TDS-50.png)](./media/apac-ind-TDS-50.png)</span></span>

    <span data-ttu-id="25436-108">La scheda **Panoramica** mostra le seguenti informazioni:</span><span class="sxs-lookup"><span data-stu-id="25436-108">The **Overview** tab shows the following information:</span></span>

    - <span data-ttu-id="25436-109">**Data** - La data della liquidazione TDS.</span><span class="sxs-lookup"><span data-stu-id="25436-109">**Date** – The date of TDS settlement.</span></span>
    - <span data-ttu-id="25436-110">**Giustificativo** - Il numero di giustificativo della transazione di liquidazione TDS.</span><span class="sxs-lookup"><span data-stu-id="25436-110">**Voucher** – The voucher number of the TDS settlement transaction.</span></span>
    - <span data-ttu-id="25436-111">**Tipo di imposta** - Il tipo di imposta per cui viene eseguito il processo di liquidazione.</span><span class="sxs-lookup"><span data-stu-id="25436-111">**Tax type** – The tax type that the settlement process is run for.</span></span>
    - <span data-ttu-id="25436-112">**Numero conto imposta (TAN)** - Il Numero conto imposta (TAN) della transazione TDS liquidata.</span><span class="sxs-lookup"><span data-stu-id="25436-112">**Tax Account Number (TAN)** – The Tax Account Number (TAN) of the settled TDS transaction.</span></span>

    - <span data-ttu-id="25436-114">**Data iniziale** - La data di inizio del periodo di liquidazione.</span><span class="sxs-lookup"><span data-stu-id="25436-114">**From date** – The start date of the settlement period.</span></span>
    - <span data-ttu-id="25436-115">**Data finale** - La data di fine del periodo di liquidazione.</span><span class="sxs-lookup"><span data-stu-id="25436-115">**To date** – The end date of the settlement period.</span></span>
    - <span data-ttu-id="25436-116">**Versione pagamento ritenuta d'acconto** - La versione del pagamento della ritenuta d'acconto: **Originale**, **Ultime correzioni** o **Elenco totale**.</span><span class="sxs-lookup"><span data-stu-id="25436-116">**Withholding tax payment version** – The version of the withholding tax payment: **Original**, **Latest corrections**, or **Total list**.</span></span>
