Questo articolo descrive l'opzione Stato del piano di benefit per dipendenti impostata per Dynamics 365 Human Resources.

## <a name="option-set-process-type"></a>Set di opzioni: Tipo di processo

Nome fisico: mshr_hrmCompProcessType

Questa enumerazione fornisce il set di opzioni per il tipo di processo nelle entità virtuali di Risorse umane.

| Valore | Etichetta | descrizione |
| --- | --- | --- |
| 200000000 | Nessuna priorità | Nessuna. |
| 200000001 | Selezionate | Selezionate. |
| 200000002 | Con rinuncia | Con rinuncia. |
