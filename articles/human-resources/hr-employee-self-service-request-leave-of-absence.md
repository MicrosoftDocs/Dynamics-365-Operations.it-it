# <a name="request-a-leave-of-absence"></a>Richiedere un congedo

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

È possibile inviare una richiesta di congedo e vedere lo stato delle richieste di congedo in Dynamics 365 Human Resources.

## <a name="request-a-leave-of-absence"></a>Richiedere un congedo

1. Nell'area di lavoro **Dipendente self-service**, selezionare **Altro** nel riquadro **Saldi permessi**.

2. Per inviare una richiesta di congedo, selezionare **Richiedi congedo**.

3. Immettere le informazioni per **Tipo di congedo**, **Data di inizio** e **Data di fine**.

4. Se è necessario inviare documentazione di supporto, selezionare **Carica** sotto **Allegati**.

5. Immettere informazioni in **Commento**, se necessario.

6. Selezionare **Invia** quando si è pronti per inviare la richiesta. Altrimenti, selezionare **Salva bozza**.


## <a name="view-leave-of-absence-request-status"></a>Visualizzare lo stato della richiesta di congedo

1. Nell'area di lavoro **Dipendente self-service**, selezionare **Altro** nel riquadro **Saldi permessi**.

2. Per visualizzare le richieste di congedo, selezionare **Visualizza congedo**.
