<span data-ttu-id="f08ed-363">L'integrazione da Human Resources a Finance tenta di abbinare i record in base all'ID.</span><span class="sxs-lookup"><span data-stu-id="f08ed-363">The integration from Human Resources to Finance attempts to match records based on the ID.</span></span> <span data-ttu-id="f08ed-364">Se i record corrispondono, l'integrazione dei dati sovrascrive i dati in Finance con i valori in Human Resources.</span><span class="sxs-lookup"><span data-stu-id="f08ed-364">If the records match, the Data Integrator overwrites the data in Finance with the values in Human Resources.</span></span> <span data-ttu-id="f08ed-365">Tuttavia, può verificarsi un problema se logicamente si tratta di record differenti e lo stesso ID è stato generato in Human Resources o Finance in base alla rispettiva sequenza numerica.</span><span class="sxs-lookup"><span data-stu-id="f08ed-365">However, an issue may occur if logically these are different records and the same ID was generated in either Human Resources or Finance based on the respective number sequence.</span></span>
