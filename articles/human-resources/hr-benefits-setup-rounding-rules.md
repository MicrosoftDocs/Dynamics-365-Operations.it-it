# <a name="set-up-rounding-rules"></a>Impostare le regole di arrotondamento

È possibile impostare regole di arrotondamento in Dynamics 365 Human Resources per soddisfare le esigenze dell'organizzazione.

1. Nell'area di lavoro **Gestione benefit**, sotto **Impostazione**, selezionare **Regole di arrotondamento**.

2. Selezionare **Nuovo**.

3. Specificare i valori per **Arrotondamento all'unità più vicina**, **Scenario** e **Tipo di arrotondamento**.

4. Selezionare **Salva**.
