# <a name="payroll---gender-option-set"></a>Retribuzioni- Set di opzioni Genere


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Questo argomento descrive l'opzione Genere impostata per Dynamics 365 Human Resources.

## <a name="option-set-gender-option"></a>Set di opzioni: opzione Genere

Questa enumerazione fornisce il set di opzioni di generi per il dipendente. Questo è disponibile nel set di opzioni mshr_hcmpersongender.

| Valore | Etichetta | Descrizione |
| --- | --- | --- |
| 200000000 | Nessuna priorità | Nessun genere è stato specificato. |
| 200000001 | Maschio | Maschio. |
| 200000002 | Femmina | Femmina. |
| 200000003 | NonSpecific | Selezione per un genere non specifico. |
