[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Questo argomento descrive l'opzione No/Sì impostata per Dynamics 365 Human Resources.

## <a name="option-set-no-yes"></a>Set di opzioni: No/Sì

Nome fisico: mshr_noyes

Questa enumerazione fornisce un set di opzioni per le proprietà booleane tipiche nelle entità virtuali Human Resources.

| Valore | Etichetta | Descrizione |
| --- | --- | --- |
| 200000000 | No | N. |
| 200000001 | Sì | Sì. |
