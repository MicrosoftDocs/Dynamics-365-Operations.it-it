# <a name="blank-yes-no"></a>Vuoto Sì No

Questo argomento descrive l'opzione Vuoto Sì No impostata per Dynamics 365 Human Resources.

Nome fisico: mshr_hrmblankyesno

Questa enumerazione fornisce un set di opzioni per le proprietà sì/no che consentono anche i campi vuoti.

| Valore | Etichetta | descrizione |
| --- | --- | --- |
| 200000000 | Vuoto | Non è stato selezionato alcun valore. |
| 200000001 | Sì | Sì. |
| 200000002 | Nessuno | N. |
