# <a name="contact-type"></a>Tipo di contatto

Questo argomento descrive l'opzione Tipo contatto impostata per Dynamics 365 Human Resources.

Nome fisico: mshr_logisticselectronicaddressmethodtype

Questa enumerazione fornisce il set di opzioni dei valori per i tipi di contatto dei candidati. 

| Valore | Etichetta | descrizione |
| --- | --- | --- |
| 200000000 | Nessuna priorità | Nessun tipo selezionato. |
| 200000001 | Telefono | Numero telefonico. |
| 200000002 | Indirizzo di posta elettronica | Indirizzo e-mail. |
| 200000003 | URL | URL sito Web. |
| 200000004 | Telex | Numero di telex. |
| 200000005 | Fax | Numero di fax. |
| 200000006 | Facebook | Conto Facebook. Identificato da ID utente. |
| 200000007 | Twitter | Account Twitter. Identificato da @username. |
| 200000008 | LinkedIn | Account LinkedIn. Identificato da nome utente. |
