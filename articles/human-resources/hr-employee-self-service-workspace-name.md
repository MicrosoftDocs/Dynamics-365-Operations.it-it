[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

<span data-ttu-id="35642-104">Se si impiegano volontari o altre persone non dipendenti, è possibile che si intenda cambiare il nome dell'area di lavoro **Dipendente self-service**.</span><span class="sxs-lookup"><span data-stu-id="35642-104">If you have volunteers or other non-employees, you might want to change the name of the **Employee self-service** workspace.</span></span> <span data-ttu-id="35642-105">È possibile modificare questa area di lavoro in **Self-service**.</span><span class="sxs-lookup"><span data-stu-id="35642-105">You can change this workspace to **Self service** instead.</span></span>

> [!NOTE]
> <span data-ttu-id="35642-106">La modifica del nome dell'area di lavoro **Dipendente self-service** modifica anche la voce di menu utilizzata internamente da Dynamics 365 Human Resources.</span><span class="sxs-lookup"><span data-stu-id="35642-106">Changing the name of the **Employee self-service** workspace also changes the menu item that is used internally by Dynamics 365 Human Resources.</span></span> <span data-ttu-id="35642-107">Se in precedenza sono state applicate le personalizzazioni di sicurezza alla voce di menu **HcmEmployeeSelfServiceWorkspace**, si consiglia di applicare le stesse modifiche a **HcmSelfServiceWorkspace** per mantenere la parità.</span><span class="sxs-lookup"><span data-stu-id="35642-107">If you previously applied security customizations to the **HcmEmployeeSelfServiceWorkspace** menu item, we recommend applying the same changes to **HcmSelfServiceWorkspace** to maintain parity.</span></span>

3. <span data-ttu-id="35642-110">Sotto **Nome visualizzato**, selezionare **Self-service**.</span><span class="sxs-lookup"><span data-stu-id="35642-110">Under **Display name**, select **Self service**.</span></span>

   ![Modificare il nome dell'area di lavoro Dipendente self-service in Self-service](./media/hr-employee-self-service-workspace-name.png)

4. <span data-ttu-id="35642-112">Selezionare **Salva**.</span><span class="sxs-lookup"><span data-stu-id="35642-112">Select **Save**.</span></span>
