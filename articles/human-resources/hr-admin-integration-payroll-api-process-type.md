# <a name="process-type"></a>Tipo di processo

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Questo argomento descrive l'opzione No/Sì impostata per Dynamics 365 Human Resources.

## <a name="option-set-process-type"></a>Set di opzioni: Tipo di processo

Nome fisico: mshr_hrmCompProcessType

Questa enumerazione fornisce un set di opzioni per il tipo di processo nelle entità virtuali delle Risorse umane.

| Valore | Etichetta | descrizione |
| --- | --- | --- |
| 200000000 | Nessuna priorità | Nessuna. |
| 200000001 | Annuale | Annuale. |
| 200000002 | Anniversario | Anniversario. |
| 200000003 | Non ciclico | Non ciclico. |
| 200000004 | Semestrale | Semestrale. |
| 200000005 | Trimestrale | Trimestrale. |
| 200000006 | Ogni mese | Mensile. |
