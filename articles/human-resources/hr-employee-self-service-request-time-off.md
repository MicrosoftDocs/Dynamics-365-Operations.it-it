# <a name="request-time-off"></a>Richiedere permessi

È possibile inviare richieste di permessi, visualizzare i saldi di ferie e congedi e lo stato delle richieste di congedo in Dynamics 365 Human Resources.

## <a name="request-time-off"></a>Richiedere permessi

1. Nell'area di lavoro **Dipendente self-service**, selezionare **Richiedi permessi** nel riquadro **Saldi permessi**.

2. Immettere informazioni per **Tipo di congedo**, **Codice motivo**, **Data di inizio** e **Data di fine**.

3. Sotto **Date**, selezionare le date per la richiesta di congedo.

4. Se è necessario inviare documentazione di supporto, selezionare **Carica** sotto **Allegati**.

5. Immettere informazioni in **Commento**, se necessario.

6. Selezionare **Invia** quando si è pronti per inviare la richiesta. Altrimenti, selezionare **Salva bozza**.

## <a name="view-leave-balances"></a>Visualizzare saldi di congedo

1. Nell'area di lavoro **Dipendente self-service**, selezionare **Altro** nel riquadro **Saldi permessi**.

2. Selezionare **Saldi**.

## <a name="view-leave-request-status"></a>Visualizzare lo stato della richiesta di congedo

1. Nell'area di lavoro **Dipendente self-service**, selezionare **Altro** nel riquadro **Saldi permessi**.

2. Per visualizzare le richieste di permesso approvate, selezionare **Permesso approvato**. Per visualizzare le richieste di permesso in sospeso, selezionare **Richieste di permesso**.
