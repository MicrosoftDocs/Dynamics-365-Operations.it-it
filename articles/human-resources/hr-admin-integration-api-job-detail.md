Questo argomento fornisce dettagli e una query di esempio per l'entità Dettagli posizione lavorativa in Dynamics 365 Human Resources.

| Proprietà<br>**Nome fisico**<br>**_Tipo_** | Utilizza | descrizione |
| --- | --- | --- |
| **ID lavoro**<br>mshr_jobid<br>*String* | Sola lettura<br>Richiesto | ID univoco per una mansione. |
| **Soglia bassa prezzo di mercato**<br>mshr_marketpricelowthreshold<br>*Decimali* | | Un valore GUID generato dal sistema per identificare in modo univoco la posizione.  |
