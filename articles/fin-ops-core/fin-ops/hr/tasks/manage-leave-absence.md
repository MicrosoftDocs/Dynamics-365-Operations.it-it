# <a name="manage-leave-of-absence"></a>Gestire i congedi

[!include [banner](../../includes/banner.md)]

In questa procedura viene descritta la creazione dei record congedo dipendente. È possibile tenere traccia del tempo di congedo per i motivi che includono le attività mediche, educative o parentali. La società di dati dimostrativi utilizzata per creare questa procedura è USMF.

1. Andare a Risorse umane > Lavoratori > Dipendenti.
2. Selezionare un dipendente nell'elenco.
3. Visualizzare informazioni dettagliate per il dipendente selezionato selezionando il nome del dipendente.
4. Fare clic sulla scheda Impiego.
5. Fare clic su Congedo.
6. Fare clic su Nuovo.
7. Nel campo Tipo di congedo fare clic sul pulsante a discesa per aprire la ricerca.
    * È possibile associare un tipo di congedo a un codice reddito nel modulo Tipi di congedo. Se un tipo di congedo è associato un codice reddito, verrà generata una riga reddito con il codice reddito associato durante il periodo di congedo immesso.  
8. Nell'elenco selezionare un tipo di congedo. 
    * Ad esempio: Adozione  
9. Immettere la data di inizio del congedo. Esempio: "26/10/2015"
    * Ad esempio: 26/10/2015  
10. Immettere la data di inizio del congedo. 
    * Ad esempio: 20/11/2015  
11. Nel campo Nota immettere una descrizione.
    * Ad esempio: Congedo per adozione  
12. Fare clic su Salva.
