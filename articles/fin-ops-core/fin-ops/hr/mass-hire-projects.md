# <a name="mass-hire-projects"></a><span data-ttu-id="62ea8-103">Progetti di assunzione collettiva</span><span class="sxs-lookup"><span data-stu-id="62ea8-103">Mass hire projects</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="62ea8-104">I progetti di assunzione collettiva consentono ai responsabili delle risorse umane di creare più posizioni e assumere in modo efficiente i lavoratori per tali posizioni.</span><span class="sxs-lookup"><span data-stu-id="62ea8-104">Mass hire projects allow human resources specialists to create multiple positions and efficiently hire workers into those positions.</span></span>

<span data-ttu-id="62ea8-106">Quando si assumono più lavoratori contemporaneamente, ad esempio per soddisfare una domanda stagionale, utilizzare progetti di assunzione collettiva.</span><span class="sxs-lookup"><span data-stu-id="62ea8-106">Use mass hire projects when you hire multiple workers at one time, such as when you hire to meet a seasonal demand.</span></span> <span data-ttu-id="62ea8-107">La creazione del progetto di assunzione collettiva è utile poiché è possibile creare record per le posizioni, i lavoratori e le assegnazioni dei lavoratori per le posizioni contemporaneamente.</span><span class="sxs-lookup"><span data-stu-id="62ea8-107">Creating a mass hire project is useful because you can create position records, worker records, and worker assignments for positions at the same time.</span></span> <span data-ttu-id="62ea8-108">Quando si creano posizioni per un progetto di assunzione collettiva, è possibile specificare le seguenti informazioni:</span><span class="sxs-lookup"><span data-stu-id="62ea8-108">When you create positions for a mass hire project, you can specify the following information:</span></span>

- <span data-ttu-id="62ea8-109">Numero di posizioni da creare</span><span class="sxs-lookup"><span data-stu-id="62ea8-109">The number of positions to create</span></span>
- <span data-ttu-id="62ea8-110">Il tipo di lavoratore per le persone che saranno assunte per le posizioni</span><span class="sxs-lookup"><span data-stu-id="62ea8-110">The worker type of the people that you will hire for the positions</span></span>
- <span data-ttu-id="62ea8-111">Il reparto e la mansione associati alle posizioni</span><span class="sxs-lookup"><span data-stu-id="62ea8-111">The department and the job that are associated with the positions</span></span>
- <span data-ttu-id="62ea8-112">Il valore equivalente al tempo pieno della posizione</span><span class="sxs-lookup"><span data-stu-id="62ea8-112">The full-time equivalent value of the position</span></span>

## <a name="example"></a><span data-ttu-id="62ea8-113">Esempio</span><span class="sxs-lookup"><span data-stu-id="62ea8-113">Example</span></span>

<span data-ttu-id="62ea8-114">In estate, in genere si assumono 15-20 studenti universitari part-time per coprire le posizioni interne disponibili nella società.</span><span class="sxs-lookup"><span data-stu-id="62ea8-114">In the summer, you usually hire 15-20 part-time college students to fill available internships in your company.</span></span> <span data-ttu-id="62ea8-115">Quest'anno, si desidera assumere cinque ragionieri, cinque elaboratori di ordini e cinque cassieri.</span><span class="sxs-lookup"><span data-stu-id="62ea8-115">This year, you want to hire five accountants, five order processors, and five cashiers.</span></span> <span data-ttu-id="62ea8-116">Anziché creare ogni record lavoratore e ogni record di posizione separatamente, creare un progetto di assunzione collettiva denominato "SummerInterns".</span><span class="sxs-lookup"><span data-stu-id="62ea8-116">Instead of creating each position record and worker record separately, you create one mass hire project called "SummerInterns".</span></span> <span data-ttu-id="62ea8-117">Le date di inizio e di fine del progetto sono correlate alle date di inizio e di fine delle durate delle posizioni create per il progetto di assunzione collettiva.</span><span class="sxs-lookup"><span data-stu-id="62ea8-117">The project start and end dates correlate with the start and end dates of the position durations for the positions you create for the mass hire project.</span></span>

<span data-ttu-id="62ea8-118">Nella pagina **Progetti di assunzione collettiva** selezionare il progetto "SummerInterns", quindi fare clic su **Apri progetto**.</span><span class="sxs-lookup"><span data-stu-id="62ea8-118">In the **Mass hire projects** page, select the "SummerInterns" project and then click **Open project**.</span></span> <span data-ttu-id="62ea8-119">Nel progetto di assunzione collettiva aperto, fare clic su **Crea posizioni** e immettere le informazioni sulla posizione contabile.</span><span class="sxs-lookup"><span data-stu-id="62ea8-119">In the open mass hire project, click **Create positions** and enter information about the accountant position.</span></span> <span data-ttu-id="62ea8-120">È possibile indicare che si desidera creare cinque posizioni contabili che utilizzano le stesse informazioni e quindi fare clic su OK.</span><span class="sxs-lookup"><span data-stu-id="62ea8-120">You can indicate that five accountant positions should be created using the same information for each one, and then click OK.</span></span> <span data-ttu-id="62ea8-121">Ripetere questo processo per le posizioni di elaboratore di ordini e di cassiere.</span><span class="sxs-lookup"><span data-stu-id="62ea8-121">Repeat this process for the order processor and cashier positions.</span></span>

<span data-ttu-id="62ea8-122">Dopo aver selezionato gli studenti da assumere per le posizioni interne, immettere le informazioni degli studenti in **Dettagli posizioni** per la posizione per cui vengono assunti.</span><span class="sxs-lookup"><span data-stu-id="62ea8-122">After selecting students to hire for the internship positions, you'll enter each student's information in the **Position details** for the position that you're hiring them for.</span></span> <span data-ttu-id="62ea8-123">Dopo aver immesso in tutti i dettagli della posizione, selezionare la posizione nella pagina Progetti di assunzione collettiva, quindi fare clic su **Assumi**.</span><span class="sxs-lookup"><span data-stu-id="62ea8-123">When you have entered all of the position details, select the position in the Mass hire projects page, and then click **Hire**.</span></span> <span data-ttu-id="62ea8-124">Un record posizione verrà creato per ogni posizione e un record lavoratore verrà creato e assegnato alla posizione corretta per ogni persona che si assume.</span><span class="sxs-lookup"><span data-stu-id="62ea8-124">A position record will be created for each position and a worker record will be created and assigned to the correct position for each person who you hire.</span></span>
