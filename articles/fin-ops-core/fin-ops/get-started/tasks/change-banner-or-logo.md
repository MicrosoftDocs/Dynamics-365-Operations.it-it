# <a name="change-the-banner-or-logo"></a><span data-ttu-id="ed9cd-103">Modificare banner o logo</span><span class="sxs-lookup"><span data-stu-id="ed9cd-103">Change the banner or logo</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="ed9cd-104">La procedura riportata di seguito elenca i passaggi che gli amministratori di sistema possono utilizzare per aggiornare l'immagine di logo o banner visualizzata per una persona giuridica.</span><span class="sxs-lookup"><span data-stu-id="ed9cd-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="ed9cd-105">La società di dati dimostrativi utilizzata per creare questa procedura è USMF.</span><span class="sxs-lookup"><span data-stu-id="ed9cd-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="ed9cd-106">Andare a **Pannello di navigazione > Moduli > Amministrazione organizzazione > Organizzazioni Persone giuridiche**.</span><span class="sxs-lookup"><span data-stu-id="ed9cd-106">Go to **Navigation pane > Modules > Organization administration > Organizations > Legal entities**.</span></span>
2. <span data-ttu-id="ed9cd-107">Nell'elenco a sinistra, selezionare la persona giuridica per cui si desidera aggiornare il banner o il logo.</span><span class="sxs-lookup"><span data-stu-id="ed9cd-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="ed9cd-108">Se è già selezionata, andare al passaggio successivo.</span><span class="sxs-lookup"><span data-stu-id="ed9cd-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="ed9cd-109">Espandere la scheda **Immagine nel dashboard**.</span><span class="sxs-lookup"><span data-stu-id="ed9cd-109">Expand **Dashboard image** tab.</span></span>
4. <span data-ttu-id="ed9cd-110">Selezionare **Modifica**.</span><span class="sxs-lookup"><span data-stu-id="ed9cd-110">Select **Change**.</span></span>
    
    <span data-ttu-id="ed9cd-111">La risoluzione ideale per un'immagine di tipo banner è 1920 x 281 pixel.</span><span class="sxs-lookup"><span data-stu-id="ed9cd-111">The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="ed9cd-112">La larghezza ideale per un'immagine di tipo logo è 350 pixel.</span><span class="sxs-lookup"><span data-stu-id="ed9cd-112">The ideal width for a logo image is 350 pixels.</span></span>
    
5. <span data-ttu-id="ed9cd-113">Selezionare **Salva**.</span><span class="sxs-lookup"><span data-stu-id="ed9cd-113">Select **Save**.</span></span>
6. <span data-ttu-id="ed9cd-114">Andare a **Pannello di navigazione > Moduli > Comune > Comune > Dashboard predefinito**.</span><span class="sxs-lookup"><span data-stu-id="ed9cd-114">Go to **Navigation pane > Modules > Common > Common > Default dashboard**.</span></span> <span data-ttu-id="ed9cd-115">La nuova immagine di logo o banner viene visualizzata nel dashboard.</span><span class="sxs-lookup"><span data-stu-id="ed9cd-115">You should see the new banner or logo image on the dashboard.</span></span>  
    
    <span data-ttu-id="ed9cd-116">Potrebbe essere necessario modificare la società, utilizzando la selezione delle società, impostando quella per cui è stato caricato il banner.</span><span class="sxs-lookup"><span data-stu-id="ed9cd-116">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
