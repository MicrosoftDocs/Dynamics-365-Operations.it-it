# <a name="create-an-operating-unit"></a><span data-ttu-id="a54da-103">Creare un'unità operativa</span><span class="sxs-lookup"><span data-stu-id="a54da-103">Create an operating unit</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="a54da-104">Un'unità operativa è un'organizzazione utilizzata per dividere il controllo delle risorse economiche e dei processi operativi in un'azienda.</span><span class="sxs-lookup"><span data-stu-id="a54da-104">An operating unit is an organization that is used to divide the control of economic resources and operational processes in a business.</span></span> <span data-ttu-id="a54da-105">Le persone in un'unità operativa hanno il compito di ottimizzare l'utilizzo delle risorse meno efficienti, di migliorare i processi e di rendere conto delle loro prestazioni.</span><span class="sxs-lookup"><span data-stu-id="a54da-105">People in an operating unit have a duty to maximize the use of scarce resources, improve processes, and account for their performance.</span></span> <span data-ttu-id="a54da-106">I tipi di unità operative includono centri di costo, Business Unit, reparti e flussi del valore.</span><span class="sxs-lookup"><span data-stu-id="a54da-106">The types of operating units include cost centers, business units, departments, and value streams.</span></span> <span data-ttu-id="a54da-107">Per creare un'unità operativa, attenersi alla seguente procedura.</span><span class="sxs-lookup"><span data-stu-id="a54da-107">Use the following procedure to create an operating unit.</span></span> <span data-ttu-id="a54da-108">La società di dati dimostrativi utilizzata per creare questa procedura è USMF.</span><span class="sxs-lookup"><span data-stu-id="a54da-108">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="a54da-109">Andare a **Pannello di navigazione > Moduli > Amministrazione organizzazione > Organizzazioni > Unità operative**.</span><span class="sxs-lookup"><span data-stu-id="a54da-109">Go to **Navigation pane > Modules > Organization administration > Organizations > Operating units.**</span></span>
2. <span data-ttu-id="a54da-110">Fare clic su **Nuovo** per aprire la finestra di dialogo a discesa.</span><span class="sxs-lookup"><span data-stu-id="a54da-110">Click **New** to open the drop dialog.</span></span>
3. <span data-ttu-id="a54da-111">Nell'elenco trovare e selezionare il record desiderato.</span><span class="sxs-lookup"><span data-stu-id="a54da-111">In the list, find and select the desired record.</span></span> <span data-ttu-id="a54da-112">Selezionare il tipo di unità operativa che si desidera creare.</span><span class="sxs-lookup"><span data-stu-id="a54da-112">Select the type of operating unit you want to create.</span></span>  
4. <span data-ttu-id="a54da-113">Nell'elenco fare clic sul collegamento nella riga selezionata.</span><span class="sxs-lookup"><span data-stu-id="a54da-113">In the list, click the link in the selected row.</span></span>
5. <span data-ttu-id="a54da-114">Digitare un valore nel campo **Nome**.</span><span class="sxs-lookup"><span data-stu-id="a54da-114">In the **Name** field, type a value.</span></span>
    + <span data-ttu-id="a54da-115">Espandere la sezione **Generale**, se necessario.</span><span class="sxs-lookup"><span data-stu-id="a54da-115">Expand the **General** section, if necessary.</span></span>  
    + <span data-ttu-id="a54da-116">Fornire informazioni generali sull'unità operativa, ad esempio il numero di identificazione, il numero DUNS e il manager.</span><span class="sxs-lookup"><span data-stu-id="a54da-116">Provide general information about the operating unit, such as an identification number, DUNS number, and manager.</span></span>    
    + <span data-ttu-id="a54da-117">Espandere la sezione **Indirizzi**, se necessario.</span><span class="sxs-lookup"><span data-stu-id="a54da-117">Expand the **Addresses** section, if necessary.</span></span>  
    + <span data-ttu-id="a54da-118">Immettere informazioni relative all'indirizzo, quali il nome della via e il numero civico, il codice postale e la città.</span><span class="sxs-lookup"><span data-stu-id="a54da-118">Enter address information, such as the street name and number, postal code, and city.</span></span> <span data-ttu-id="a54da-119">Fare clic su **Aggiungi** per immettere un nuovo record indirizzo o su Modifica per modificare un record indirizzo esistente.</span><span class="sxs-lookup"><span data-stu-id="a54da-119">Click **Add** to enter a new address record, or click Edit to modify an existing address record.</span></span>   
    + <span data-ttu-id="a54da-120">Espandere la sezione **Informazioni contatto** se necessario.</span><span class="sxs-lookup"><span data-stu-id="a54da-120">Expand the **Contact information** section, if necessary.</span></span>  
    + <span data-ttu-id="a54da-121">Immettere informazioni sui metodi di comunicazione, ad esempio indirizzi di posta elettronica, URL e numeri di telefono.</span><span class="sxs-lookup"><span data-stu-id="a54da-121">Enter information about methods of communication, such as email addresses, URLs, and telephone numbers.</span></span> <span data-ttu-id="a54da-122">Per immettere un nuovo record di comunicazione, fare clic su Nuovo.</span><span class="sxs-lookup"><span data-stu-id="a54da-122">To enter a new communication record, click New.</span></span> <span data-ttu-id="a54da-123">Per modificare un record di comunicazione esistente, fare clic su **Altre opzioni > Avanzate**.</span><span class="sxs-lookup"><span data-stu-id="a54da-123">To modify an existing communication record, click **More options > Advanced**.</span></span>   
6. <span data-ttu-id="a54da-124">Se lo si desidera, modificare **Numero unità operativa** come necessario.</span><span class="sxs-lookup"><span data-stu-id="a54da-124">Optionally, change the **Operating unit number** as needed.</span></span> <span data-ttu-id="a54da-125">Da notare che questo numero è un identificatore univoco per il record **Parte** corrispondente e non può essere uguale a qualsiasi altra unità operativa.</span><span class="sxs-lookup"><span data-stu-id="a54da-125">Note that this number is a unique idenitifier for the correspondng **Party** record and cannot be the same as any other operating unit.</span></span>
7. <span data-ttu-id="a54da-126">Selezionare **Salva**.</span><span class="sxs-lookup"><span data-stu-id="a54da-126">Select **Save**.</span></span>
