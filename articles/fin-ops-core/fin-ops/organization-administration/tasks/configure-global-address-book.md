# <a name="configure-the-global-address-book"></a><span data-ttu-id="395d5-103">Configurare la rubrica globale</span><span class="sxs-lookup"><span data-stu-id="395d5-103">Configure the global address book</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="395d5-104">Utilizzare questa procedura per impostare i valori predefiniti e i criteri di sicurezza della rubrica globale.</span><span class="sxs-lookup"><span data-stu-id="395d5-104">Use this procedure to set the default values and security policies for the global address book.</span></span> 

<span data-ttu-id="395d5-105">La società di dati dimostrativi utilizzata per creare questa attività è USMF.</span><span class="sxs-lookup"><span data-stu-id="395d5-105">The demo data company used to create this task is USMF.</span></span> <span data-ttu-id="395d5-106">Questa attività è destinata del team di pianificazione e configurazione.</span><span class="sxs-lookup"><span data-stu-id="395d5-106">This task is intended for the Planning and configuration team.</span></span>

1. <span data-ttu-id="395d5-107">Nel pannello di navigazione andare a **Moduli > Amministrazione organizzazione > Rubrica globale > Parametri Rubrica globale**.</span><span class="sxs-lookup"><span data-stu-id="395d5-107">In the Navigation pane, go to **Modules > Organization administration > Global address book > Global address book parameters**.</span></span>
2. <span data-ttu-id="395d5-108">Nel campo **Sequenza nome** selezionare la modalità di visualizzazione dei nomi.</span><span class="sxs-lookup"><span data-stu-id="395d5-108">In the **Name sequence** field, select how names should be shown.</span></span>
3. <span data-ttu-id="395d5-109">In **Elimina parti senza ruoli**, selezionare se eliminare le parti a cui non sono stati assegnati ruoli.</span><span class="sxs-lookup"><span data-stu-id="395d5-109">In **Delete parties with no roles**, select whether to delete parties with that have not been assigned a role.</span></span>
4. <span data-ttu-id="395d5-110">In **Usa verifica duplicati**, selezionare se verificare i record duplicati.</span><span class="sxs-lookup"><span data-stu-id="395d5-110">In **Use duplicate check**, select whether to check for duplicate records.</span></span>
5. <span data-ttu-id="395d5-111">In **Visualizza numero DUNS negli indirizzi**, selezionare se visualizzare il numero DUNS sugli indirizzi.</span><span class="sxs-lookup"><span data-stu-id="395d5-111">In **Display DUNS number on addresses**, select whether to display the DUNS number on addresses.</span></span>
6. <span data-ttu-id="395d5-112">In **Controlla numero DUNS univoco**, selezionare se verificare i numeri univoci DUNS.</span><span class="sxs-lookup"><span data-stu-id="395d5-112">In **Check for unique DUNS number**, select whether to check for unique DUNS numbers.</span></span>
7. <span data-ttu-id="395d5-113">Selezionare un'opzione nel campo **Parte**.</span><span class="sxs-lookup"><span data-stu-id="395d5-113">In the **Party** field, select an option.</span></span>
8. <span data-ttu-id="395d5-114">Selezionare un'opzione nel campo **Cliente**.</span><span class="sxs-lookup"><span data-stu-id="395d5-114">In the **Customer** field, select an option.</span></span>
9. <span data-ttu-id="395d5-115">Selezionare un'opzione nel campo **Fornitore**.</span><span class="sxs-lookup"><span data-stu-id="395d5-115">In the **Vendor** field, select an option.</span></span>
10. <span data-ttu-id="395d5-116">Selezionare un'opzione nel campo **Prospect**.</span><span class="sxs-lookup"><span data-stu-id="395d5-116">In the **Prospect** field, select an option.</span></span>
11. <span data-ttu-id="395d5-117">Selezionare un'opzione nel campo **Concorrente**.</span><span class="sxs-lookup"><span data-stu-id="395d5-117">In the **Competitor** field, select an option.</span></span>
12. <span data-ttu-id="395d5-118">Fare clic sulla scheda **Sicurezza ubicazione privata**.</span><span class="sxs-lookup"><span data-stu-id="395d5-118">Click the **Private location security** tab.</span></span>
13. <span data-ttu-id="395d5-119">Nell'elenco trovare e selezionare il record desiderato.</span><span class="sxs-lookup"><span data-stu-id="395d5-119">In the list, find and select the desired record.</span></span> <span data-ttu-id="395d5-120">Premere il tasto Maiusc per selezionare più ruoli da aggiungere al riquadro **Ruoli selezionati**, quindi fare clic sulla freccia per aggiungere i ruoli selezionati.</span><span class="sxs-lookup"><span data-stu-id="395d5-120">Press the Shift key to select multiple roles to add to the **Selected roles** pane and then click the arrow to add the selected roles.</span></span>  
14. <span data-ttu-id="395d5-121">Fare clic su **Salva**.</span><span class="sxs-lookup"><span data-stu-id="395d5-121">Click **Save**.</span></span>
