[!include [banner](../includes/banner.md)]

<span data-ttu-id="57c8e-104">Questo argomento spiega come scaricare la versione più recente di [Configurazioni per la creazione di report elettronici](general-electronic-reporting.md#Configuration) dalla [libreria di risorse condivise](../lifecycle-services/asset-library.md) in Microsoft Dynamics Lifecycle Services (LCS).</span><span class="sxs-lookup"><span data-stu-id="57c8e-104">This topic explains how to download the newest version of [Electronic reporting (ER) configurations](general-electronic-reporting.md#Configuration) from the [Shared asset library](../lifecycle-services/asset-library.md) in Microsoft Dynamics Lifecycle Services (LCS).</span></span>

1. <span data-ttu-id="57c8e-105">Accedere all'applicazione utilizzando uno dei seguenti ruoli:</span><span class="sxs-lookup"><span data-stu-id="57c8e-105">Sign in to the application by using one of the following roles:</span></span>

    - <span data-ttu-id="57c8e-106">Sviluppatore per la creazione di report elettronici</span><span class="sxs-lookup"><span data-stu-id="57c8e-106">Electronic reporting developer</span></span>
    - <span data-ttu-id="57c8e-107">Consulente funzionale per la creazione di report elettronici</span><span class="sxs-lookup"><span data-stu-id="57c8e-107">Electronic reporting functional consultant</span></span>
    - <span data-ttu-id="57c8e-108">Amministratore di sistema</span><span class="sxs-lookup"><span data-stu-id="57c8e-108">System administrator</span></span>

2. <span data-ttu-id="57c8e-109">Andare a **Amministrazione organizzazione** &gt; **Aree di lavoro** &gt; **Creazione di report elettronici**.</span><span class="sxs-lookup"><span data-stu-id="57c8e-109">Go to **Organization administration** &gt; **Workspaces** &gt; **Electronic reporting**.</span></span>
3. <span data-ttu-id="57c8e-110">Nella sezione **Provider di configurazione** selezionare il riquadro **Microsoft**.</span><span class="sxs-lookup"><span data-stu-id="57c8e-110">In the **Configuration providers** section, select the **Microsoft** tile.</span></span>
4. <span data-ttu-id="57c8e-111">Nel riquadro **Microsoft** selezionare **Archivi**.</span><span class="sxs-lookup"><span data-stu-id="57c8e-111">On the **Microsoft** tile, select **Repositories**.</span></span>

    <span data-ttu-id="57c8e-112">[![Riquadro Microsoft nella pagina delle configurazioni di localizzazione](./media/update-er-from-lcs-for-ms-open-ms-repositories-list.png)](./media/update-er-from-lcs-for-ms-open-ms-repositories-list.png)</span><span class="sxs-lookup"><span data-stu-id="57c8e-112">[![Microsoft tile on the Localization configurations page](./media/update-er-from-lcs-for-ms-open-ms-repositories-list.png)](./media/update-er-from-lcs-for-ms-open-ms-repositories-list.png)</span></span>

5. <span data-ttu-id="57c8e-113">Nella pagina **Archivi di configurazioni** selezionare nella griglia l'archivio esistente di tipo **LCS**.</span><span class="sxs-lookup"><span data-stu-id="57c8e-113">On the **Configuration repositories** page, in the grid, select the existing repository of the **LCS** type.</span></span> <span data-ttu-id="57c8e-114">Se questo archivio non viene visualizzato nella griglia, effettuare le seguenti operazioni:</span><span class="sxs-lookup"><span data-stu-id="57c8e-114">If this repository doesn't appear in the grid, follow these steps:</span></span>

    1. <span data-ttu-id="57c8e-115">Selezionare **Aggiungi** per aggiungere un archivio.</span><span class="sxs-lookup"><span data-stu-id="57c8e-115">Select **Add** to add a repository.</span></span>
    2. <span data-ttu-id="57c8e-116">Selezionare **LCS** come tipo di archivio.</span><span class="sxs-lookup"><span data-stu-id="57c8e-116">Select **LCS** as the repository type.</span></span>
    3. <span data-ttu-id="57c8e-117">Selezionare **Crea archivio**.</span><span class="sxs-lookup"><span data-stu-id="57c8e-117">Select **Create repository**.</span></span>
    4. <span data-ttu-id="57c8e-118">Se viene richiesta l'autorizzazione, seguire le istruzioni sullo schermo.</span><span class="sxs-lookup"><span data-stu-id="57c8e-118">If you're prompted about authorization, follow the on-screen instructions.</span></span>
    5. <span data-ttu-id="57c8e-119">Immettere un nome e una descrizione per l'archivio.</span><span class="sxs-lookup"><span data-stu-id="57c8e-119">Enter a name and description for the repository.</span></span>
    6. <span data-ttu-id="57c8e-120">Selezionare **OK** per confermare la nuova voce di archivio.</span><span class="sxs-lookup"><span data-stu-id="57c8e-120">Select **OK** to confirm the new repository entry.</span></span>
    7. <span data-ttu-id="57c8e-121">Nella griglia, selezionare il nuovo archivio di tipo **LCS**.</span><span class="sxs-lookup"><span data-stu-id="57c8e-121">In the grid, select the new repository of the **LCS** type.</span></span>

6. <span data-ttu-id="57c8e-122">Selezionare **Apri** per visualizzare l'elenco delle configurazioni ER per l'archivio selezionato.</span><span class="sxs-lookup"><span data-stu-id="57c8e-122">Select **Open** to view the list of ER configurations for the selected repository.</span></span>

    <span data-ttu-id="57c8e-123">[![Pagina Archivi di configurazione](./media/update-er-from-lcs-for-ms-make-lcs-repository.png)](./media/update-er-from-lcs-for-ms-make-lcs-repository.png)</span><span class="sxs-lookup"><span data-stu-id="57c8e-123">[![Configuration repositories page](./media/update-er-from-lcs-for-ms-make-lcs-repository.png)](./media/update-er-from-lcs-for-ms-make-lcs-repository.png)</span></span>

    > [!TIP]
    > <span data-ttu-id="57c8e-124">Se si verificano problemi con l'accesso all'archivio LCS per scaricare le configurazioni dalla libreria di risorse condivise in LCS, è possibile scaricare le configurazioni dal [repository globale](er-download-configurations-global-repo.md).</span><span class="sxs-lookup"><span data-stu-id="57c8e-124">If you have trouble accessing the LCS repository to download configurations from the Shared asset library in LCS, you can download configurations from the [Global repository](er-download-configurations-global-repo.md) instead.</span></span>

7. <span data-ttu-id="57c8e-125">Nella struttura di configurazioni nel riquadro sinistro, selezionare la configurazione per la creazione di report elettronici desiderata.</span><span class="sxs-lookup"><span data-stu-id="57c8e-125">In the configurations tree in the left pane, select the required ER configuration.</span></span>
8. <span data-ttu-id="57c8e-126">Nella scheda dettaglio **Versioni** selezionare la versione richiesta della configurazione ER.</span><span class="sxs-lookup"><span data-stu-id="57c8e-126">On the **Versions** FastTab, select the required version of the selected ER configuration.</span></span>
9. <span data-ttu-id="57c8e-127">Selezionare **Importa** per scaricare la versione selezionata da LCS nell'istanza corrente.</span><span class="sxs-lookup"><span data-stu-id="57c8e-127">Select **Import** to download the selected version from LCS to the current instance.</span></span>

    > [!NOTE]
    > <span data-ttu-id="57c8e-128">Il pulsante **Importa** non è disponibile per le versioni di configurazione di creazione di report elettronici già presenti nell'istanza corrente.</span><span class="sxs-lookup"><span data-stu-id="57c8e-128">The **Import** button is unavailable for ER configuration versions that are already present in the current instance.</span></span>

    <span data-ttu-id="57c8e-129">[![Configurazione della pagina dell'archivio](./media/update-er-from-lcs-for-ms-download-configuration.png)](./media/update-er-from-lcs-for-ms-download-configuration.png)</span><span class="sxs-lookup"><span data-stu-id="57c8e-129">[![Configuration repository page](./media/update-er-from-lcs-for-ms-download-configuration.png)](./media/update-er-from-lcs-for-ms-download-configuration.png)</span></span>
