# <a name="len-er-function"></a>Funzione ER LEN

[!include [banner](../includes/banner.md)]

La funzione `LEN` restituisce il numero di caratteri nella stringa specificata come valore *Intero*.

## <a name="syntax"></a>Sintassi

```vb
LEN (text)
```

## <a name="arguments"></a>Argomenti

`text`: *Stringa*

Un valore *Stringa* che specifica il testo.

## <a name="return-values"></a>Valori restituiti

*Intero*

Il valore numerico risultante.

## <a name="example"></a>Esempio

`LEN ("Sample")` restituisce **6**.
