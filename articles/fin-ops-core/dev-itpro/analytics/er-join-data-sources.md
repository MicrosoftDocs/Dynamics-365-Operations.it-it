[!include[banner](../includes/banner.md)]

<span data-ttu-id="12f4b-104">Durante la configurazione di mapping o formati del modello di report elettronici (ER), è possibile [aggiungere](#review) le origini dati richiesti di tipo **Join**.</span><span class="sxs-lookup"><span data-stu-id="12f4b-104">While configuring Electronic reporting (ER) model mappings or formats, you can [add](#review) required data sources of the **Join** type.</span></span> <span data-ttu-id="12f4b-105">In fase di progettazione, un'origine dati **Join** è configurata come un set di diverse origini dati, ognuna delle quali restituisce un elenco di record.</span><span class="sxs-lookup"><span data-stu-id="12f4b-105">At design time, a **Join** data source is configured as a set of several data sources each of which returns a list of records.</span></span> <span data-ttu-id="12f4b-106">Per ogni origine dati tranne la prima, è necessario definire le condizioni necessarie per unire i record delle origini dati correnti e precedenti.</span><span class="sxs-lookup"><span data-stu-id="12f4b-106">For every data source except the first one, you need to define necessary conditions to join records of the current and previous data sources.</span></span> <span data-ttu-id="12f4b-107">In fase di esecuzione, un'origine dati configurata di tipo **Join** [restituisce](#executeERformat) un unico elenco di record uniti contenente campi dai record di origini dati nidificate.</span><span class="sxs-lookup"><span data-stu-id="12f4b-107">At runtime, a configured data source of **Join** type [returns](#executeERformat) a single joined list of records containing fields from the records of nested data sources.</span></span>

<span data-ttu-id="12f4b-108">Attualmente sono supportati i seguenti tipi di join:</span><span class="sxs-lookup"><span data-stu-id="12f4b-108">The following types of joins are currently supported:</span></span>

- <span data-ttu-id="12f4b-109">Left outer join:</span><span class="sxs-lookup"><span data-stu-id="12f4b-109">Outer (left) join:</span></span>
    - <span data-ttu-id="12f4b-110">Unire tutti i record della prima origine dati (all'estrema sinistra) e quindi qualsiasi corrispondenza in base ai record delle condizioni configurate della seconda origine (all'estrema destra).</span><span class="sxs-lookup"><span data-stu-id="12f4b-110">Join all records of the first (left-most) data source and then any matching in accordance to configured conditions records of the second (right-most) data source.</span></span>
- <span data-ttu-id="12f4b-111">Right inner join:</span><span class="sxs-lookup"><span data-stu-id="12f4b-111">Inner (right) join:</span></span>
    - <span data-ttu-id="12f4b-112">Unire solo i record della prima origine dati (all'estrema sinistra) e quindi solo i record della seconda origine (all'estrema destra) corrispondenti tra loro in base alle condizioni configurate.</span><span class="sxs-lookup"><span data-stu-id="12f4b-112">Join only records of the first (left-most) data source and only records of the second (right-most) data source matching to each other in accordance to configured conditions.</span></span>

<span data-ttu-id="12f4b-113">Nell'origine dati configurata **Join**, quando tutte le origini dati sono di tipo **Record di tabella**, l'esecuzione dell'origine dati Join può essere [effettuata a livello di database](#analyze) utilizzando una singola istruzione SQL.</span><span class="sxs-lookup"><span data-stu-id="12f4b-113">In the configured **Join** data source, when all data sources are the **Table records** type, execution of the Join data source can be [performed at the database level](#analyze) using a single SQL statement.</span></span> <span data-ttu-id="12f4b-114">Questo rendiconto riduce il numero di chiamate al database, migliorando le prestazioni di mapping del modello.</span><span class="sxs-lookup"><span data-stu-id="12f4b-114">This statement reduces the number of database calls, which improves model-mapping performance.</span></span> <span data-ttu-id="12f4b-115">In caso contrario, l'esecuzione dell'origine dati **Join** viene eseguita in memoria.</span><span class="sxs-lookup"><span data-stu-id="12f4b-115">Otherwise, execution of **Join data** source is performed in memory.</span></span>
