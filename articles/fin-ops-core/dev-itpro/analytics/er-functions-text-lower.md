# <a name="lower-er-function"></a>Funzione ER LOWER

[!include [banner](../includes/banner.md)]

La funzione `LOWER` restituisce la stringa di testo specificata come valore *Stringa* dopo che Ã¨ stata convertita in lettere minuscole.

## <a name="syntax"></a>Sintassi

```vb
LOWER (text)
```

## <a name="arguments"></a>Argomenti

`text`: *Stringa*

Un valore *Stringa* che specifica il testo.

## <a name="return-values"></a>Valori restituiti

*Stringa*

Il valore di testo risultante.

## <a name="example"></a>Esempio

`LOWER ("Sample")` restituisce **"sample"**.
