# <a name="list-of-er-functions-in-the-mathematical-category"></a>Elenco delle funzioni ER nella categoria matematica

[!include [banner](../includes/banner.md)]

Le funzioni matematiche della creazione di report elettronici (ER) possono essere utilizzate per eseguire molti calcoli matematici comuni. Questo articolo fornisce un riepilogo di queste funzioni.

## <a name="list-of-supported-functions"></a>Elenco delle funzioni supportate

| Funzione | Descrizione |
|----------|-------------|
| [Abs](er-functions-mathematical-abs.md)             | Questa funzione restituisce il valore assoluto (modulo) del numero specificato come valore *Reale*. In altre parole, restituisce il numero senza il segno. |
| [Potenza](er-functions-mathematical-power.md)         | Questa funzione restituisce un valore *Reale* che rappresenta il risultato dell'aumento del numero positivo specificato alla potenza specificata. |
| [Round](er-functions-mathematical-round.md)         | Restituisce il numero specificato come valore *Reale* dopo che è stato arrotondato al numero di posizioni decimali specificato. |
| [RoundDown](er-functions-mathematical-rounddown.md) | Restituisce il numero specificato come valore *Reale* dopo che è stato arrotondato per difetto al numero di posizioni decimali specificato. |
| [RoundUp](er-functions-mathematical-roundup.md)     | Restituisce il numero specificato come valore *Reale* dopo che è stato arrotondato per eccesso al numero di posizioni decimali specificato. |
