# <a name="upper-er-function"></a>Funzione ER UPPER

[!include [banner](../includes/banner.md)]

La funzione `UPPER` restituisce la stringa di testo specificata come valore *Stringa* dopo che Ã¨ stata convertita in lettere maiuscole.

## <a name="syntax"></a>Sintassi

```vb
UPPER (text )
```

## <a name="arguments"></a>Argomenti

`text`: *Stringa*

Il percorso valido di un'origine dati del tipo *Stringa*.

## <a name="return-values"></a>Valori restituiti

*Stringa*

Il valore di testo risultante.

## <a name="example"></a>Esempio

`UPPER ("Sample")` restituisce **"SAMPLE"**.
