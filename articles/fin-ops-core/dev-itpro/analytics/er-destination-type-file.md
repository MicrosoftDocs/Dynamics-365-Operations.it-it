# <a name="file-destination"></a>Destinazione file

[!include [banner](../includes/banner.md)]

È possibile configurare una destinazione file per ogni componente CARTELLA o FILE di un formato ER configurato per generare documenti in uscita. In base all'impostazione della destinazione, un documento generato è disponibile per il download dal browser Web.

Nella pagina **Impostazioni destinazione**, se si imposta **Abilitato** su **Sì**, viene visualizzata una finestra di dialogo Apri o Salva al termine dell'esecuzione della configurazione.
