# <a name="ABS">Funzione ER ABS</a>

[!include [banner](../includes/banner.md)]

La funzione `ABS` restituisce il valore assoluto (modulo) del numero specificato come valore *Reale*. In altre parole, restituisce il numero senza il segno.

## <a name="syntax"></a>Sintassi

```
ABS (number)
```

## <a name="arguments"></a>Argomenti

`number`: *Reale*

Un valore numerico di cui si desidera il modulo.

## <a name="return-values"></a>Valori restituiti

*Reale*

Il valore numerico risultante.

## <a name="example"></a>Esempio

`ABS (-1)` restituisce **1**.
