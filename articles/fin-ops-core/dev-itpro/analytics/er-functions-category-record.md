# <a name="list-of-er-functions-in-the-record-category"></a>Elenco delle funzioni ER nella categoria dei record

[!include [banner](../includes/banner.md)]

Le funzioni di record della creazione di report elettronici (ER) possono essere utilizzate per estrarre informazioni ed eseguire operazioni sulle origini dati del tipo di dati *Contenitore (record)*. Questo articolo fornisce un riepilogo di queste funzioni.

## <a name="list-of-supported-functions"></a>Elenco delle funzioni supportate

| Funzione | Descrizione |
|----------|-------------|
| [NullContainer](er-functions-record-nullcontainer.md) | Questa funzione restituisce un valore *Contenitore (record)* nullo che ha la stessa struttura dell'elenco record o del record specificato. |
| [EmptyRecord](er-functions-record-emptyrecord.md) | Questa funzione restituisce un valore *Contenitore (record)* nullo che ha la stessa struttura dell'elenco record o del record specificato. |
