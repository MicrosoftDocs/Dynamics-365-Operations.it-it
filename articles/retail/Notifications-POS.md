# <a name="show-order-notifications-in-the-point-of-sale"></a><span data-ttu-id="b3dbf-104">Visualizzare le notifiche degli ordini nel POS</span><span class="sxs-lookup"><span data-stu-id="b3dbf-104">Show order notifications in the point of sale</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="b3dbf-105">Nel moderno ambiente di vendita al dettaglio, ai collaboratori del POS vengono assegnate varie attività, come aiutare i clienti, inserire transazioni, eseguire conteggi di inventario e ricevere ordini nel POS.</span><span class="sxs-lookup"><span data-stu-id="b3dbf-105">In the modern retail environment, store associates are assigned various tasks, such as helping customers, entering transactions, doing stock counts, and receiving orders in the store.</span></span> <span data-ttu-id="b3dbf-106">Il client POS fornisce un'applicazione unica in cui i collaboratori possono eseguire tutte queste attività e molto altro.</span><span class="sxs-lookup"><span data-stu-id="b3dbf-106">The point of sale (POS) client provides a single application where associates can perform all these tasks and many others.</span></span> <span data-ttu-id="b3dbf-107">Poiché è necessario svolgere diversi compiti durante il giorno, potrebbe essere necessario che i collaboratori vengano avvisati quando qualcosa richiede la loro attenzione.</span><span class="sxs-lookup"><span data-stu-id="b3dbf-107">Because various tasks must be performed during the day, associates might have to be notified when something requires their attention.</span></span> <span data-ttu-id="b3dbf-108">Il framework di notifica nel POS agevola tale operazione consentendo ai rivenditori di configurare notifiche basate su ruoli.</span><span class="sxs-lookup"><span data-stu-id="b3dbf-108">The notification framework in the POS helps by letting retailers configure role-based notifications.</span></span> <span data-ttu-id="b3dbf-109">Nell'aggiornamento dell'applicazione 5 di Microsoft Dynamics 365 for Retail, le notifiche possono essere configurate solo per le operazioni di POS.</span><span class="sxs-lookup"><span data-stu-id="b3dbf-109">In Microsoft Dynamics 365 for Retail with application update 5, these notifications can be configured only for POS operations.</span></span>

## <a name="enable-notifications-for-order-fulfillment-operations"></a><span data-ttu-id="b3dbf-112">Attivare le notifiche per le operazioni di evasione dell'ordine</span><span class="sxs-lookup"><span data-stu-id="b3dbf-112">Enable notifications for order fulfillment operations</span></span>

<span data-ttu-id="b3dbf-113">Per attivare le notifiche per le operazioni di evasione dell'ordine, attenersi alla procedura seguente.</span><span class="sxs-lookup"><span data-stu-id="b3dbf-113">To enable notifications for order fulfillment operations, follow these steps.</span></span>

1. <span data-ttu-id="b3dbf-114">Passare a **Vendita al dettaglio** &gt; **Impostazione canale** &gt; **Impostazioni POS** &gt; **POS** &gt; **Operazioni**.</span><span class="sxs-lookup"><span data-stu-id="b3dbf-114">Go to **Retail** &gt; **Channel setup** &gt; **POS setup** &gt; **POS** &gt; **Operations**.</span></span>
2. <span data-ttu-id="b3dbf-115">Cercare l'operazione **Evasione ordine** e selezionare la relativa casella di controllo **Abilita notifiche** per specificare che il framework di notifica deve ascoltare il gestore per questa operazione.</span><span class="sxs-lookup"><span data-stu-id="b3dbf-115">Search for the **Order fulfillment** operation, and select the **Enable notifications** check box for it to specify that the notification framework should listen to the handler for this operation.</span></span> <span data-ttu-id="b3dbf-116">Se il gestore è implementato, le notifiche per l'operazione verranno quindi visualizzate nel POS.</span><span class="sxs-lookup"><span data-stu-id="b3dbf-116">If the handler is implemented, notifications for this operation will then be shown in the POS.</span></span>
3. <span data-ttu-id="b3dbf-117">Passare a **Vendita al dettaglio** &gt; **Dipendenti** &gt; **Lavoratori** &gt;, nella scheda Vendita al dettaglio, aprire le autorizzazioni POS associate al lavoratore.</span><span class="sxs-lookup"><span data-stu-id="b3dbf-117">Go to **Retail** &gt; **Employees** &gt; **Workers** &gt;, under Retail tab, open the POS permissions associated with the worker.</span></span> <span data-ttu-id="b3dbf-118">Espandere la scheda dettaglio **Notifiche**, aggiungere l'operazione **Evasione ordine** e impostare il campo **Ordine di visualizzazione** su **1**.</span><span class="sxs-lookup"><span data-stu-id="b3dbf-118">Expand the **Notifications** FastTab, add the **Order fulfillment** operation, and set the **Display order** field to **1**.</span></span> <span data-ttu-id="b3dbf-119">Se più di una notifica è configurata, questo campo viene utilizzato per disporre le notifiche.</span><span class="sxs-lookup"><span data-stu-id="b3dbf-119">If more than one notification is configured, this field is used to arrange the notifications.</span></span> <span data-ttu-id="b3dbf-120">Le notifiche che hanno un valore di **Ordine di visualizzazione** inferiore vengono visualizzate sopra a quelle con un valore superiore.</span><span class="sxs-lookup"><span data-stu-id="b3dbf-120">Notifications that have a lower **Display order** value appear above notifications that have a higher value.</span></span> <span data-ttu-id="b3dbf-121">Le notifiche con un valore di **Ordine di visualizzazione** di **1** vengono visualizzate nella parte superiore.</span><span class="sxs-lookup"><span data-stu-id="b3dbf-121">Notifications that have a **Display order** value of **1** are at the top.</span></span>

    <span data-ttu-id="b3dbf-122">Le notifiche vengono visualizzate solo per le operazioni che vengono aggiunte nella scheda dettaglio **Notifiche** ed è possibile aggiungere operazioni in tale posizione solo se la casella di controllo **Abilita notifiche** per queste operazioni è stata selezionata nella pagina **Operazioni POS**.</span><span class="sxs-lookup"><span data-stu-id="b3dbf-122">Notifications are shown only for operations that are added on the **Notifications** FastTab, and you can add operations there only if the **Enable notifications** check box for those operations has been selected on the **POS operations** page.</span></span> <span data-ttu-id="b3dbf-123">Inoltre, le notifiche per un'operazione vengono mostrate ai lavoratori solo se l'operazione viene aggiunta alle autorizzazioni POS per tali lavoratori.</span><span class="sxs-lookup"><span data-stu-id="b3dbf-123">Additionally, notifications for an operation are shown to workers only if the operation is added to the POS permissions for those workers.</span></span>

    > [!NOTE]
    > <span data-ttu-id="b3dbf-124">Le notifiche possono essere sostituite a livello di utente.</span><span class="sxs-lookup"><span data-stu-id="b3dbf-124">Notifications can be overridden at the user level.</span></span> <span data-ttu-id="b3dbf-125">Aprire il record del lavoratore, selezionare **Autorizzazioni POS** e quindi modificare la sottoscrizione di notifica dell'utente.</span><span class="sxs-lookup"><span data-stu-id="b3dbf-125">Open the worker's record, select **POS permissions**, and then edit the user's notification subscription.</span></span>

4. <span data-ttu-id="b3dbf-126">Passare a **Vendita al dettaglio** &gt; **Impostazione canale** &gt; **Impostazioni POS** &gt; **Profili POS** &gt; **Profili funzionalità**.</span><span class="sxs-lookup"><span data-stu-id="b3dbf-126">Go to **Retail** &gt; **Channel setup** &gt; **POS setup** &gt; **POS profiles** &gt; **Functionality profiles**.</span></span> <span data-ttu-id="b3dbf-127">Nel campo **Intervallo di notifica**, specificare la frequenza con cui le notifiche devono essere recuperate.</span><span class="sxs-lookup"><span data-stu-id="b3dbf-127">In the **Notification interval** field, specify how often notifications should be pulled.</span></span> <span data-ttu-id="b3dbf-128">Per alcune notifiche, il POS deve effettuare chiamate in tempo reale all'applicazione di back-office.</span><span class="sxs-lookup"><span data-stu-id="b3dbf-128">For some notifications, the POS must make real-time calls to the back-office application.</span></span> <span data-ttu-id="b3dbf-129">Queste chiamate utilizzano la capacità di calcolo dell'applicazione di back-office.</span><span class="sxs-lookup"><span data-stu-id="b3dbf-129">These calls consume the compute capacity of your back-office application.</span></span> <span data-ttu-id="b3dbf-130">Pertanto, quando si imposta l'intervallo di notifica, è consigliabile prendere in considerazione sia i requisiti aziendali che l'impatto delle chiamate in tempo reale all'applicazione di back-office.</span><span class="sxs-lookup"><span data-stu-id="b3dbf-130">Therefore, when you set the notification interval, you should consider both your business requirements and the impact of real-time calls to the back-office application.</span></span> <span data-ttu-id="b3dbf-131">Un valore di **0** (zero) disattiva notifiche.</span><span class="sxs-lookup"><span data-stu-id="b3dbf-131">A value of **0** (zero) turns off notifications.</span></span>
5. <span data-ttu-id="b3dbf-132">Passare a **Vendita al dettaglio** &gt; **IT vendita al dettaglio** &gt; **Programmazione della distribuzione**.</span><span class="sxs-lookup"><span data-stu-id="b3dbf-132">Go to **Retail** &gt; **Retail IT** &gt; **Distribution schedule**.</span></span> <span data-ttu-id="b3dbf-133">Selezionare la programmazione **1060** (**Personale**) per sincronizzare le impostazioni di sottoscrizione della notifica e quindi selezionare **Esegui ora**.</span><span class="sxs-lookup"><span data-stu-id="b3dbf-133">Select the **1060** (**Staff**) schedule to synchronize notification subscription settings, and then select **Run now**.</span></span> <span data-ttu-id="b3dbf-134">Quindi, selezionare la programmazione **1070** (**Configurazione canale**) per sincronizzare l'intervallo di autorizzazione, quindi selezionare **Esegui ora**.</span><span class="sxs-lookup"><span data-stu-id="b3dbf-134">Next, select the **1070** (**Channel configuration**) schedule to synchronize the permission interval, and then select **Run now**.</span></span>
