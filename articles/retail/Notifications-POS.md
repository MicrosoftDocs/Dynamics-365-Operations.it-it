# <a name="display-notifications-in-point-of-sale"></a><span data-ttu-id="50071-103">Notifiche di visualizzazione in POS</span><span class="sxs-lookup"><span data-stu-id="50071-103">Display notifications in Point of Sale</span></span>

[!include[banner](includes/banner.md)]

<span data-ttu-id="50071-104">Nell'odierno ambiente di vendita al dettaglio, ai collaboratori del POS vengono assegnate varie attività, come aiutare i clienti, inserire transazioni, eseguire conteggi di inventario e ricevere ordini nel POS.</span><span class="sxs-lookup"><span data-stu-id="50071-104">In today's modern retail environment, store associates are assigned various tasks, such as helping customers, entering transactions, performing stock counts, and receiving orders in store.</span></span> <span data-ttu-id="50071-105">Il client POS consente ai collaboratori di eseguire queste attività e molto altro, tutto in un'unica applicazione.</span><span class="sxs-lookup"><span data-stu-id="50071-105">The Point of Sale (POS) client empowers the associates to do these tasks and much more, all in a single application.</span></span> <span data-ttu-id="50071-106">Con diversi compiti da svolgere durante un giorno, potrebbe essere necessario che i collaboratori vengano avvisati quando qualcosa richiede la loro attenzione.</span><span class="sxs-lookup"><span data-stu-id="50071-106">With various tasks to be performed during a day, associates may need to be notified when something requires their attention.</span></span> <span data-ttu-id="50071-107">Il framework di notifica nel POS risolve questo problema consentendo ai rivenditori di configurare notifiche basate su ruoli.</span><span class="sxs-lookup"><span data-stu-id="50071-107">The notification framework in the POS solves this problem by allowing the retailers to configure role-based notifications.</span></span> <span data-ttu-id="50071-108">Nell'Aggiornamento applicazione 5 di Dynamics 365 for Retail, le notifiche possono essere configurate solo per le operazioni di POS.</span><span class="sxs-lookup"><span data-stu-id="50071-108">With Dynamics 365 for Retail with Application update 5, these notifications can be only configured for POS operations.</span></span>

<span data-ttu-id="50071-109">Attualmente, il sistema offre la possibilità di visualizzare le notifiche per l'evasione degli ordini, tuttavia, il framework è progettato per essere estensibile, in modo che in futuro gli sviluppatori possano scrivere un gestore di notifiche per qualsiasi operazione e visualizzare le notifiche nel POS.</span><span class="sxs-lookup"><span data-stu-id="50071-109">Currently, the system provides the capability to display notifications for order fulfillment operation, however, the framework is designed to be extensible, so that in the future, developers will be able to write a notification handler for any operation and display the notifications in POS.</span></span>  

## <a name="enable-notifications-for-order-fulfillment-operations"></a><span data-ttu-id="50071-110">Attivare le notifiche per le operazioni di evasione dell'ordine</span><span class="sxs-lookup"><span data-stu-id="50071-110">Enable notifications for order fulfillment operations</span></span>

<span data-ttu-id="50071-111">Per attivare le notifiche per le operazioni di evasione dell'ordine, fare riferimento alle seguenti operazioni:</span><span class="sxs-lookup"><span data-stu-id="50071-111">To enable notifications for the order fulfillment operations, refer to the following the steps:</span></span>

 - <span data-ttu-id="50071-112">Passare alla pagina **Operations** (**Retail** > **Impostazione canale** > **Impostazioni POS** > **POS** > **Operazioni**).</span><span class="sxs-lookup"><span data-stu-id="50071-112">Go to the **Operations** page (**Retail** > **Channel setup** > **POS setup** > **POS** > **Operations**).</span></span>

- <span data-ttu-id="50071-116">Passare alle autorizzazioni di POS associate ai lavoratori e nella scheda dettaglio **Notifiche** aggiungere l'operazione di evasione ordine con "ordine di visualizzazione" 1.</span><span class="sxs-lookup"><span data-stu-id="50071-116">Go to the POS permissions associated with the workers and under the **Notifications** FastTab, add the Order fulfillment operation with the "Display order" as 1.</span></span> <span data-ttu-id="50071-117">Quando è presente più di una notifica configurata, l'ordine di visualizzazione viene utilizzato per gestire le notifiche dall'alto verso il basso, in cui il valore 1 indica l'elemento nella parte superiore.</span><span class="sxs-lookup"><span data-stu-id="50071-117">When there is more than one notification configured, the display order is used to arrange the notification from top to bottom with 1 being on top.</span></span> <span data-ttu-id="50071-118">È possibile aggiungere solo quelle operazioni per le quali è stata selezionata la casella di controllo **Abilita notifiche**.</span><span class="sxs-lookup"><span data-stu-id="50071-118">Only those operations can be added for which the **Enable notifications** check box has been selected.</span></span> <span data-ttu-id="50071-119">Inoltre, le notifiche vengono visualizzate solo per le operazioni che sono state aggiunte in questo campo e solo ai lavoratori per cui le operazioni sono state aggiunte alle autorizzazioni corrispondenti di POS.</span><span class="sxs-lookup"><span data-stu-id="50071-119">Also, the notifications will be displayed only for the operations that have been added here and only to those workers for whom the operations have been added to the corresponding POS permissions.</span></span> 

> [!NOTE]
> <span data-ttu-id="50071-120">Le notifiche possono essere ignorate a livello utente passando al record del lavoratore e selezionando **Autorizzazioni POS** e quindi modificando la sottoscrizione di notifica dell'utente.</span><span class="sxs-lookup"><span data-stu-id="50071-120">Notifications can be overridden at the user level by navigating to the worker's record and selecting **POS Permissions** and then editing that user's notification subscription.</span></span>

 - <span data-ttu-id="50071-121">Passare alla pagina **Profilo funzionalità** (**Retail** > **Impostazione canale** > **Impostazione POS** > **Profili POS** > **Profili funzionalità**).</span><span class="sxs-lookup"><span data-stu-id="50071-121">Go to the **Functionality profile** page (**Retail** > **Channel setup** > **POS setup** > **POS profiles** > **Functionality profiles**).</span></span> <span data-ttu-id="50071-122">Aggiornare la proprietà **Intervallo di notifica** per impostare l'intervallo in minuti in cui le notifiche devono essere recuperate.</span><span class="sxs-lookup"><span data-stu-id="50071-122">Update the **Notification interval** property, to set the interval in minutes at which the notifications should be pulled.</span></span> <span data-ttu-id="50071-123">Si consiglia di impostare il valore su 10 minuti per evitare comunicazioni superflue alla sede centrale.</span><span class="sxs-lookup"><span data-stu-id="50071-123">We recommend setting this value to 10 minutes to avoid unnecessary communication to the headquarters.</span></span> <span data-ttu-id="50071-124">Impostando l'intervallo di notifica su "0 "le notifiche verranno disattivate.</span><span class="sxs-lookup"><span data-stu-id="50071-124">Setting the notification interval to "0" will turn off notifications.</span></span>  

 - <span data-ttu-id="50071-125">Passare a **Retail** > **IT vendita al dettaglio** > **Programmazione della distribuzione**.</span><span class="sxs-lookup"><span data-stu-id="50071-125">Go to **Retail** > **Retail IT** > **Distribution schedule**.</span></span> <span data-ttu-id="50071-126">Selezionare la pianificazione "1060, Personale" per sincronizzare le impostazioni di sottoscrizione della notifica e quindi fare clic su **Esegui ora**.</span><span class="sxs-lookup"><span data-stu-id="50071-126">Select schedule "1060-Staff" to sync notification subscription settings and then click **Run now**.</span></span> <span data-ttu-id="50071-127">Quindi, sincronizzare l'intervallo di autorizzazione selezionando "1070 (configurazione canale)", quindi fare clic su **Esegui ora**.</span><span class="sxs-lookup"><span data-stu-id="50071-127">Next, sync the permission interval by selecting the "1070-Channel configuration" and then click **Run now**.</span></span>
