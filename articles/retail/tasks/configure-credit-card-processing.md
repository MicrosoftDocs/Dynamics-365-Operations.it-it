# <a name="configure-credit-card-processing"></a> Configurare l'elaborazione delle carte di credito

[!include[task guide banner](../includes/task-guide-banner.md)]

In questa procedura vengono descritti i passaggi per visualizzare l'elenco dei provider dei servizi di pagamento e configurare un conto pagamenti per la contabilità clienti. Questa procedura utilizza i dati dimostrativi della società USRT ed è destinata ad amministratori e professionisti IT.


## <a name="view-a-list-of-payment-providers"></a>Visualizzare un elenco di provider di servizi di pagamento
1. Fare clic su Contabilità clienti > Impostazione pagamenti > Servizi di pagamento.
2. Fare clic su Visualizza provider disponibili.

## <a name="configure-payment-account"></a>Configurare un conto pagamenti
1. Fare clic su Nuovo.
2. Digitare un valore nel campo Servizio di pagamento.
3. Selezionare un'opzione nel campo Connettore pagamenti.
4. Attivare l'espansione della sezione Conto servizio di pagamento.
5. Nel campo Ambiente: digitare 'PROD'.
6. Fare clic su Tipi carta di credito.
7. Nel campo Giornale di registrazione pagamenti fare clic sul pulsante a discesa per aprire la ricerca.
8. Nell'elenco fare clic sul collegamento nella riga selezionata.
9. Scegliere Aggiungi.
10. Digitare un valore nel campo Valuta.
11. Nell'elenco trovare e selezionare il record desiderato.
12. Nel campo Giornale di registrazione pagamenti fare clic sul pulsante a discesa per aprire la ricerca.
13. Nell'elenco fare clic sul collegamento nella riga selezionata.
14. Scegliere Aggiungi.
15. Digitare un valore nel campo Valuta.
16. Nell'elenco trovare e selezionare il record desiderato.
    * È possibile ripetere questi passaggi per un numero illimitato di tipi di carta in base alle necessità.  
17. Nel campo Giornale di registrazione pagamenti fare clic sul pulsante a discesa per aprire la ricerca.
18. Nell'elenco fare clic sul collegamento nella riga selezionata.
19. Scegliere Aggiungi.
20. Digitare un valore nel campo Valuta.
21. Fare clic su Salva.
22. Chiudere la pagina.
23. Fare clic su Convalida.
24. Fare clic sulla casella di controllo Elaboratore predefinito nuove carte di credito.
25. Fare clic su Salva.
