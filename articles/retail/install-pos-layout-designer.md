[!include[banner](includes/banner.md)]

## <a name="install-the-one-click-designer"></a><span data-ttu-id="be7a8-109">Installare lo strumento di progettazione con un clic</span><span class="sxs-lookup"><span data-stu-id="be7a8-109">Install the one-click designer</span></span>
1.  <span data-ttu-id="be7a8-110">In Microsoft Dynamics 365 for Retail, utilizzare il menu in alto a sinistra per accedere a **Vendita al dettaglio** **e commercio** &gt; **Impostazione canale** &gt; **Impostazioni POS** &gt; **POS** &gt; **Layout schermo**.</span><span class="sxs-lookup"><span data-stu-id="be7a8-110">In Microsoft Dynamics 365 for Retail, use the menu in the upper left to navigate to **Retail** **and commerce** &gt; **Channel setup** &gt; **POS setup** &gt; **POS** &gt; **Screen layouts**.</span></span>
2.  <span data-ttu-id="be7a8-111">Selezionare il layout con tipo applicazione **Modern POS per Windows** o **POS cloud**, quindi fare clic su **Progettazione layout**.</span><span class="sxs-lookup"><span data-stu-id="be7a8-111">Select any layout that has an application type of **Modern POS for Windows** or **Cloud POS**, and then click **Layout designer**.</span></span>
3.  <span data-ttu-id="be7a8-112">Nella barra di notifica visualizzata nella parte inferiore della finestra di Internet Explorer, fare clic su **Apri** per avviare l'installazione della progettazione con un clic.</span><span class="sxs-lookup"><span data-stu-id="be7a8-112">On the notification bar that appears at the bottom of the Internet Explorer window, click **Open** to install the one-click designer.</span></span> <span data-ttu-id="be7a8-113">(La barra di notifica potrebbe essere visualizzata in un posto diverso in altri browser).</span><span class="sxs-lookup"><span data-stu-id="be7a8-113">(The notification bar might appear in a different place in other browsers.)</span></span>
4.  <span data-ttu-id="be7a8-114">Nella finestra di messaggio **Esecuzione applicazione - Avviso di sicurezza** visualizzata, fare clic su  **Esegui** per installare l'host della progettazione Retail.</span><span class="sxs-lookup"><span data-stu-id="be7a8-114">In the **Application Run - Security Warning** message box that appears, click **Run** to install the Retail designer host.</span></span> <span data-ttu-id="be7a8-115">Un indicatore di stato mostra lo stato di avanzamento di installazione.</span><span class="sxs-lookup"><span data-stu-id="be7a8-115">A progress indicator shows the progress of the installation.</span></span>
5.  <span data-ttu-id="be7a8-116">Dopo che l'installazione è stata completata, nella pagina **Accedi** immettere nome utente e password di Microsoft Dynamics 365 for Retail e quindi fare clic **Accedi** per avviare lo strumento di progettazione.</span><span class="sxs-lookup"><span data-stu-id="be7a8-116">After the installation is completed, on the **Sign in** page, enter your Microsoft Dynamics 365 for Retail user name and password, and then click **Sign in** to start the designer.</span></span>
6.  <span data-ttu-id="be7a8-117">Dopo che le credenziali vengono convalidate e si avvia la progettazione, è possibile iniziare a progettare il proprio layout o modificare il layout esistente.</span><span class="sxs-lookup"><span data-stu-id="be7a8-117">After your credentials are validated and the designer starts, you can design your own layout or modify the existing layout.</span></span> <span data-ttu-id="be7a8-118">[![Layout nello strumento di progettazione con un clic](./media/screenlayoutdesign_mposdownload-1024x664.png)](./media/screenlayoutdesign_mposdownload.png)</span><span class="sxs-lookup"><span data-stu-id="be7a8-118">[![Layout in the one-click designer](./media/screenlayoutdesign_mposdownload-1024x664.png)](./media/screenlayoutdesign_mposdownload.png)</span></span>
