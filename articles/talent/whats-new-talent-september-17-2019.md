# <a name="whats-new-or-changed-in-dynamics-365-for-talent-september-17-2019"></a>Novità o modifiche in Dynamics 365 for Talent (17 settembre 2019)

Questo argomento descrive le funzionalità nuove o modificate in Dynamics 365 for Talent.

## <a name="changes-in-attract"></a>Modifiche in Attract
Questa versione include correzioni di bug minori per Dynamics 365 Talent: Attract.

## <a name="changes-in-onboard"></a>Modifiche in Onboard
Questa versione include correzioni di bug minori per Dynamics 365 Talent: Onboard.

## <a name="changes-in-core-hr"></a>Modifiche di Core HR
Le modifiche descritte in questo sezione sono valide per la build numero 8.1.2492. I numeri tra parentesi in alcune intestazioni si riferiscono ai numeri del supporto in Microsoft Dynamics Lifecycle Services (LCS).

### <a name="you-can-enable-preview-features-in-sandbox-and-trial-environments"></a>È possibile attivare le funzionalità di anteprima negli ambienti Sandbox e di prova

Quando si esegue il provisioning di una nuova istanza di Talent, è possibile specificare se il tipo di istanza è Produzione o Sandbox. Le istanze Sandbox consentono di provare le nuove funzionalità prima dell'utilizzo. Se si desidera aggiornare una delle istanze esistenti al tipo di istanza Sandbox, contattare il supporto tecnico per avviare la richiesta di modifica.

Per ulteriori informazioni su come vengono pubblicate le modifiche, vedere [Eseguire il provisioning di Talent](./provisioning-talent.md).

### <a name="human-resources-staff-cant-see-performance-review-details-once-assigned-to-them-by-workflow-370308"></a>Il personale delle risorse umane non può visualizzare i dettagli della revisione delle prestazioni una volta assegnato dal flusso di lavoro (370308)

Con l'aggiornamento di questa settimana, i professionisti delle risorse umane saranno in grado di vedere i dettagli della revisione delle prestazioni che sono stati assegnati attraverso l'elaborazione del flusso di lavoro. Per visualizzare le revisioni, spostarsi a **Dipendente self-service > Elementi di lavoro assegnati all'utente**.

### <a name="job-family-field-missing-in-the-manage-changes-page-for-job-details-346031"></a>Il campo Famiglia di processi manca nella pagina Gestisci modifiche per i dettagli del processo (346031)

In questa versione, il campo iFamiglie di processi è stato aggiunto alla pagina **Gestione modifiche** per i dettagli del processo.

## <a name="in-preview"></a>In anteprima

### <a name="streamlined-employee-entry-and-navigation"></a>Inserimento e navigazione dei dipendenti semplificati

Questa funzionalità è ora disponibile negli ambienti sandbox. Per attivare questa funzionalità, passare ad **Amministrazione sistema > Collegamenti > Impostazioni > Parametri di sistema > Funzionalità di anteprima**. Selezionare **Navigazione e modulo lavoratore avanzati**. In questo modo le modifiche saranno valide per tutti gli utenti. È possibile disattivare questa opzione in qualsiasi momento.

Per ulteriori informazioni, vedere [Inserimento e navigazione dei dipendenti semplificati](./streamlined-employee-entry.md). Per visualizzare le modifiche, guardare il video [Panoramica della seconda ondata della versione Dynamics 365 for Talent 2019](https://aka.ms/ROGT19RW2ROV).
