# <a name="create-an-onboarding-template-by-using-dynamics-365-talent---onboard"></a>Creare un modello per l'inserimento utilizzando Dynamics 365 Talent - Onboard

[!include [banner](includes/banner.md)]

Microsoft Dynamics 365 Talent: Onboard include vari modelli che consentono di creare rapidamente una guida per l'inserimento. È possibile utilizzare uno o più di questi modelli oppure creare i propri modelli. Onboard fornisce testo di esempio che è possibile utilizzare quando si creano i propri modelli. Di conseguenza, il processo è semplice anche se si inizia da zero.

## <a name="create-an-onboarding-template-from-an-existing-template"></a>Creare un modello per l'inserimento da un modello esistente

1. Nel menu a sinistra, selezionare **Modelli**.
2. In **Modelli popolari della raccolta** o **Modelli personali**, selezionare un modello. Per visualizzare più modelli, selezionare **Altro nella raccolta di modelli**.
3. Eseguire uno dei passaggi riportati di seguito.

    - Se il modello proviene dalla raccolta, selezionare **Salva come modello personale**, immettere un nuovo nome per il modello e selezionare **Salva**.
    - Se il modello proviene da **Modelli personali**, selezionare **Salva come modello**, immettere un nuovo nome per il modello e selezionare **Salva**.

    [![Creazione di un modello per l'inserimento da un modello esistente](./media/onboard-save-template.png)](./media/onboard-save-template.png)

## <a name="create-a-new-onboarding-template"></a>Creare un nuovo modello di inserimento

1. Nel menu a sinistra, selezionare **Modelli**.
2. In **Modelli personali**, selezionare il riquadro **Aggiungi** (segno più \[**+**\]).

    [![Creazione di un nuovo modello](./media/onboard-create-new-template.png)](./media/onboard-create-new-template.png)

3. Nella finestra di dialogo **Crea un modello di guida** immettere un nome per il modello, quindi selezionare **Salva**.
