# <a name="whats-new-or-changed-in-dynamics-365-talent---core-hr-january-17-2019"></a>Novità o modifiche in Dynamics 365 Talent - Core HR (17 gennaio 2019)

[!include [banner](includes/banner.md)]

**Build 8.1.2107**

Questo argomento descrive le funzionalità nuove o modificate di Core HR.

## <a name="minor-changes-and-bug-fixes-included-in-this-release"></a>Modifiche e correzioni di bug secondarie incluse in questa versione

### <a name="new-position-assignment-start-datetime-default-has-been-corrected"></a>Il nuovo valore di default della data/ora di inizio dell'assegnazione della posizione è stata corretta
Durante il trasferimento di dipendenti, per impostazione predefinita il sistema utilizzerà la data selezionata e l'ora all'inizio della giornata, per il fuso orario dell'utente.

### <a name="all-employees-have-the-same-position-description-in-the-exited-worker-task-management-list"></a>Tutti i dipendenti hanno la stessa descrizione della posizione nell'elenco di gestione delle attività dei lavoratori che hanno lasciato la società
È stato effettuato un aggiornamento nell'area di lavoro **Gestione attività** per visualizzare le descrizioni della posizione corrette per tutti i dipendenti che hanno lasciato la società di recente.

### <a name="action-requested-by-field-is-now-populated-on-workers-action-page"></a>Il campo "Azione richiesta da" è ora popolato nella pagina dei lavoratori
Con questa modifica, il valore predefinito del campo **Azione richiesta da** è ora l'utente che richiede la modifica.

### <a name="ideas-portal-updated"></a>Portale Ideas aggiornato
Nell'app, i collegamenti per inviare un'idea per Talent sono stati aggiornati nel nuovo portale Ideas. Tutte le idee immesse nel portale precedente sono state migrate, di conseguenza possono ancora essere votate da altri utenti.
